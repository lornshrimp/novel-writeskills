# 小说分发目录说明

`小说分发/` 用于承载平台分发方案与流程文档，定位为**分发流程目录**，不是题材目录。

## 目录边界

- 这里主要放分发 SOP、平台落盘策略、执行编排、检查清单与复盘文档。
- 本目录不强制按“题材名-能力名”创建题材层 Skill。
- 本目录下若新增 Skill，不要求与题材层 Skill 建立一一对应关系。

## 规则入口

本目录的治理规则见：

- `.github/instructions/distribution-directory-boundary.instructions.md`

## 与通用能力的关系

若分发场景中沉淀出可跨题材复用的稳定能力，应回收至 `通用skills/`；
`小说分发/` 仅保留分发流程侧补充规则与落地说明。
