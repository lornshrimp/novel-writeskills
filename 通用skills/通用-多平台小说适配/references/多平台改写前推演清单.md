# 多平台改写前推演清单

在写任一平台版本之前，先回答：

1. 本平台最关键的 3 条风格约束是什么？
2. 它和其他平台相比，最该拉开的 2 个差异点是什么？
3. 本次准备在哪些表达位上压相似度？
   - 标题
   - 开头前三段
   - 中段回报承载方式
   - 对话策略
   - 证据呈现方式
   - 章末钩子类型
   - 作者有话说口吻
4. 本平台最怕写成哪一种别的平台气质？
5. 遮住标题后，读者还能不能认出平台风格？

## 合规降相似度方向

- 结构重排
- 改对白与动作链
- 改叙述焦点
- 改证据承载方式
- 改回报顺序

## 禁止方向

- 随机噪声
- 灌水凑字数
- 无意义换同义词
- 为过相似度而丢平台感
