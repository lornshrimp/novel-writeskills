# 都市悬疑平台差异矩阵补充

## 默认结论

- 起点：看开篇抓力、持续追读、爽点与长线供血。
- 番茄：更强调快反馈、强钩子、短段落和即时情绪刺激。
- 豆瓣：更强调文学质感、现实纹理、克制表达与余味。
- 知乎盐选：更强调付费节点前后的反转与压缩叙事效率。
- 微信订阅号：更强调钩子链、连载留存与传播切口。
- 出版社 / 纸媒：更强调整体文学完成度与删繁压缩能力。
- GoodNovel / My Fiction / WebNovel：更强调快节奏、强视觉、强事件点与明确情绪回报。

## 适配时必须先问

- 本平台最重视的是“追读”、还是“余味”、还是“付费卡点”？
- 同一章在不同平台，最该被放大的是什么？
- 哪个平台版本最不能沿用中性底稿轻改？
