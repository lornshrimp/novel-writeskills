# 人物传记审阅检查清单

## 一、身份与归属

- 姓名、年龄、职业、组织归属是否一致
- 对外身份与真实身份是否分层清楚
- 谁能证明他的身份，谁能否定他的身份

## 二、动机—代价闭环

- 想要什么
- 害怕失去什么
- 每次关键行动会付什么代价
- 是否存在零代价推进

## 三、能力边界

- 会什么 / 不会什么是否写清
- 资源来源是否可靠
- 是否明确不能用、不敢用、用了会出事的情形

## 四、关系网

- 至少有亲密 / 利益 / 对立三类关系
- 每条关系是否能生成冲突事件或情感回报
- 权力结构是否清楚

## 五、写作抓手

- 口头禅是否可用
- 习惯动作是否可拍
- 触发器是否具体
- 留痕偏好是否可复盘
- 反制模板是否像人，不像教程

## 六、心理弧线

- 是否按阶段推进
- 是否有触发事件支持
- 是否有回摆、失败和误判
- 是否与索引映射一致

## 七、索引映射

- 是否能回指到总纲 / 分卷 / 章节
- 编号是否清晰
- 事件作用是否写明
