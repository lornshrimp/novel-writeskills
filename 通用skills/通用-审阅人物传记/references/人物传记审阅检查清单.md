# 人物传记审阅检查清单

## 一、身份与归属

- 姓名、年龄、职业、组织归属是否一致
- 对外身份与真实身份是否分层清楚
- 谁能证明他的身份，谁能否定他的身份

## 二、动机—代价闭环

- 想要什么
- 害怕失去什么
- 每次关键行动会付什么代价
- 是否存在零代价推进
- 是否写清读者为何会替其担心（危机 / 困难 / 弱者 / 脆弱）

## 三、能力边界

- 会什么 / 不会什么是否写清
- 资源来源是否可靠
- 是否明确不能用、不敢用、用了会出事的情形

## 四、关系网

- 至少有亲密 / 利益 / 对立三类关系
- 每条关系是否能生成冲突事件或情感回报
- 权力结构是否清楚

## 五、写作抓手

- 口头禅是否可用
- 习惯动作是否可拍
- 触发器是否具体
- 留痕偏好是否可复盘
- 反制模板是否像人，不像教程
- 外貌描写是否同时承担性格 / 态度 / 关系信息，而不是只列五官

## 六、读者联结

- 是否至少存在 1 个认同点
- 是否至少存在 1 个同情入口
- 是否至少存在 1 个喜爱点或可敬点
- 是否存在真实内心冲突，而不是一路笃定到底
- 若人物天然不讨喜，是否补了足够的吸引点支撑读者继续跟随

## 七、心理弧线

- 是否按阶段推进
- 是否有触发事件支持
- 是否有回摆、失败和误判
- 是否与索引映射一致

## 八、缺陷与反模式化

- 是否至少有 1 个会真实反噬的缺陷
- 缺陷是否进入关键选择，而不只是挂名设定
- 是否存在身体 / 情感 / 关系 / 资源上的脆弱处
- 是否容易滑入刻板印象
- 是否补了至少 1 条成立的反模式化修正

## 九、索引映射

- 是否能回指到总纲 / 分卷 / 章节
- 编号是否清晰
- 事件作用是否写明
