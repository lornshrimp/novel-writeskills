# 人物传记审阅检查清单

## 一、阅读链路与对象边界

- 是否已确认写作口径 Prompt、关键研究、总纲、线索台账 / 嫌疑池 / 伏笔回收、关联分卷大纲、目标人物传记与关键登场章节
- 若上游材料缺失，是否明确记录阻断点、替代依据与由此带来的风险

## 二、基本信息检查

- 基本信息是否完整
- 格式是否规范
- 内容逻辑是否自洽
- 是否在原有细分结构上显式覆盖背景、性格、外貌、其它四大基础栏位
- “其它”栏是否至少落到能力边界、关系网、写作抓手、秘密 / 爆点、心理弧线 / 索引等作者侧关键项，而不是空泛兜底
- 目标人物文件名、索引口径、阶段称呼是否统一

## 三、人物一句话定义与角色功能位

- 是否能用一句话讲清公开身份、隐秘角色、核心欲望、核心恐惧
- 是否明确该人物在故事中的功能位：主角 / 搭档 / 反派 / 关键证人 / 路人功能位等
- 若删掉该人物，哪一块剧情会塌

## 四、首次出场与前三章留存验证

- 首次出场是否落在最能体现人物价值 / 压力 / 反差的熟悉情境里
- 首次出场是否已经抛出矛盾、欲望、危险、误读或站队点，而不是只交代履历
- 前三章内是否已经兑现一次有效回报（专业动作 / 情绪回报 / 反击 / 关系拉扯 / 误判翻车）
- 首秀是否留下可截图、可转述、可催更的态度句、动作点或关系钩子

## 五、人物—情境—观点合拍度

- 这个人物是否像目标读者熟悉的人、怕成为的人、或想跟随的人，而不是抽象标签集合
- 是否选择了最能凸显其价值与问题的日常情境，而不是“哪里都能演”的空舞台
- 人物的情绪站位、价值表达、对他人的看法是否符合其身份与目标读者的现实想法
- 是否做到了“人物、情境、观点”一起工作，而不是只靠人物简介撑住吸引力

## 六、可演出性与可写性评估（P0）

- 三秒立人：是否有一眼记住的外貌 / 动作 / 语言抓手
- 行动驱动：目标、欲望、底线是否能直接驱动选择
- 缺陷可复用：缺陷是否能反复制造麻烦与代价
- 关系网络可用：关系是否是“资源 / 制衡 / 把柄 / 情感债”的可操作网络
- 与题材机制绑定：是否明确绑定至少 1 个本题材的机制抓手 / 压迫手段 / 破局手段

## 七、身份与归属

- 姓名、年龄、职业、组织归属是否一致
- 对外身份与真实身份是否分层清楚
- 谁能证明他的身份，谁能否定他的身份

## 八、动机—代价闭环

- 想要什么
- 害怕失去什么
- 每次关键行动会付什么代价
- 是否存在零代价推进
- 是否写清读者为何会替其担心（危机 / 困难 / 脆弱 / 弱点）

## 九、能力边界

- 会什么 / 不会什么是否写清
- 资源来源是否可靠
- 是否明确不能用、不敢用、用了会出事的情形

## 十、关系网与镜像对手

- 至少有亲密 / 利益 / 对立三类关系
- 每条关系是否能生成冲突事件或情感回报
- 权力结构是否清楚
- 是否存在镜像 / 反镜像的重要人物

## 十一、新颖性与趋势适配

- 人物设定 / 成长弧 / 关系网络是否避免套路并具辨识度
- 职业处境 / 时代焦虑 / 现实压力是否真的进入冲突链与证据链，而非停留在标签

## 十二、人物标签与成长弧

- 外貌特征是否有 `1–2` 处可被反复引用的具体表述
- 标志性动作是否稳定可复现
- 口头禅 / 语言风格是否有高频句式或词汇偏好
- 成长弧四步是否成立：初始缺陷 → 触发事件 → 认知转变 → 行为体现
- 跨卷延续性是否成立

## 十三、可复用写作素材库

- 是否至少补齐 `4` 类素材
- 每类是否给出 `3–7` 条可直接贴回正文的内容
- 口癖 / 句式、动作、雷点、把柄、道具绑定是否都能直接写进场景

## 十四、秘密与爆点

- 是否至少有 `1` 个能推动情节的秘密 / 把柄
- 秘密是否具备暴露条件、触发代价与后果
- 若秘密过于抽象，是否已被标记为承重风险

## 十五、读者联结

- 是否至少存在 1 个认同点
- 是否至少存在 1 个同情入口
- 是否至少存在 1 个喜爱点或可敬点
- 是否存在真实内心冲突，而不是一路笃定到底
- 若人物天然不讨喜，是否补了足够的吸引点支撑读者继续跟随

## 十六、心理弧线与缺陷反噬

- 是否按阶段推进
- 是否有触发事件支持
- 是否有回摆、失败和误判
- 是否至少有 1 个会真实反噬的缺陷
- 缺陷是否进入关键选择，而不只是挂名设定

## 十七、大纲完整性与索引映射

- 是否能回指到总纲 / 分卷 / 章节
- 编号是否清晰
- 事件作用是否写明
- 已实现、遗漏、弱化、偏离的情节是否能逐项核对

## 十八、网文留存与辨识度

- 是否能用一句话讲清这个人物为什么值得被记住
- 首秀三章内是否已经兑现至少 `1` 类持续供血（专业碾压 / 情绪回报 / 关系拉扯 / 危险推进 / 反差幽默 / 反击爽点）
- 是否至少有 1 个可截图 / 可转述 / 可站队的态度点、反应点或关系点
- 这个人物主要的持续供血类型是否明确，而不是只能靠设定说明续命
- 读者为什么会继续跟随他：认同 / 同情 / 好奇 / 敬佩中是否至少成立两项
- 若人物离线 3 卷以上再回来，是否仍然具备足够的回归压力

## 十九、镜像对手与独立承重

- 是否存在与其价值观、方法论、欲望或恐惧形成镜像 / 反镜像的重要人物
- 这层镜像是否会在中后段持续制造选择压力、关系反噬或立场冲突
- 重要配角是否具备独立番外 / 单元叙事承重能力，而不只是主角挂件
