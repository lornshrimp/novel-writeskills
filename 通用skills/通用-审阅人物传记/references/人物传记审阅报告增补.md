# 人物传记审阅报告增补

## 1. 报告里必须写重的部分

### 1.1 可演出性与可写性评估

不能只说“人物挺立体”，必须明确回答：

- 三秒立人是否成立
- 行动驱动是否够硬
- 缺陷是否能反复制造麻烦与代价
- 关系网络是否真能推动事件
- 人物是否绑定了题材机制抓手

### 1.2 人物标签与成长弧

每份报告最好明确指出：

- 这个人物最能被读者记住的外貌 / 动作 / 语言钉子是什么
- 成长弧四步是否成立：初始缺陷 → 触发事件 → 认知转变 → 行为体现
- 这种成长是否会在各卷持续，而不是只在设定文档里喊口号

### 1.3 可复用写作素材库

报告不能停在分析层，必须至少补齐 `4` 类可直接贴回正文的素材：

- 口癖 / 句式库
- 标志动作库
- 雷点 / 底线 / 可被激怒点
- 秘密与把柄
- （可选）道具与场景绑定

### 1.4 大纲完整性与联动潜力

优先评价：

- 这个人物在大纲里承诺过什么
- 哪些情节已经实现，哪些缺失，哪些被弱化，哪些已经偏离
- 若此人物继续推进，和哪些章节 / 单元案 / 关键人物最有联动价值

### 1.5 最小回写建议

推荐句式：

- 在 `【动机段】` 补写 `【具体欲望 + 具体代价】`，让人物选择不再漂浮。
- 在 `【缺陷段】` 增加 `【一次真实反噬】`，让缺陷从标签变成剧情发动机。
- 在 `【写作抓手】` 补齐 `【口头禅 / 动作模板】`，让人物真正能写进正文。
- 在 `【关系网】` 增补 `【资源 / 制衡 / 把柄 / 情感债】`，让关系不再只是名单。
- 在 `【秘密】` 中把 `【抽象隐情】` 改成 `【能推动章节的具体把柄】`，提升承重力。

### 1.6 冲击 9.2+ 综合评分的优化思路

在完成基础修复建议后，必须继续回答：

- 若要把这个人物推到 `9.2+`，最该优先增强的是哪三处
- 如何同时提升：人物复杂性、成长弧完整度、标签落地度、章节联动潜力与读者回归压力
- 至少给出 `1–3` 处可直接写回传记的条目级改写

### 1.7 登场镜头与台词交付

每份报告至少要补：

- `1` 段可直接贴回正文的“登场镜头”（约 `120–250` 字，含动作 + 台词 + 对抗微火花）
- `3` 句可直接复用的台词（分别用于：压迫 / 试探 / 翻脸或服软）

### 1.8 作者向依据与正文向落地双交付

报告允许说明作者侧判断依据，但不能只停在作者脑内：

- 每条关键建议都应同时给出“为什么要这么补”与“补完后怎么写进正文”
- 不得只写抽象结论，不给镜头、动作、台词或条目级补丁

### 1.9 首次出场与前三章留存验证

每份报告最好明确指出：

- 这个人物第一次亮相是否已经落在最能体现其价值 / 压力 / 反差的熟悉情境里
- 前三章内是否已经给出一次有效回报、误读、反击或关系拉扯
- 首秀是否留下了可截图、可转述、可催更的态度句、动作点或关系钩子

### 1.10 人物—情境—观点合拍度

每份报告最好明确指出：

- 这个人物是否像目标读者熟悉的人、怕成为的人、或想跟随的人
- 是否选择了最能凸显其价值与问题的日常情境，而不是“哪里都能演”的空舞台
- 人物的情绪站位和价值表达，是否符合其身份与目标读者最容易产生共鸣或争论的现实想法

### 1.11 竞对差距量化卡（必须写重）

报告里不得只写“人物不如竞对立得住”；默认按统一母版顺序写：

1. `Top 10 候选池收束`
2. `3–5` 本主压制样本四层深拆速览
3. `统一量化卡`
4. `汇总判断 / 最危险落后项 / 优先反超位 / 盲区说明`

其中“统一量化卡”固定列必须保留：

- 我方当前分（0–5）
- 竞对均值分（0–5）
- 分差（我方-竞对）
- 关键证据
- 72 小时补丁
- 不应照搬点

人物传记层的核心维度默认量化：

- 首秀抓力与前三章回报
- 一句话记忆点
- 动机—代价闭环
- 关系网事件生成能力
- 缺陷反噬与成长弧
- 回归压力与持续供血

每项都要写：我方当前分、竞对均值分、分差、关键证据、72 小时补丁、不应照搬点。

并在量化卡后补齐：

- 竞对总差距判断：领先 / 持平 / 落后
- 当前最危险的 `2` 个落后项
- 最需要优先反超的 `2` 个维度
- 阻断点 / 判断盲区

### 1.12 高分封顶规则（必须写明）

- 若竞对量化卡未完成：综合评分默认不得高于 `7.0/10`
- 若竞对项低于 `7.0/10`：综合评分默认不得高于 `8.4/10`
- 若竞对项低于 `6.0/10`：综合评分默认不得高于 `7.4/10`
- 若缺少 `Top 10 候选池`、`3–5` 本主压制样本四层深拆速览或统一量化卡任一固定列，视为竞对模块未完成，综合评分默认不得高于 `7.0/10`
- 若首秀抓力、动机代价、关系网事件功能三项里有 `2` 项以上明显弱于竞对，不得写成“人物已立稳”
