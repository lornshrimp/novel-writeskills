# 人物传记问题分级与回写建议

## 高优先问题

- 核心人物没有明确代价
- 能力明显万能
- 关系网只有“认识谁”，没有事件功能
- 心理弧线只有感悟，没有触发事件
- 写作抓手太空，无法落正文
- 缺陷不进入关键选择，只停留在标签层
- 核心秘密完全不可验证，无法进入剧情链
- 核心人物没有一句话记忆点，离线后几乎没有回归压力
- 主角与重要对手没有镜像层，冲突只能靠站位硬推
- 三秒立人失败，读者很难在首轮出场记住该人物
- 首秀落点错误，前三章里既没有有效回报也没有可催更钩子
- 行动驱动不足，只能靠剧情推着走
- 人物、情境、观点不合拍，像“有设定的人”而不像“会在这个场景里活的人”
- 可复用写作素材库缺失，导致作者知道人设却写不出人
- 持续供血类型不明确，人物只能靠设定说明硬撑存在感
- 大纲中承诺的人物成长或关键事件没有在传记中得到支撑

## 报告里必须写重的部分

- 一句话定义是否成立
- 首秀抓力与前三章供血是否成立
- 人物—情境—观点是否合拍
- 可演出性与可写性是否成立
- 动机—代价是否成环
- 写作抓手是否能直接拿去写
- 可复用素材库是否足够硬
- 读者为什么会继续跟这个人物
- 冷线回归时为什么还值得重新拉热
- 大纲承诺的人物功能与成长是否真的落在传记里
- 最小回写建议是否指向具体段位

## 快速判定口径

- `读者联结与辨识度` ≤ 6 且该人物属于主角 / 重要对手：至少按 P1 处理
- `动机与代价` ≤ 5：至少按 P1 处理
- `关系网事件生成能力` ≤ 5：若该人物承担关键剧情功能，至少按 P1 处理
- `冷线回归压力` ≤ 5：若该人物属于中后段必回角色，至少按 P1 处理
- `可演出性 / 可写性` ≤ 6：若该人物属于主角 / 高戏份配角，至少按 P1 处理
- `首秀抓力与追读供血` ≤ 6：若该人物属于主角 / 关键配角，至少按 P1 处理
- `大纲联动与可回指性` ≤ 5：若该人物承担关键推进作用，至少按 P1 处理

## 建议句式

- 在 `【动机段】` 补写 `【具体代价】`，让后续行动不再零成本。
- 在 `【能力边界】` 明确 `【不能用的场景】`，防止正文万能化。
- 在 `【首秀段】` 增加 `【熟悉情境里的第一冲突 + 第一回报】`，让人物不是履历板而是开场就能抓人。
- 在 `【情境段】` 把人物放回 `【最能凸显其价值 / 压力 / 反差的日常场景】`，让读者更容易代入和站队。
- 在 `【写作抓手】` 增补 `【动作 / 台词模板】`，让人物从标签变成镜头。
- 在 `【秘密】` 中把 `【抽象隐情】` 改为 `【可验证 / 可落剧情的秘密】`，提升剧情承重。
- 在 `【缺陷段】` 增加 `【一次真实反噬】`，让人物不再只是“口头有缺点”。
- 在 `【关系网】` 补写 `【镜像对手 / 关键站队对象】`，让冲突不再只靠外部事件硬推。
- 在 `【一句话定义】` 中压缩出 `【记忆点】`，让编辑和读者能更快记住这个人物。
- 在 `【网文留存】` 中写清 `【持续供血类型】`，让后续章节知道该靠什么继续喂读者。
- 在 `【标签与成长弧】` 补齐 `【初始缺陷 → 触发事件 → 认知转变 → 行为体现】`，防止成长线只剩结论。
- 在 `【素材库】` 增补 `【口癖 / 动作 / 雷点 / 把柄】`，让作者真正有材料可写。

## 来源层速判

- `人物结构层`：公开身份、隐秘角色、动机、代价、边界、关系主轴、镜像对手
- `动态状态层`：位置、情绪、已知信息、持有物、身体状态、章节行为记录、当前弧阶段
- `正文抓手层`：登场镜头、语言指纹、可复用素材库、动作模板、台词模板、首秀钩子

默认优先级：先修人物结构层，再修动态状态层，最后补正文抓手层；不要先润饰台词，再回头发现动机和代价根本没立住。

## Green / Yellow / Red 入库判定速记

- `Green`：一句话定义清楚、动机代价成环、关系网能生事、动态状态可维护、素材库可直接回贴正文。
- `Yellow`：主干可用，但首秀钩子、动态状态、章节行为记录、语言指纹情绪变体或回归压力仍弱。
- `Red`：四大栏位缺失、能力万能、代价缺位、关系网无事件功能、首秀发空、离线后无回归压力。

## 9.2+ 升级时必须补出的交付物

- `1–3` 处可直接写回传记的条目级补丁
- `1` 段可直接贴回正文的登场镜头（约 `120–250` 字）
- `3` 句可直接复用的台词（压迫 / 试探 / 翻脸或服软）
