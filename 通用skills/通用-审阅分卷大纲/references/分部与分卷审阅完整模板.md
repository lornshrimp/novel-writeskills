# 分部与分卷审阅完整模板（通用骨架）

适用于：分部大纲、分卷大纲、章清单、单元案执行版。

## 通用层与题材层的拆分原则（强制）

- 本文件只承载跨题材复用的**分部 / 分卷审阅通用骨架**。
- 各题材 Skill 必须在此骨架上补写自己的“题材吸引力与差异化核查”“节奏/场景/平台专项段”“题材专项评分项”。
- 题材 Skill **不得删减**本模板的通用主骨架，只能在题材专项章节中补写本题材裁判口径。
- 若对象存在上游分配来源，报告必须先写“上游分配已落地 / 缺失 / 悬空”的诊断，再进入节奏与钩子判断。

## 落盘位置与文件名格式（强制）

- 分部大纲、分卷大纲审阅报告直接位于 `审阅意见/` 根目录。
- 单元案审阅报告如由本 Skill 输出，也默认沿用 `审阅意见/` 根目录。
- 文件名示例：
  - `审阅意见/分部审阅报告_[名称或编号]_[审阅日期].md`
  - `审阅意见/分卷审阅报告_[名称或编号]_[审阅日期].md`
  - `审阅意见/单元案审阅报告_[名称或编号]_[审阅日期].md`

## 🧰 工程结构与可执行性核查（作者向｜强制｜P0）
- [ ] **上游分配全量落地（P0）**：是否把上一级已分配的主线、支线、伏笔、回收、人物推进、阶段回报切实落到本层，而不是挂名承接。
- [ ] **可拆章性（P0）**：是否能把每一章/每一场的目标、动作、阻力、回报、代价拆成可执行清单，而不是抽象口号。
- [ ] **冲突落点可见（P0）**：关键冲突是否都有可拍镜头、可留痕工件或可验证后果；若仅有概念无载体，判为不可写风险。
- [ ] **证据链/程序链/因果链可跑通（P0）**：反制抓手是否前置铺垫、可留痕、可回收；关键转折是否有记录、编号、第三方节点或明确规则后果。
- [ ] **写作导航性（P0）**：是否存在作者可直接复用的结构化交付物（如：章清单、卷级启动表、章末钩子策略表、伏笔-回收表、人物职责表）。

## 🎨 文学性与新颖性（加分项｜P2/P3）
- [ ] 文学表达的张力：是否有可记忆的场景意象/一句话钉子，但不过度代替卷纲施工。
- [ ] 结构与机制新意：是否避免同质化套路，能一句话讲清“本卷/本部独有机制卖点”。

---

## 🔥 网文卷级留存与平台节奏复核（强制）
- [ ] **前 5 章追读链**：前 `1–5` 章是否形成“每章回应一个旧问题 + 新增一个更贵的问题”的连续追读链，而不是只靠第一章抓眼。
- [ ] **卷中回报密度**：卷中是否持续提供信息推进、关系变位、资源到账、规则反噬、身份修正等真实回报，而不是连续空跑。
- [ ] **新鲜冲突比例**：本卷回报中，是否至少有 `30%` 以上属于新冲突导入或结构性改写，而不是只验证已知信息。
- [ ] **平台节拍适配**：当前卷节奏、章长、钩子密度更偏哪类平台；若与目标平台错配，是否已显式指出。

- [ ] 主线/支线比例：主线推进是否被支线长期稀释；支线是否服务主线或人物弧，而不是单纯充字数。
- [ ] 冲突金字塔：个人困境→关系网阻力→制度/机制压迫是否逐层加码；是否存在对抗“越写越轻”的战力崩坏。
- [ ] 悬念链与回收：伏笔是否留痕可追溯；回收是否靠证据链/程序链/因果链完成；是否存在坑填不上或无声蒸发。

## 使用说明（强制）

- 若当前任务是题材 Skill 执行，必须把题材专属检查项、题材专项评分项与题材化结论重点补进上面的占位章节。
- 若题材 Skill 只输出摘要版或删掉其中任一主章节，视为未完成保真迁移。
- 若未先写“上游分配落地诊断”就直接评价节奏与钩子，视为流程违规。
- 若未单独判断前 5 章追读链、卷中回报密度与卷末续购强度，视为未完成网文卷级审阅。
- 若未写明“借鉴哪本竞对的什么点、弱于哪本竞对的什么点、本轮先改什么”，视为竞对接入未完成。
