# 分部与分卷审阅完整模板（通用骨架）

适用于：分部大纲、分卷大纲、章清单、单元案执行版。

## 1. 通用层与题材层的拆分原则（强制）

- 本文件只承载跨题材复用的**分部 / 分卷审阅通用骨架**。
- 各题材 Skill 必须在此骨架上补写自己的“题材吸引力与差异化核查”“节奏/场景/平台专项段”“题材专项评分项”。
- 题材 Skill **不得删减**本模板的通用主骨架，只能在题材专项章节中补写本题材裁判口径。
- 若对象存在上游分配来源，报告必须先写“上游分配已落地 / 缺失 / 悬空”的诊断，再进入节奏与钩子判断。

## 2. 落盘位置与文件名格式（强制）

- 分部大纲、分卷大纲审阅报告直接位于 `审阅意见/` 根目录。
- 单元案审阅报告如由本 Skill 输出，也默认沿用 `审阅意见/` 根目录。
- 文件名示例：
  - `审阅意见/分部审阅报告_[名称或编号]_[审阅日期].md`
  - `审阅意见/分卷审阅报告_[名称或编号]_[审阅日期].md`
  - `审阅意见/单元案审阅报告_[名称或编号]_[审阅日期].md`

## 8. ⚡ 快审门禁总表（快节奏 + 强冲突｜可复制｜强制）

### 8.1 快节奏门禁

- [ ] 黄金三章成立：卷首前三章形成“麻烦入场 → 正面对抗 → 首次升级/回报”
- [ ] 卷中防失温：无连续 `4+` 章等效“无新变量/无回报/无关系变化”
- [ ] 章功能轮换：钩子/蓄力/爆发/回报/转承分布合理，非单功能堆叠
- [ ] 卷末续购：卷末缺口更贵且下一卷接棒对象明确

### 8.2 强冲突门禁

- [ ] 冲突三维压迫：外部压力、关系/身份压力、价值选择压力至少覆盖两项
- [ ] 升级链可追踪：每章可回答“比上章升级了什么”
- [ ] 代价落点清晰：本卷至少 `2-3` 个可验证不可逆代价
- [ ] 关系位移成立：关键关系至少一次可感知翻面/重排

### 8.3 快审判定（分卷层）

- [ ] 任一门禁项 `Fail` → 至少判 `P1`
- [ ] 若“卷中防失温”或“升级链可追踪”失败 → 综合评分不得高档
- [ ] 必须输出最小回炉动作：
