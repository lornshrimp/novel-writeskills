# 分部与分卷审阅专属检查清单

## 一、先判上游分配是否全量落地

- 是否已列出本层应承接的上游内容
- 这些内容是否有明确落点、执行方式与可拆章程度
- 是否存在遗漏、缩水、悬空与只挂名不落实

## 二、分层检查口径

### 分部层

- 是否正确承接总纲层分配
- 是否能指导后续分卷落地

### 分卷层

- 是否把上一级分配内容落到卷概览、章清单、工件与章节规划
- 5/15 节奏是否标清
- 钩子类型是否有分布策略

### 单元案层

- 启动 → 砸实 → 对抗 → 翻案 → 余波 是否完整
- 是否既能阶段闭环，又能给主线供血

## 三、常见严重问题

- 没有 5/15 节奏布点
- 章中没有回报，章末只有硬钩
- 上游已分配内容大量缺失、悬空或被偷换成一句摘要
- 单元案闭环了，但没反哺主线
