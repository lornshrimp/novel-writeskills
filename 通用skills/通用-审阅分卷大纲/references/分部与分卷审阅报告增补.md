# 分部与分卷审阅报告增补

## 报告前置要求

- 先写清：上一级分配给本层的内容，哪些已落地，哪些缺失，哪些只是挂名未落实。

## 报告里必须写重的部分

### 1. 可执行性与可拆章性

- 作者是否能直接据此拆章
- 哪些段落仍是评论大纲，不是执行大纲

### 2. 节奏与钩子策略

- 小高潮 / 大高潮的位置
- 钩子类型是否轮换
- 哪几章最容易“章首猛但中段塌”
- 建议补什么类型的中段回报

### 3. 前五章留存、卷中回报与续购诊断

- 前 5 章是否形成连续追读链
- 卷中最容易失温的章组在哪里，为什么
- 单元案是否既能独立成立，又能反哺主线
- 卷末是否留下更贵的新缺口，能推着读者进入下一卷

### 4. 平台节拍与拆章适配

- 当前卷更偏哪类平台成品，为什么
- 卷长 / 章长 / 钩子密度与目标平台是否匹配
- 是否存在“结构能看，但一拆章就失温”的风险

### 5. 具体修改建议

建议用“插入位置 + 章组补丁 + 预期收益”的格式，例如：

- 在【某章组】增加【中段回报】，避免本段空转。
- 将【当前钩子】改为【转折 / 揭露 / 危机】，避免连续同型疲劳。
- 在【本部 / 本卷承接清单】补列【缺失的上游分配项】，并落到【对应章组 / 工件】。
- 在【卷末】补一个【更贵的新缺口 / 跨卷后果】，提升下一卷续购驱动力。
- 在【前 5 章】补一条【回应旧问题 + 新增更危险问题】的追读链，避免只靠首章独木支撑。

### 6. 竞对差距量化卡（必须写重）

报告里不得只写“这卷没有竞对强”；默认按统一母版顺序写：

1. `Top 10 候选池收束`
2. `3–5` 本主压制样本四层深拆速览
3. `统一量化卡`
4. `汇总判断 / 最危险落后项 / 优先反超位 / 盲区说明`

其中“统一量化卡”固定列必须保留：

- 我方当前分（0–5）
- 竞对均值分（0–5）
- 分差（我方-竞对）
- 关键证据
- 72 小时补丁
- 不应照搬点

分卷层的核心维度默认量化：

- 前 5 章追读链
- 卷中回报密度
- 钩子类型轮换
- 卷末续购力度
- 卖点落点强度
- 单元案反哺主线效率

每项都要写：我方当前分、竞对均值分、分差、关键证据、72 小时补丁、不应照搬点。

并在量化卡后补齐：

- 竞对总差距判断：领先 / 持平 / 落后
- 当前最危险的 `2` 个落后项
- 最需要优先反超的 `2` 个维度
- 阻断点 / 判断盲区

### 7. 高分封顶规则（必须写明）

- 若竞对量化卡未完成：综合评分默认不得高于 `7.0/10`
- 若竞对项低于 `7.0/10`：综合评分默认不得高于 `8.4/10`
- 若竞对项低于 `6.0/10`：综合评分默认不得高于 `7.4/10`
- 若缺少 `Top 10 候选池`、`3–5` 本主压制样本四层深拆速览或统一量化卡任一固定列，视为竞对模块未完成，综合评分默认不得高于 `7.0/10`
- 若前 5 章留存、卷中回报、卷末续购三项里有 `2` 项以上明显弱于竞对，不得写成“可以直接放行正文”
