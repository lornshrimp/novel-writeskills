# 分部与分卷审阅报告增补

## 报告前置要求

- 先写清：上一级分配给本层的内容，哪些已落地，哪些缺失，哪些只是挂名未落实。

## 报告里必须写重的部分

### 1. 可执行性与可拆章性

- 作者是否能直接据此拆章
- 哪些段落仍是评论大纲，不是执行大纲

### 2. 节奏与钩子策略

- 小高潮 / 大高潮的位置
- 钩子类型是否轮换
- 哪几章最容易“章首猛但中段塌”
- 建议补什么类型的中段回报

### 3. 具体修改建议

建议用“插入位置 + 章组补丁 + 预期收益”的格式，例如：

- 在【某章组】增加【中段回报】，避免本段空转。
- 将【当前钩子】改为【转折 / 揭露 / 危机】，避免连续同型疲劳。
- 在【本部 / 本卷承接清单】补列【缺失的上游分配项】，并落到【对应章组 / 工件】。
