# 人物传记最小结构模板

## 使用分层建议

- **核心人物**：默认使用下方完整骨架，不建议只保留摘要版。
- **重要人物**：默认保留完整骨架中的大多数模块，至少不得删掉写作抓手、基本信息、完整心理弧线与三级索引。
- **次要人物**：可压缩“详细内容”与“补强模块”，但 `四大基础栏位`、`写作抓手交付`、`基本信息`、`完整心理弧线`、`三级大纲情节对应索引` 仍应保留最小可用信息。

## 推荐抬头格式

```markdown
# 人物名（英文名 / 拼音 / 代号可选）——人物传记

> **人物类型**：核心角色 / 重要角色 / 次要角色
> **心理弧线步数**：30–40步 / 20–30步 / 10–20步
> **传记版本**：v1.0
> **最后更新**：YYYY-MM-DD
```

- 核心 / 重要人物默认显式标注这四项；次要人物至少保留 `人物类型` 与 `最后更新`。
- 若人物存在正式英文名、旧名、案卷代号、网名、江湖称呼，可在标题或“姓名”小节中并列写明。
- `心理弧线步数` 不是装饰信息，而是后文结构完整度与索引密度的裁判依据。

## 默认完整骨架

以下骨架默认面向“可落盘、可回指、可持续补丁”的作者侧完整版人物传记；次要人物可按后文“模块强弱分层”压缩。

## 模块强弱分层

- **核心人物**：默认完整保留 `0–11`，尤其不能缺 `写作抓手交付`、`完整心理弧线`、`身份认同危机`、`内在冲突体系`、`价值观演进`、`本轮修订 / 自检`。
- **重要人物**：允许压缩 `3.4 主要事迹`、`5–8 补强模块` 的展开篇幅，但仍应保留模块入口，避免后续无法追加。
- **次要人物**：可删减 `5–8` 的大段分析，也可把 `3.4 / 3.7 / 3.8 / 10 / 11` 压到简版，但不得把“抓手、基本信息、弧线、索引”直接省没。

## 四大基础栏位说明

- 上述“背景、性格、外貌、其它”是模板级强制总览，不替代后续细分章节；后续章节是对这四类信息的展开。
- 若沿用现有详细章节直接回写人物传记，也必须确保四类都能被快速定位，不能只写背景和设定、把性格或外貌写成一句带过。
- “其它”不是兜底空栏，默认至少应覆盖能力边界、关系网、写作抓手、秘密 / 爆点、心理弧线或索引映射中的关键内容。

## 写作抓手最低标准

- 口头禅 / 句式要稳定、可复用，最好能体现人物的判断方式、关系姿态或职业习惯。
- 习惯动作与微反应不要只写“皱眉、叹气”这类公版动作；应尽量写成能与压力、说谎、得意、失控、回稳等场景绑定的动作。
- 触发器必须具体，且出现后能改变语气、动作、决策或身体反应。
- 底线 / 开关不能只写抽象价值判断，要写成一旦触发会立刻换挡的行为阈值。
- 留痕偏好不只写“喜欢记笔记”，而要写清他 / 她信任什么载体、为什么信、会如何固定证据或情绪痕迹。
- 反制模板要尽量写成“先做什么 → 再补什么链 → 最怕哪一步被反杀”的动作链，而不是一句“会反击”。

## 基本信息与细节库要求

- 基本信息不只是身份证摘要，还要解释人物在社会结构里的位置、走到此刻的路径、与原生关系的真实拉扯。
- `职业 / 身份轨迹` 建议写成时间段式或阶段式，方便和总纲、卷纲、事件线对表。
- `可复盘细节库` 优先收纳能在正文里反复回指的伤疤、物件、签字习惯、固定时间、固定场所、固定动作、固定口头语等。
- `历史地位 / 结构功能` 要说明这个人物为什么值得长期维护：是主视角承担者、价值冲突中枢、主题载体、制度遗产，还是某一卷的供血角色。

## 详细内容与心理弧线要求

- `详细内容` 不应只是简历扩写，而要解释人物怎么被塑形、怎样误判、为什么会在关键节点做出那个选择。
- 核心 / 重要人物默认要把能力 / 专业 / 资源与代价绑定书写，避免只剩“很强”“很能打”“很聪明”的空标签。
- `完整心理弧线` 建议显式标注总步数；每一步最好至少包含 **触发事件、变化结果、可回指载体** 三项。
- 关键步骤最好能落到“年龄 / 时间段 / 事件编号 / 物件 / 地点 / 关系节点”中的至少两类，方便后续索引与审阅。
- 若人物承担长线主线或高压规则冲突，优先补 `身份认同危机`、`内在冲突体系`、`情感变化链条`、`价值观演进` 四块，它们通常就是人物后劲所在。

## 作者侧定位提醒

- 这是作者侧的人物裁判源 / 调用手册，不是读者向人物小传。
- 目标是可写、可审、可回指、可补丁，而不是只在第一次阅读时“很酷”。
- 允许带少量情绪叙述，但必须服务于可调用的抓手、可验证的后果与可继续扩写的结构入口。

## 推荐补强项

- 身份认同危机
- 内在冲突体系
- 规则 / 现实受压点与反制
- 情感变化链条
- 价值观演进
- 公众形象 vs 自我认知错位
- 外部目标 vs 内部需求差值
- 象征物 / 记忆锚点 / 空间锚点
- 可复盘细节库
