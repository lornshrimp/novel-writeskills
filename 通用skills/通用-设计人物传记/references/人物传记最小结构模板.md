## 写作抓手最低标准

- 口头禅 / 句式要稳定、可复用
- 习惯动作要能在压力场景中重复出现
- 触发器必须具体，且出现后能改变语气或动作
- 压力反应不能只写“会崩”“会怒”，要写成可拍出来的身体 / 语言 / 决策变化
- 安慰机制不能只写“会自我调整”，要写清靠什么物件、空间、动作或人来回稳
- 留痕偏好要解释其叙事用途
- 反制模板要写成“先做什么 → 再补什么链 → 最怕哪一步被反杀”

## 作者侧定位提醒

- 这是作者侧的人物裁判源 / 调用手册
- 目标是可写、可审、可回指，而不是读者向人物小传
- 允许带少量情绪叙述，但必须服务于可调用的抓手与可验证后果

## 外层识别与时间线补充提醒

- 外层识别至少要落到“外貌 / 声音 / 动作 / 物件”四类中的两类，避免人物只能靠标签记忆。
- 成长时间线不是履历流水账，而是要写清哪些事件塑造了人物今天的伤口、偏见、执念与判断方式。
- 若时间线里有年龄、升学、入职、退场、创伤、离散等关键节点，优先写成便于和总纲 / 分卷互相校对的硬锚点。

## 偏好与厌恶补充提醒

- 人物喜欢什么，不只是口味问题，往往会暴露他的秩序感、羞耻点、阶层痕迹或情感缺口。
- 人物厌恶什么，也不只是“讨厌”，更可能指向旧伤、价值观或恐惧来源。
- 若人物一切偏好都太正确、太中性、太无害，写起来通常不够像活人。
- 重要人物最好至少有一个“看似小、其实很说明问题”的偏好锚点，例如固定食物、固定歌、固定去处、固定收藏或固定回避物。
- 地域、家庭、成长环境留下的习惯，优先写成能回贴正文的抓手，而不是只写籍贯标签。

## 人物抓手补充提醒

- 优先把人物写成“有职业 / 有缺陷 / 有私人动机”的可调用角色，而不是纯标签人设。
- 执念、痛点、秘密三项，至少应有一项能在卷内早期直接触发冲突。
- 人物初次出场时，尽量让外貌描写同时承担性格、态度或关系张力，不要只罗列五官。
- 口头禅、动作、留痕与情绪反应最好能彼此咬合，形成稳定辨识度。
- 公众形象与自我认知最好存在一个真实差值：别人眼里的他 / 她，与他 / 她自认的自己并不完全相同。
- 外部目标与内部需求不要写成同一句话；前者决定人物现在追什么，后者决定人物为什么总在关键时刻失手或卡住。
- 重要人物至少要有一个“读者本来以为会这样，但实际上不是这样”的反模式化点。

## 推荐补强项

- 身份认同危机
- 内在冲突体系
- 规则异化受压点与反制
- 情感变化链条
- 价值观演进
- 公众形象 vs 自我认知错位
- 外部目标 vs 内部需求差值
- 象征物 / 记忆锚点 / 空间锚点
- 可复盘细节库
