# 腾讯专栏人物行动显性格增补（2026-05-04）

来源：子良评书第五期——作品的灵魂是角色（起点编辑子良 / 写作技巧专栏）

---

## P0 规则：角色靠「做了什么」显性格

### 核心原则
- 角色性格必须靠**行动**显现，而非说明性描写或旁白标注
- "没做坏事"≠善良；**"拒绝作恶 + 积极做好事"** 才能凸显善良
- 外貌描写只在第一印象阶段有效，故事展开后读者全靠性格认人
- 说明性介绍（"他是个善良的人"）不能给读者留下深刻印象

### 正确写法示例（行动展现性格）
| 性格目标 | 错误写法 | 正确写法 |
|---|---|---|
| 守诺 | "张三是个很守承诺的人" | 张三无敌九州后，仍赴青牛之约，放弃滔天权势孤身前往绝境 |
| 道义 | "许乐有正义感" | 许乐放弃金光大道，为死去的数千无辜民众讨公道，舍身杀幕后黑手 |
| 真性情 | "周咚咚很可爱" | 周咚咚吃饭时两耳不闻窗外事，如处结界之中 |

### 塑造善良角色的双路径
1. **做好事**（正向行动）：救治流浪猫狗、帮助陌生人
2. **拒绝作恶**（负向选择）：穷困潦倒时仍拒绝坑蒙拐骗

## P0 规则：起点男频对人物的基本要求

> 以下为起点男频读者对人物的最低容忍线，违反会直接导致弃书

### 主角
- 有敌人时：**必须强大**（至少心理强大，不能持续懦弱）
- 没有明确敌人时：**至少不懦弱**

### 主角同阵营角色
- **不能拖后腿**
- 专门给主角惹麻烦的女性配角，在起点男频极度不受欢迎（直接劝退）

### 主角敌对阵营角色
- 除搞笑角色外：**不能是蠢货**（反派智商要有底线）

## 审阅触发规则

设计人物传记时，每个主要人物检查：
- [ ] 该人物的核心性格是否有至少1-2处具体行动证明？
- [ ] 是否避免了纯描述型性格标注（"他勇敢、善良、机智"）？
- [ ] 如果是起点男频：主角有没有出现持续懦弱/被配角拖累的设计？
