# 心理弧线与索引规则

## 心理弧线不是情绪列表

合格的心理弧线必须满足：

- 每一步都有触发事件
- 每一步都能在正文中被看见
- 关键步骤尽量能回指到某个载体、物件、称呼、地点或事件
- 弧线推进会改变关系、选择与代价
- 每个阶段最好都写清：人物会失去什么、获得什么、哪个误信或偏见被击穿

## 推荐步数

- 核心人物：30–40 步
- 重要人物：20–30 步
- 次要人物：10–20 步

## 四阶段口径

1. 原始状态
2. 冲击觉醒
3. 深度探索
4. 觉醒重生

补充建议：不要只写四个阶段标题；至少要补出每一阶段的关键触发事件、阶段后果与对应锚点。

## 索引规则

人物传记末尾应保留索引字段：

- 来源层级：总纲 / 分部 / 分卷 / 章节
- 编号
- 时间节点
- 事件描述
- 心理阶段 / 步骤号
- 本事件对人物的后果

## 常见失误

- 只写感悟，不写触发事件
- 索引只有编号，没有事件作用
- 人物成长太干净，没有回摆、失败与误判
- 只写四个大标题，不写每阶段失去 / 获得 / 被击穿了什么
