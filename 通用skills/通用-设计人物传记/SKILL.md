# 通用-设计人物传记

人物传记应写成作者可长期调用的裁判源，而不是漂亮但难落地的人设简介。

## 常见触发词 / 用户说法速查

- 帮我做人物传记
- 重写这个角色档案
- 这个人物还不够立
- 给角色补动机和代价
- 帮我把人物关系网整理出来

## 何时使用

- 新建某个角色的人物传记
- 把旧传记重写为可写、可审、可对齐大纲的版本
- 给人物补充写作抓手、关系网、心理弧线与索引映射

## 不适用情形

- 只做人物传记审阅，不改稿：优先使用 `通用-审阅人物传记`。
- 只写正文中的单场人物对话或人物表现，不建人物裁判源。

## 默认执行顺序

1. 先确认目标人物文件与写入策略。
2. 判断人物层级：核心 / 重要 / 次要。
3. 先按模板搭建外层识别、成长时间线、动机、价值观、边界、关系、抓手、秘密、弧线、索引。
4. 再补公众形象 vs 自我认知、外部目标 vs 内部需求、压力反应与锚点物。
5. 最后走质量控制与验证流程。

## 人物硬要求（新增）

- 人物至少要同时具备：**真实短板**、**务实动机**、**一个会反复拉扯他的关系或执念**。
- 传记里默认要写清角色想要什么、为什么现在必须要、为了得到它愿意失去什么。
- 主角与重要角色都应有成长弧光：不是“会不会变强”这么空，而是价值判断、关系处理、行动方式或自我认知会发生什么可写变化。
- 角色魅力不能只靠标签、设定或外形；必须落到口头禅、动作习惯、压力反应、做错事后的补救方式等可直接回贴正文的抓手。
- “强”不是完美无缺；越是承担长线供血的角色，越要避免无短板、无代价、无反噬的万能写法。
- 核心 / 重要人物默认还要写清：**关键经历时间线**、**公众形象 vs 自我认知**、**当前外部目标 vs 内部需求**、**价值观 / 误信**。
- 重要人物最好至少有一个**象征物 / 记忆锚点 / 空间锚点**，方便在正文里做回指、反差与情绪唤起。

## 最低交付

- 能写清外层识别、动机、边界、关系、抓手、弧线与索引中的核心字段。
- 核心 / 重要人物至少要补出成长时间线、公众形象 vs 自我认知、外部目标 vs 内部需求、价值观 / 误信中的关键字段。
- 能说明角色为什么值得继续写、继续审、继续回指。
- 能作为正文与大纲的可调用裁判源使用。

## 硬规则

- 传记不是宣传文案，禁止只写抽象褒贬。
- 人物必须有动机，也必须有代价。
- 能力不能万能，关系不能只是“认识谁”。
- 写作抓手必须可直接贴回正文使用。
- 禁止把成长时间线写成流水履历；必须写清哪些事件真的塑造了人物现在的判断、伤口与偏见。
- 禁止只写“别人眼中的他 / 她”或只写“自我独白”；公众形象与自我认知的差值要能制造戏剧。
- 除非用户明确要求只脑暴，不得只在聊天里给传记长文而不写入对应文件。
- 主角至少要写清执念、缺陷、秘密三者中的两项；反派必须写清可落地动机来源。

## 与其他 Skill / Prompt 的边界

- 本 Skill 负责人物传记本体的设计与重写。
- 若用户只要审阅和问题分级，应优先使用 `通用-审阅人物传记`。
