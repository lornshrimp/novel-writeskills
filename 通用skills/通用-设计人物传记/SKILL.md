# 通用-设计人物传记

人物传记应写成作者可长期调用的裁判源，而不是漂亮但难落地的人设简介。

## 常见触发词 / 用户说法速查

- 帮我做人物传记
- 重写这个角色档案
- 这个人物还不够立
- 给角色补动机和代价
- 帮我把人物关系网整理出来

## 何时使用

- 新建某个角色的人物传记
- 把旧传记重写为可写、可审、可对齐大纲的版本
- 给人物补充写作抓手、关系网、心理弧线与索引映射

## 不适用情形

- 只做人物传记审阅，不改稿：优先使用 `通用-审阅人物传记`。
- 只写正文中的单场人物对话或人物表现，不建人物裁判源。

## 题材竞对分析接入要求（强制）

- 若当前任务服务某个题材，且该题材目录下的 `竞对分析/` 已存在与当前项目、当前平台或当前赛道相关的竞对分析报告，设计人物传记前必须先读取并纳入本轮判断。
- 若目标平台为起点中文网，或未注明平台但默认按起点口径处理，且当前题材目录下尚无可用竞对主报告，开始人物传记设计前应优先调用 `通用-分析竞对作品`，至少补齐 **Top10 候选池、`3–5` 本主样本与四层深拆速览**；若任务限制导致本轮无法补齐，必须显眼记录盲区，并降低本轮人物传记成熟度判断。
- 不得把这些报告只当成背景资料；人物传记必须显式说明本轮准备在哪些维度追平、反超竞对，以及哪些主样本暴露出的旧坑必须避开。
- 人物传记的默认目标不是“资料更全更细”而已，而是让当前作品至少在**主角抓手、人物付费价值、关系拉扯、成长弧线、记忆点**这 `5` 个默认关键维度里形成可举证的追平路径，并尽量在其中 `2–3` 项建立明确领先方案。

## 起点化竞对吸收口径（新增）

当人物传记服务起点中文网，或未标平台而默认按起点基线处理时，默认把竞对吸收拆成以下四层，并翻译成人物级设计约束：

- **市场数据层**：用榜单位次、均订 / 首订 / 收订比代理信号、推荐资源位等，反推这名角色是否真的支撑题材归类、持续付费价值与长线商业竞争位，而不是只在作者视角里“人设很完整”。
- **内容创作层**：重点吸收主样本在人物抓手、欲望与代价、关系拉扯、身份反差、能力边界、语言指纹、成长弧线与反派动机包装上的做法，把它翻成当前传记的裁判源字段与写作抓手交付。
- **运营策略层**：重点吸收主样本如何通过角色制造追读工程、本章说讨论点、站队点、关系张力与持续回报，把它翻成传记里的触发器、底线 / 开关、留痕偏好与可复盘细节库。
- **受众反馈层**：重点吸收本章说 / 书友圈 / 书评区里高频被夸的人物魅力点、被骂的脸谱化病灶、老白读者对降智 / 空心 / 无代价主角的敏感点，提前写进本轮传记的避坑与加码清单。

起点语境下，默认尽量使用以下术语，而不是写成空泛的“这个人物更立了”：`均订`、`高订`、`首订`、`追订`、`收订比`、`分类月票榜`、`畅销榜`、`新书榜`、`三江`、`本章说 / 段评`、`书友圈`、`老白读者`。拿不到精确值时必须写成代理信号，不得伪造具体数字。

## 默认执行顺序

0. 若按起点口径执行且当前题材目录下缺少可用竞对主报告，先调用 `通用-分析竞对作品`，补齐 `Top10 候选池 → 3–5 本主样本 → 四层深拆` 的最小结论集。
1. 先确认目标人物文件与写入策略。
2. 若按起点口径执行，先吸收竞对主样本四层结论，并写清本轮人物要追平 / 反超的关键项。
3. 判断人物层级：核心 / 重要 / 次要。
4. 核心 / 重要人物优先采用“抬头元信息 → 写作抓手交付 → 基本信息 → 详细内容 → 完整心理弧线 → 索引 / 修订 / 自检”的整包结构，不要把关键模块零散埋在长段叙述里。
5. 先按模板确认四大基础栏位——背景、性格、外貌、其它——都已落位；再在此基础上搭建外层识别、成长时间线、动机、价值观、边界、关系、抓手、秘密、弧线、索引。
6. 再补公众形象 vs 自我认知、外部目标 vs 内部需求、压力反应与锚点物。
7. 最后走质量控制与验证流程。

## 人物硬要求（新增）

- 无论人物层级如何，人物传记模板在原有细分结构基础上都必须显式覆盖 **背景、性格、外貌、其它** 四大基础栏位；允许继续沿用现有章节名，但不得缺栏，也不得把任一类压缩成无法快速检索的零散句子。
- 其中“其它”默认指除背景、性格、外貌外，直接决定人物可写性与可调用性的作者侧关键项，如能力边界、关系网、写作抓手、偏好厌恶、秘密爆点、心理弧线与索引映射。
- 人物至少要同时具备：**真实短板**、**务实动机**、**一个会反复拉扯他的关系或执念**。
- 传记里默认要写清角色想要什么、为什么现在必须要、为了得到它愿意失去什么。**欲望越具体，人物就越立得住，故事就越有驱动力。** 不要写「一个想升职的职场人」，要写「一个被裁员的北漂，想在30岁前攒够首付，给爸妈一个安稳的晚年」。
- 主角与重要角色都应有成长弧光：不是“会不会变强”这么空，而是价值判断、关系处理、行动方式或自我认知会发生什么可写变化。
- 角色魅力不能只靠标签、设定或外形；必须落到口头禅、动作习惯、压力反应、做错事后的补救方式等可直接回贴正文的抓手。
- “强”不是完美无缺；越是承担长线供血的角色，越要避免无短板、无代价、无反噬的万能写法。
- 核心 / 重要人物默认应在标题下显式标注：**人物类型、心理弧线步数、传记版本、最后更新**。
- 写作抓手默认集中放在 **写作抓手交付（正文可直接调用）** 区块，至少细分为：**口头禅 / 句式习惯、习惯动作与微反应、触发器、底线 / 开关、留痕偏好、反制动作模板**。
- 核心 / 重要人物的“基本信息”不止是身份摘要，还应尽量补足：**职业 / 身份轨迹、家族关系 / 原生关系、一句话卖点 / 差异化、题材或时代压力锚点、可复盘细节库、历史地位 / 结构功能**。
- 核心 / 重要人物默认还要写清：**关键经历时间线**、**公众形象 vs 自我认知**、**当前外部目标 vs 内部需求**、**价值观 / 误信**。
- 核心人物默认还应补出：**身份认同危机、内在冲突体系、情感变化链条、价值观演进、本轮修订 / 自检** 中的关键模块，避免只有“大而全设定”而没有“持续可维护结构”。
- 重要人物最好至少有一个**象征物 / 记忆锚点 / 空间锚点**，方便在正文里做回指、反差与情绪唤起。

## 完整模板、动态状态与行为记录补强（新增）

- 主角 / 反派 / 核心重要人物默认按“完整模板”建档：基本信息、性格核心、语言指纹、背景、角色弧线、当前状态、章节行为记录。
- 普通配角可先用精简模板，但只要卷级职责上升、出场频率增多或开始承担独立冲突，默认升级为完整模板。
- “当前状态”属于每章都可能变的动态字段，至少关注位置、情绪状态、已知信息、持有物、身体状态、最后更新章节；“章节行为记录”采用逐章追加，不回改旧记录。
- 传记里的语言指纹不能只写平时怎么说，最好补出情绪变化时的语言变化，以及对上位者 / 同级 / 亲密对象 / 敌对对象的声口差。
- 关系总览只在关系发生实质变化时刷新；人物传记正文中若出现关系字段，应能与关系总览互相回指。
- 默认把“稳定字段”和“动态字段”分开维护：单章情绪波动、临时失态、短期伪装，不等于人物底色改写。

## 最低交付

- 至少能把人物信息清楚归并到背景、性格、外貌、其它四大基础栏位中，且四类都可被直接定位。
- 能写清外层识别、动机、边界、关系、抓手、弧线与索引中的核心字段。
- 核心 / 重要人物至少要补出成长时间线、公众形象 vs 自我认知、外部目标 vs 内部需求、价值观 / 误信中的关键字段。
- 核心 / 重要人物至少应补出“写作抓手交付”“可复盘细节库”“完整心理弧线（含总步数）”三类可直接调用模块。
- 能说明角色为什么值得继续写、继续审、继续回指。
- 能作为正文与大纲的可调用裁判源使用。

## 硬规则

- 传记不是宣传文案，禁止只写抽象褒贬。
- 若按起点口径执行，动手前必须先写清：本轮人物传记准备追平 / 反超哪类主样本的哪项主角抓手、关系拉扯或付费价值。
- 人物必须有动机，也必须有代价。
- 能力不能万能，关系不能只是“认识谁”。
- 写作抓手必须可直接贴回正文使用。
- 核心 / 重要人物的关键抓手不要只写成一行概括，优先列表化，方便在正文里逐项调用和回指。
- 禁止把成长时间线写成流水履历；必须写清哪些事件真的塑造了人物现在的判断、伤口与偏见。
- 禁止只写“别人眼中的他 / 她”或只写“自我独白”；公众形象与自我认知的差值要能制造戏剧。
- 心理弧线不要只写四个阶段标题；关键步骤默认应能回指到具体事件、物件、称呼、地点或编号。
- 除非用户明确要求只脑暴，不得只在聊天里给传记长文而不写入对应文件。
- 主角至少要写清执念、缺陷、秘密三者中的两项；反派必须写清可落地动机来源。

## 与其他 Skill / Prompt 的边界

- 本 Skill 负责人物传记本体的设计与重写。
- 若用户只要审阅和问题分级，应优先使用 `通用-审阅人物传记`。
