# 按平台拆分的推荐型书评模板组

## 起点推荐型模板

- 先写追读抓手
- 再写主角抓手 / 规则快感 / 长线钩子
- 结尾强调“适合追更型读者”

## 番茄推荐型模板

- 先写一个最直接的上头点
- 再写停不下来的体感和连续刺激
- 结尾强调“很容易一口气读下去”

## 豆瓣推荐型模板

- 先写一个最有余味的场景或人物感受
- 再写现实纹理、反转合理性、人性复杂度
- 结尾强调“合上书后会想很久”

## 知乎推荐型模板

- 先抛一个危险问题
- 再写制度 / 关系 / 现实困局如何把问题拧紧
- 结尾强调“值得讨论”

## 微信订阅号推荐型模板

- 先给可传播切口
- 再给 2–3 个能钩人的细节
- 结尾给轻互动式追更诱因

## 出版社推荐型模板

- 先写阅读价值和独特气质
- 再写人物厚度、结构完成度、长期余味
- 结尾强调耐读与回味

## 海外平台推荐型模板

- 先把异常、职业或核心冲突写得直观
- 再写角色代价、关系冲突与事件推进
- 结尾强调 binge-read / next-click / emotional payoff
