# 普通读者推荐型书评框架与种草切口

## 核心口径

- 书评优先写“我的真实阅读体验”，而不是“这本书的完整内容”。
- 最有效的种草点通常来自：具体细节、真实体感、阅读后劲，而不是空泛夸赞。

## 四个优先维度

1. **情节钩子密度**：为什么读者会一直往下翻。
2. **人物真实感**：为什么主角、配角、反派不像工具人。
3. **环境代入感**：为什么场景会让读者觉得离自己很近。
4. **阅读后劲**：合上书后，什么细节还留在脑子里。

## 黄金写法

- 少说“好看、上头、节奏紧凑”，多说“哪一个瞬间让我坐直了、出汗了、翻页了、后怕了”。
- 少讲完整因果，多讲“问题感、镜头感、情绪曲线”。
- 优先把评价词接到具体镜头或体感上。

## 三个常用结构

### 反差种草式

- 我本来以为……结果……
- 适合短评和社交传播。

### 沉浸体验式

- 我读这本书时，最强的体感是什么。
- 适合长评、豆瓣、公众号。

### 问题导向式

- 如果你是主角，你会不会……
- 适合社群讨论、知乎、微博。

## 禁行项

- 不要剧透关键反转。
- 不要把书评写成剧情梳理。
- 不要强行拔高成学术评论。
