# 平台与角度适配矩阵

## 目标平台（默认：起点中文网）

不同平台的读者构成与推荐逻辑不同，书评需在语言风格、侧重维度、标题策略上作相应调整：

| 平台 | 读者特征 | 核心侧重 | 语言风格 |
| --- | --- | --- | --- |
| **起点中文网**（默认） | 国内付费追读者，熟悉网文节奏与术语 | 爽点密度、能力规则代价、主线悬念钩子；开头抓规则矛盾或核心爽点；可用“月票 / 加更 / 追文”等术语 | 精准有力、带网文节奏感；允许“停不下来”“充了钱不后悔”等口语 |
| **番茄小说** | 免费用户为主，泛阅读习惯 | 情绪带入感、日常共鸣、“读了停不下来”的体验叙述 | 更口语、接地气；少用专业术语；强调情绪感受 |
| **百度百家号** | 百度生态泛资讯读者，以下沉市场男性与搜索流量承接用户为主 | 信息密度、搜索友好度、现实投射与轻度猎奇；优先写“这本书会把什么现实困境和异常规则绑在一起”，强调易懂、可复述、可转发 | 中度口语化、信息锚定明确；少绕弯，先讲身份、反常点和现实压力；适合“像在给朋友讲一条很上头的见闻”式表达 |
| **七猫免费小说** | 免费阅读用户、下沉市场男性与碎片化追读者为主 | 高频情绪反馈、底层共情、可控规则爽感；优先写主角如何在现实困局里被异常力量推着破局，强调“追起来很顺、爽点给得勤” | 口语化、短平快、强情绪；避免悬浮抒情，突出“很快进戏、很快见钩子、很快有回报” |
| **知乎盐选** | 有一定阅读品味，欣赏深度与独特性 | 社会议题隐喻、叙事结构、类型融合的独特价值；可适度使用专业术语 | 可略偏深度，允许一定的分析腔，但不能写成论文 |
| **豆瓣阅读** | 文学性读者，注重文笔与人物深度 | 文笔风格、人物塑造、叙事技巧与主题；可讨论文学价值 | 较文学化；强调“阅读体验”与“美感” |
| **小红书** | Z 世代女性与情绪共鸣型读者为主，偏爱强画面、强情绪、强安利感 | 日常真实感、情绪击中、可截图传播的异常细节；优先写“这一幕多戳人”“这个设定多适合安利朋友”，并突出细节体感与关系情绪 | 短句、口语化、情绪词更鲜明；允许更强的安利口吻与细节放大，但避免写成纯尖叫体 |
| **新浪微博** | 社交信息流读者，年轻用户占比高，重互动与话题性 | 话题钩子、评论欲、可转发的悬念句；优先写“这本书最适合引发讨论的异常细节或道德困境”，让读者看完想评论“如果是我会怎么办” | 更短、更冲、更适合单屏阅读；允许轻度话题化表达与结尾提问感，像一条能带讨论的安利微博 |
| **B站** | Z 世代为主，偏爱画面感、轻吐槽、互动感与二创友好内容 | 名场面、反差感、可弹幕化的吐槽与强镜头感；优先写“哪一幕最有画面、哪条规则最有梗、哪种反转最适合追更” | 短句、轻吐槽、允许少量弹幕感表达；节奏快，强调镜头感和互动感，但别写成纯玩梗 |
| **MyFiction** | 泛英语区读者，跨文化阅读 | 中式元素普适化转化；标题含类型关键词（Paranormal / Urban Fantasy）；结尾加互动引导 | 普适化语言；避免文化特异性；短句优先；英文或中英混合 |
| **WebNovel** | 国际读者，英文书评 | 类型标签（paranormal / thriller）、人物代入、chapter hook；强调“binge-reading”体验 | 英文；活泼直接；突出类型标签与情绪吸引力 |
| **GoodNovel** | 偏女性读者，注重情感线与角色关系 | 情感共鸣、关系链代价、角色魅力；可适度强调情感弧 | 温暖而带情绪感；强调“被角色击中”的体验 |
| **微信订阅号** | 公众号推文读者，碎片化浏览 | 开头极度吸睛；小标题分段清晰；突出日常裂缝式问题；允许加表情符号 | 活泼、推文化；多用短段落；小标题清晰 |
| **传统出版社** | 实体书读者，重视文学价值 | 文学性、出版意义、题材社会价值；语言相对正式 | 严肃但不干燥；强调“读完能回味的东西” |

## 写作角度（默认：综合角度）

角度决定书评的聚焦维度——综合角度均衡覆盖，其他角度各有侧重：

| 角度 | 核心聚焦 | 典型适用平台 |
| --- | --- | --- |
| **综合角度**（默认） | 均衡覆盖设定、情节、人物、文笔、情绪体验五个维度 | 起点、全书首次推荐 |
| **类型设定角度** | 职业考据真实感、超自然体系创新、能力规则代价；突出“和别的书不一样在哪” | 起点、番茄 |
| **情节节奏角度** | 单元剧与主线融合结构、悬念铺设、爽点密度；突出“为什么停不下来” | 起点、番茄、七猫、WebNovel |
| **人物塑造角度** | 主角成长弧光与非完美性、配角立体感、人物动机的真实共鸣 | 豆瓣、知乎、GoodNovel |
| **文笔氛围角度** | 感官描写的代入效果、语言节奏风格、氛围与叙事的平衡 | 豆瓣、B站、传统出版社 |
| **社会议题角度** | 都市传说的现实隐喻、职业尊严与阶层议题、情绪共鸣的社会根源 | 知乎、豆瓣、百家号、传统出版社 |
| **情绪体验角度** | 阅读情绪曲线（压迫 → 爆发 → 缓冲 → 再钩）、爽点回报感、“停不下来”的具体体验节点 | 番茄、起点、七猫、小红书、微信订阅号 |
| **市场口碑角度** | 市场数据支撑、读者口碑提炼（好评点与争议点）、目标受众定位 | 百家号、MyFiction、出版决策场景 |

## 平台与角度的联动原则

- 若只指定平台，未指定角度，则角度默认为“综合角度”，语言风格与侧重按平台要求调整。
- 若只指定角度，未指定平台，则平台默认为“起点中文网”。
- 若两者均未指定，则默认以 **起点中文网** 读者为受众，采用 **综合角度** 撰写。
- 若平台与角度之间存在天然适配（如 MyFiction + 市场口碑角度），则在对应维度上适度加强。

## 使用提醒

- 平台差异解决的是“读者为什么愿意继续看 / 愿意转发”，不是让你把同一篇书评只换几个词。
- 角度差异解决的是“你优先强调什么”，不是把其余维度全部删空。
- 若同时存在题材包装层，平台语言必须与题材边界一起生效，不得为了平台口气把题材骨相写丢。
- 百家号与七猫更看重“信息清楚、冲突明确、追起来爽不爽”；别一上来空谈文学性。
- 小红书、微博、B站更看重“这一幕值不值得转、评、截、二创”；要把异常细节和情绪抓手写得够具体。
