# 通用-输出My Fiction版｜跨题材共性与迁移来源

本文件记录 `通用-输出My Fiction版` 当前承接的跨题材共性来源，以及无损迁移期间的执行边界。

## 当前来源

- `都市悬疑/.github/prompts/输出My Fiction版.prompt.md`
- `异能志怪/.github/prompts/输出My Fiction版.prompt.md`

## 当前可稳定上收的 My Fiction 平台共性

- 面向英文连载场景，强调关系黏性、互动感、持续追更与作者口吻拉力
- 输出结构固定为英文正文 + `## 作者有话说`
- 标题应为英文，并兼顾关系张力、阅读意愿与章节抓力
- 平台风格必须先于降相似度，不得先写泛英文稿再套平台外观

## 当前仍保留在题材层的内容

- 各题材本身的调查链 / 规则链 / 现实锚点 / 风险结构
- 题材专属的人物关系、代价形态与执行补丁

## 使用边界

- 命中 `通用-输出My Fiction版` 时，必须同时加载对应题材的 `题材名-输出My Fiction版`。
- 迁移期内若仍需回溯细节，以题材 refs 与源 Prompt 为补充。
