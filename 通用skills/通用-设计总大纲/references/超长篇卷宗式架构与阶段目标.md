# 超长篇卷宗式架构与阶段目标

适用于 100 万字以上，尤其是 200 万字级长篇的总纲设计。

## 核心结论

- 都市悬疑超长篇不能只靠“一个谜底拖到底”，必须采用**卷宗式递进架构**。
- 建议把全书拆成 5–6 个功能明确的卷宗单元，每卷既有阶段闭环，又持续为主线供血。
- 每卷应明确：阶段目标、主线拼图、主要代价、卷尾新缺口。

## 推荐阶段划分

1. **入局卷群**：立主角抓手、抛第一异常、让主线谜面成型。
2. **扩盘卷群**：扩嫌疑池、扩社会面、让支线与主线开始交错供血。
3. **翻面卷群**：旧认知被推翻，主角或阵营关系出现不可逆代价。
4. **逼近卷群**：核心拼图集中回收，真正的敌人 / 机制 / 旧案源头逐步显影。
5. **终局卷群**：证据、动机、关系与余波四线闭合，不靠解释收尾。

## 总纲必须写明的锚点

- 每卷的阶段胜负是什么。
- 每卷给 `M0` 输送了哪块拼图。
- 每卷卷尾要留下什么更危险的问题。
- 哪些角色弧线在本卷前进，哪些在本卷折返或破裂。

## 警戒线

- 禁止单元案只做热闹，不给主线供血。
- 禁止中段卷群只铺垫，不给阶段回报。
- 禁止终局卷群靠临时解释把前文糊过去。
