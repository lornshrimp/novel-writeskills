# 腾讯专栏写作两层级增补

来源：《玄幻主编太山告诉你，写作一定要注意的这两个层级》2.7万阅读，主编说专栏
主讲人：太山，阅文集团玄幻频道主编，从业13年创作+5年编辑

---

## P0 核心提炼

### 第一层级：基本需求（做到这层才能成为实力作家）

#### 1. 方向选择

- PC向作品：行文逻辑性要求高 + 相对新颖有创新成份
- 移动向作品：文字简练 + 故事矛盾冲突剧烈 + 追求剧情爽度（对逻辑和创新要求低）
- **创作之初必须明确选哪个方向**，结合自身创作特点选择，不要盲目跟风

#### 2. 行文

- 玄幻行文不是辞藻华丽，而是：**将背景/设定的介绍，以和风细雨的方式，巧妙融入故事内容**
- 介绍少了：读者看不懂云里雾里；介绍多了：通篇设定没有故事
- 核心原则：**短期会用到的，用简单语言夹在故事中介绍（甚至可以通过书中人物之口说出）；短期用不到的，干脆不提，需要时再写**

#### 3. 节奏

- 节奏 = 各种板块的搭接次序和篇幅
- 玄幻主要板块：设定介绍 / 装X逆袭 / 平淡过渡 / 练功修炼 / 副本任务 / 打斗热血 / 暧昧互动
- 要把这些板块以合理有序、更具卖点的方式拼接；每个板块多少字、后面接什么板块

#### 4. 故事（大纲管理规则）

- 动笔前只需要简易整体故事大纲（不需要全部列完）
- **细纲规则**：每5万-10万字专门制定一个细纲，具体到修炼/打斗
- **大纲可以在创作过程中经常性调整；细纲一旦确认，不要轻易改动**

---

### 第二层级：进阶提升（做到这层才能成大神/写出当红作品）

#### 1. 人物

- 目前玄幻中能将人物写好的作者不多 → 做好这点就独树一帜
- 人物刻画维度：穿着打扮 / 外形特征 / 语言习惯 / 行为动作 / 剧情刻画
- **技巧：选取某一点，进行夸张化描写**（如配角胖子形象=立体好记忆；给某人物设定口头禅）
- 通过剧情结合刻画人物是最难也最好的办法

#### 2. 情绪（文字性格）

- 情绪 = 作品文字的"性格"
- 有自己文字性格的作品更容易凝聚铁杆粉丝
- 示例：有的大神作品大开大合气势蓬勃；有的善于渲染仇恨调动情绪（"恨天小郎君"）

#### 3. 想象力

- 有好的想法随时记录，在合适地方用进去
- 不能过度追求想象力至损害作品逻辑一致性

#### 4. 创意

- 创意和想象力类似，但创意针对整个作品构思，想象力更多着眼某个故事片段
- **有好创意是加分项，但故事本身才是核心**；创意只能带红一本书，之后仍要回归故事质量

---

### 关于创意（通用规则）

> "恐怕不是非创意不可。小说的核心，说到底还是故事。"

- 警惕"没有创意必扑"的错误认知
- 开篇因新奇创意引人注目，但中后期没能驾驭 → 写崩烂尾的案例不少
- **着重作品本身的质量；可以构思创意，但不能太执着**

---

## 与现有规则的关系

- "设定用到才写"（行文第2点）与通用-设计技术设定的设定融入原则高度一致，互为印证
- "细纲确认后不轻易改动"是大纲层面的补充约束，与通用-设计总大纲的卷纲稳定性要求一致
- "选一个维度夸张化"是人物辨识度的具体操作方式，可与通用-设计人物传记的能力边界设计联用
