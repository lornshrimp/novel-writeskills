# 九部件规划与六问总纲质检表

适用于把总纲从“概念很多、事件不少”校准成“真正能支撑长线阅读与后续拆卷”的战略层蓝图。

## 一、先把总纲当作九部件工程，而不是灵感拼盘

总纲层至少要能回答九个结构部件有没有落位。

### 四个连续部分

1. 布局
2. 反应
3. 进攻
4. 解决

### 五个里程碑

1. 开场扰动 / 开场悬念
2. 第一情节点
3. 中间点
4. 第二关键点或等价升级点
5. 第二情节点

## 一点五、质检来源层（新增）

- `顺序职责层`：布局、反应、进攻、解决四段与五个里程碑是否各司其职。
- `牵引供血层`：卖点、主线谜团、主角目标、冲突升级与回报顺序是否可持续牵引阅读。
- `终局准备层`：第二情节点、终局余波与最终解决是否提前准备，而不是末段临时救火。

说明：

- 不要求所有项目都机械照抄百分比。
- 但必须回答：故事如何从铺垫进入强制入局，如何从反应转为进攻，如何在终局前交齐最后关键信息。

## 二、总纲层最容易漏掉的不是内容，而是顺序职责

常见误判：

- 以为写了很多大事件，就等于有结构。
- 以为主线够大，就不需要中间点改向。
- 以为终局够炸，就可以不铺第二情节点。

总纲真正要回答的是：

- 什么信息在前给
- 什么压力在中段升级
- 什么认知在后段落桌
- 为什么读者会在那个时间点收到那个回报

## 三、六问总纲质检法

研究材料里可视作故事讲述六个最重要的问题，在总纲层可转译为以下六问。

### 1. 引人入胜吗？

- 这一总纲最先勾住读者的问题是什么？
- 开场扰动是不是足够具体，而不是只有设定气势？
- 主卖点是否能在前段就被感知，而不是藏到中后段？

### 2. 主人公成立吗？

- 主角为什么必须是他，而不是任何一个设定替身？
- 他的目标、短板、心魔、代价在总纲层是否已经成立？
- 四个部分里，他的状态变化是否可见？

### 3. 冲突够硬吗？

- 谁 / 什么在持续阻挡主角？
- 这种敌对势力会不会只在说明里存在、在正文里失踪？
- 第二部分和第三部分的阻力是否同质化，还是已经升级？

### 4. 背景有用吗？

- 过去的旧伤、旧案、旧债如何影响现在的选择？
- 总纲有没有把背景变成正在工作的力量，而不是背景设定堆积？
- 主题、社会后果、人物世界观是否真会进入事件链？

### 5. 格局清楚吗？

- 四部分的职责是否明确？
- 第一情节点 / 中间点 / 第二情节点是否各有不同作用？
- 读者拿信息、压力、希望、回报的顺序是否合理？

### 6. 解决有准备吗？

- 第二情节点之后，是否还要靠新设定救场？
- 结尾能否由主角成为主要催化剂，而不是外援接管？
- 卖点、主题、情绪和代价是否都能在终局层收口？

## 四、总纲层的最小九部件表

| 部件 | 总纲必须回答什么 | 常见失误 |
| --- | --- | --- |
| 布局 | 现状、缺口、预兆、重大危机前的稳定秩序 | 铺设定太久，故事迟迟不开始 |
| 第一情节点 | 主角的新任务 / 新处境何时被强制成立 | 只是事件变大，没有真正入局 |
| 反应段 | 主角先怎么躲、试、错、查 | 主角太早无敌，中段像流程表 |
| 中间点 | 哪个新认知 / 新误差 / 新压力把故事扳向新轨 | 只有小反转，没有大改向 |
| 进攻段 | 主角如何开始主动布局和反击 | 看似主动，其实仍在被推着走 |
| 第二关键点 | 敌对势力如何再次显影并升级压迫 | 中段后半没压迫，只剩推进 |
| 第二情节点 | 最后一批关键信息何时落桌 | 终局前还要临时补设定 |
| 解决段 | 主角怎样成为解决问题的关键催化剂 | 靠巧合 / 外援 / 解释收尾 |
| 终局余波 | 代价、回收、余震、长线缺口怎样分层 | 只写结果，不写余波 |

## 五、总纲层最实用的一页纸做法

如果总纲太大，先压成一页纸也要能看出：

1. 一句话卖点
2. 一句话主线谜面
3. 主角在四部分里的状态变化
4. 三个主里程碑：第一情节点 / 中间点 / 第二情节点
5. 卖点在前段、中段、终局各怎样显影
6. 哪些 `M0 / X / C` 线在供血，哪些只是辅助

如果这一页纸压不出来，说明总纲仍然发散。

## 六、与本项目现有体系的接口

### 对 `M0`

- 用九部件校准主线谜团的阅读顺序
- 确保不是只有谜面，没有阶段性理解升级

### 对 `X`

- 标出哪些支线在反应段主要施压
- 哪些支线在进攻段反咬主线或转成筹码

### 对 `C`

- 说明单元案主要承担哪一段的供血职责
- 不要让全部 `C` 只会“推进调查”，不会改局

## 七、硬性警报

出现以下任两项，应判总纲未达标：

- 开场只见世界观，不见故事缺口
- 第一情节点说不成一句话
- 中间点不是改向，只是信息增量
- 第三部分比第二部分只多了字数，没有更多主动性
- 第二情节点后还需要新规则、新人物、新证据来救场
- 卖点只能写在简介里，无法映射到九部件

## 七点五、Green / Yellow / Red 质检判定（新增）

- `Green`：顺序职责、牵引供血、终局准备三层都清楚，总纲具备长线可拆性。
- `Yellow`：主干能看，但至少一层发虚，例如中间点不是改向、终局准备不足或卖点映射不稳。
- `Red`：九部件只是名义存在，结构顺序、牵引与终局准备至少一层明显失效。

## 八、一句话收口

**总纲不是把大事排成时间线，而是让读者在正确的时候拿到正确的问题、压力、认知、希望与终局回报。**
