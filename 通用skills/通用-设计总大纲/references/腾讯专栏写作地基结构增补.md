适用范围：默认口径（起点 / 未标平台）；适用于所有品类大纲设计

## P0 规则：核心公理（作者子良认为的故事基础规律）

1. **故事最重要的是结构**
2. **故事的灵魂是人物**
3. 畅销作品的共同点，大概率是作品畅销的必要条件（要向畅销作品学习结构）
4. 正确的理论能适用于所有对象；能适用于绝大多数对象的理论可以近似认为正确
5. 技巧与内容彼此独立，而有关联

---

## P0 规则：核心故事结构模型

### 目标-行动-收获-收获重要性展示

起点男频故事结构的基础单元：

```
目标 → 行动 → 收获 → 收获重要性展示（即时正反馈）
```

关键：**收获重要性展示**（展示收获的价值）是最常被省略但最不能省的一步。

相当于游戏设计里的"即时正反馈"——玩家（读者）必须能感知到行动带来了实质性收益。

### 各品类对应的结构形式

| 品类 | 具体结构形式 |
| --- | --- |
| 升级文 | 困境 → 练功 → 升级 → 碾压破解困境 |
| 都市日常文 | 日常/困境 → 利用金手指提升 → 能力/社会地位提升 → 破解困境 |
| 悬疑文 | 悬念 → 探索 → 线索 → 悬念解开 |
| 势力经营文 | 困境 → 建设 → 势力提升 → 破解困境 |

## P0 规则：开篇章节的核心职责

### 开篇三步走

1. **明确人设** + 金手指出现 + 开始探索（第一章）
2. **详述准备过程**，进行细节设定，获得第一个道具/线索（第二章）
3. **凸显金手指在现实生活中的优越性**（第三章）

### 真实感的来源

**细节铸就真实感**：

- 详细的准备过程 → 真实感
- 详细的设定 → 真实感（起点对真实感要求高）
- 注意："太多设定"会显得古早，需要控制量

### 即时正反馈的作用

升级文里最典型的即时正反馈结构：

```
练功前被打（建立问题感）→ 练功 → 练功后打怪兽（答案呈现）
```

这个结构针对的是读者的**获得感**，是维持章节阅读动力的核心机制。

---

## 总大纲设计时的增补检查项

- [ ] 全书每个主要阶段是否都遵循"目标-行动-收获-收获重要性展示"循环？
- [ ] 每个收获是否有对应的"重要性展示"剧情（避免收获后立刻进入下一收获而不展示价值）？
- [ ] 开篇三章是否明确了主角人设、金手指出现、第一次行动的完整节奏？
- [ ] 是否充分利用了真实感设计（详细准备过程、细节设定融入剧情）？
