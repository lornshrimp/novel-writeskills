## 14. 三层验收映射（新增）

- `边界验收层`：总纲主文件与 `故事面/` 是否职责清楚、交接清楚。
- `战略验收层`：卖点、竞对超越路线、`M0 / X / C`、跨卷推进、终局闭环是否站得住。
- `维护验收层`：模板组合声明、下游接口、章节规划入口、节奏 / 决策维护口是否齐。

## 15. Green / Yellow / Red 模板验收判定（新增）

- `Green`：三层都过，可作为总纲层标准母版或项目总纲包模板使用。
- `Yellow`：主干完整，但仍缺一层维护或交接细节，适合继续补强后再下压。
- `Red`：模板能看但不能稳用，边界、战略或维护层至少一层明显缺失。
