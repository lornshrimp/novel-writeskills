## 当前可稳定上收的 WebNovel 平台共性

- 面向全球年轻读者的跨文化可读性、移动端节奏与英文平台门禁
- 输出结构固定为英文正文 + `## 作者有话说`
- 标题必须为英文，且正文除 marker 行外不得出现中文/CJK
- 开头快抓人、中段高频推进、结尾强钩子
- 平台风格必须先于降相似度，不得把 WebNovel 写成泛英文平台稿

## 当前仍保留在源 Prompt 的详细规则

在无损迁移完成前，以下内容仍以源 Prompt 正文为详细依据：

- 各题材在 WebNovel 平台下的研究补强条款
- 题材特有的规则链 / 证据链 / 现实锚点 / 跨文化解释边界
- 精细化节拍、标题倾向、作者有话说口吻与补丁规则

## 使用边界

- 命中 `通用-输出WebNovel版` 时，必须同时加载对应题材的 `题材名-输出WebNovel版`。
- 若当前任务仍处于迁移试点阶段，必须继续回看源 Prompt 与试点台账，不得假设本文件已经完全替代旧 Prompt。
