# 微空间执行检查表

- [ ] 这个空间的日常用途已经清楚
- [ ] 空间里的常态动作已经清楚
- [ ] 反常细节足够小，但一眼不对劲
- [ ] 反常不是一步到顶，至少经过一次可感知升级
- [ ] 场景里至少有一个现实针脚
- [ ] 角色不是只在“感到害怕”，而是做了动作
- [ ] 角色的动作会立刻带来后果
- [ ] 主要动作与主要后果之间是顺接的，不是硬拐的
- [ ] 若动作失手，失手本身也造成了升级，而不是白白落空
- [ ] 反常能接回规则链、证据链或后续冲突
- [ ] 场景中至少有一项后果：误判 / 信息缺口 / 证据入场 / 风险升级 / 关系张力 / 规则显影
- [ ] 压迫与回报没有互相打架，场景不是只有压迫没有收益
- [ ] 至少有一句能拍出来的“落针句”，不是全靠抽象判断推进
- [ ] 场景结尾不是感慨，而是带着新问题 / 新代价 / 未完成动作离开
- [ ] 删掉“诡异 / 阴森 / 压抑 / 不安”这些词后，场景仍然成立
- [ ] 角色不是一直在“感到 / 觉得 / 意识到”，而是真的在看、听、碰、核对
- [ ] 这地方仍然像日常空间，不像被写成废弃鬼屋
