# 微空间动作后果速配表（专项补充版）

> 通用动作—后果骨架已上收至 `通用-执行场景单元/references/场景动作后果速配表.md`。

本文件只补充**微空间语境下**更常见的落地方式。

## 微空间优先落地方式

| 动作 | 在微空间里更常见的主后果 | 典型微空间落地 |
| --- | --- | --- |
| 走近 | 风险升级 | 进入盲区 / 死角 / 错层 |
| 伸手触碰 | 规则显影 | 反馈方式不符合常识 |
| 回头确认 | 误判 | 方向判断被打乱 |
| 贴近去闻 / 去听 | 证据入场 | 声源 / 气味更具体，能挂到载体上 |
| 停住不动 | 信息缺口 | 暂时保命，但错失关键接触点 |
| 退出 / 后退 | 关系张力 | 被同伴 / 目标对象误解 |
| 再按一次 / 再试一次 | 风险升级 | 系统反馈更糟，异常被二次坐实 |
| 重放 / 对照 | 证据入场 | 画面 / 记录出现口径差 |
| 截图 / 拍照 | 新风险 | 留证据的同时留下自己的痕迹 |
| 开门 / 推门 | 规则显影 | 本该锁着的空间被打开，或开法不对 |
| 叫人 / 发消息 | 关系张力 | 对方反馈与现场不一致 |

## 微空间补充提醒

- 在微空间里，后果最好更贴脸、更快、更带空间限制。
- 同一个动作，如果不能改变角色与空间的关系，就还不够像微空间动作。
