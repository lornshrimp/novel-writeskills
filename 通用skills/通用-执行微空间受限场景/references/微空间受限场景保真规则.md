# 微空间受限场景保真规则

## 一、微空间不是背景，而是冲突发动器

微空间 / 受限场景的价值，不在“写一个小地方”，而在于：

- 读者足够熟悉
- 人物足够受限
- 反常足够贴脸
- 后果足够立即

因此它必须同时承担：

- 现实锚点任务
- 压迫任务
- 证据链入场任务
- 下一步行动触发任务

## 二、日常场景异化口径

- 反常必须足够小，但足够不对劲。
- 反常必须能回到证据链，而不是只做气氛。
- 反常出现后，人物必须立刻做动作。

## 三、都市 / 日常微空间公式

每次使用微空间时，至少补写三项：

1. 这个空间的**常态动作**是什么。
2. 哪个细节首先**违反常态**。
3. 人物为了确认异常，做了什么**立即有后果的动作**。

推荐公式：

`空间常态 → 微反常 → 立即动作 → 证据 / 风险 / 误判后果`

## 四、现实锚点密度口径

微空间场景必须让读者觉得：

> 这事像发生在我楼下 / 我手机里 / 我单位里 / 我值班时会碰到的地方。

若场景只剩“阴森、压抑、诡异”，却没有可辨识的现实针脚，则它还不够像真实受限场景，只像空泛恐怖。

## 四点五、开场密度口径

- 前 7 行内，至少交代出：空间类型、角色当前动作、第一处不对劲或迫近风险。
- 前 300 字内，至少出现一次真正能推动动作的刺激，而不是只做环境渲染。
- 若第一处异常要靠前文设定才能成立，则说明当前场景起手太慢。

## 四点六、感官组织口径

- 优先使用可被立即感知的细节：光线错格、门缝、潮气、血腥味、金属擦响、门禁延迟、震动残留。
- 优先让触觉、听觉、嗅觉参与，不要只剩视觉形容词。
- 删除“诡异、阴森、压抑、不安”后，场景仍应成立；否则说明场景主要靠标签词撑着。
- 身体反应优先于情绪判断：先写后颈发凉、掌心出汗、呼吸卡住，再写角色怎么理解它。

## 五、动作纪律

人物不能只是“感觉到可怕”，必须做动作。常见有效动作包括：

- 走近
- 伸手
- 回头
- 检查
- 退出
- 停住
- 按掉 / 重放 / 对照 / 截图 / 贴近去闻 / 试探性触碰

没有动作，压迫就无法转化成后果；没有后果，场景就只是氛围。

## 六、后果类型

一个合格的微空间场景，反常出现后至少应带来以下之一：

- 误判
- 信息缺口
- 证据入场
- 风险升级
- 关系张力
- 规则显影

其中至少一项应在本场落地，不要把所有后果都拖到下一场才结算。

## 七、回报与压迫的配平

- 只有压迫没有回报，读者会疲劳。
- 只有回报没有压迫，读者会觉得廉价。

因此最好做到：先压迫 → 再让人物拿回一点掌控感 → 再打开更坏后果。

## 八、高频失败信号

- 反常过大，一上来就脱离现实
- 反常过小，但没有后续动作和后果
- 细节全是抽象词，没有可复盘载体
- 角色只在感受，没有做事
- 空间只是地点名，没有功能感和常态动作
- 开场迟迟不给异常，只在预热
- 解释多于动作，像在读设定说明
- 只有压迫没有回报，角色始终拿不回任何信息或掌控感
- 惊悚只靠“看见鬼”，没有日常故障、流程摩擦、职业细节或现实针脚作托底

## 九、最终验收口径

一个真正合格的微空间 / 受限场景，应同时满足：

- 读者认得出这个空间的日常用途
- 第一处不对劲足够小，但足够刺眼
- 人物为了确认异常做了动作
- 场景里有现实针脚和可复盘载体
- 场景结束后局势发生了变化，而不是只有气氛加重
