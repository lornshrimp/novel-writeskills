# 候选书名文件模板

> 适用：`通用-设计书名`
>
> 用途：把书名设计结果落成可复审、可回写、可收口的方案文件。
>
> 说明：本模板强调“候选池 + 评分 + 选择理由 + 推荐结论 + 去重记录 + 自检”。不要只交一张名字清单。

## 9. 自检清单

- [ ] 是否先读了工作区中与命名直接相关的材料

- [ ] 是否为不同平台选择了不同命名主轴，而不是换词平移
- [ ] 是否所有正式候选都先过了“像不像书名”这一关
- [ ] 是否每个候选都写了评分或排序依据
- [ ] 是否对未入 shortlist 的候选写了降级理由

- [ ] 是否把内部草案和正式候选表分开
