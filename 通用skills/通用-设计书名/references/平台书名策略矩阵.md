# 平台书名策略矩阵

## 默认平台集合

- 起点中文网
- 番茄小说网
- 七猫免费小说
- 豆瓣阅读
- 知乎盐选
- 传统出版社
- 微信订阅号
- 百度百家号
- 小红书
- B站
- 新浪微博
- WebNovel
- MyFiction
- GoodNovel

## 平台适配的核心原则

- 平台适配的是“命名形态”，不是简单换词。
- 不同平台的头号推荐，必须尽量拉开主轴与句法区隔。

## 默认策略

### 起点中文网

- 推荐长度：2–8 字优先，最长不超过 10 字为佳
- 风格：硬、稳、能装下长线世界观
- 优先结构：长线容器型 / 高概念设定型 / 系列感标题
- 避免：过长叙述句、生活化过头的轻标题

### 番茄小说网

- 推荐长度：4–10 字优先，少量可到 12 字
- 风格：狠、直、快，但仍需像书名
- 优先结构：单次危机型 / 反常事件型 / 强钩子情绪型
- 避免：像短视频标题、像一句故事文案的长句

### 七猫免费小说

- 推荐长度：4–10 字优先，建议首屏可读、停顿少、最好一眼能念顺
- 风格：直给、易懂、情绪抓手强、首屏可读，但仍需保留书名感
- 优先结构：现实问题型 / 危机事件型 / 逆袭入口型
- 加分方向：问题清楚、危机清楚、情绪清楚，读者第一眼能快速判断“这本书讲什么、劲往哪儿使”
- 避免：过分留白、纯抽象、第一眼读不懂冲突或卖点、像文艺副标题而不像网文书名

### 豆瓣阅读

- 推荐长度：3–8 字优先
- 风格：克制、冷、可复义
- 优先结构：意象型 / 人性命题型 / 现实悬念型
- 避免：爽文口吻、喊话式书名、过度营销感

### 知乎盐选

- 推荐长度：8–18 字可接受
- 风格：强冲突、强动作、强人称代入
- 优先结构：第一人称叙述型 / 职业口述型 / 冲突动作型
- 避免：空泛命题句、太像论文题目

### 传统出版社

- 推荐长度：2–8 字优先
- 风格：准确、克制、出版面孔稳定
- 优先结构：象征型 / 现实悬疑型 / 长销型标题
- 避免：网络腔、过度喊口号、太像章节名

### 微信订阅号

- 推荐长度：2–8 字优先；少量可做传播型副方案
- 风格：短、能转发、能引讨论
- 优先结构：社交话题型 / 日常反常型 / 情绪转发型
- 避免：纯抒情、纯抽象、太长太绕

### 百度百家号

- 推荐长度：4–10 字优先
- 风格：现实入口清楚、职业锚点明确、异常点一眼能懂
- 优先结构：搜索词 + 冲突型 / 职业 + 事件型 / 问题切口型
- 避免：过分留白、太文艺、读者第一眼看不出在讲什么

### 小红书

- 推荐长度：4–10 字优先；少量可到 12 字
- 风格：情绪感强、日常感强、异常切口清楚
- 优先结构：情绪冲突型 / 日常反差型 / 体感异常型
- 避免：过硬设定、过技术、像男频长线母题而不是点开入口

### B站

- 推荐长度：4–9 字优先；少量可做 10–12 字反差型标题
- 风格：画面感、反差感、年轻感，但必须仍像书名
- 优先结构：画面名场面型 / 反差型 / 可讨论型
- 避免：纯梗标题、过于二创化、像视频标题而不像书名

### 新浪微博

- 推荐长度：4–10 字优先
- 风格：切口短、现场感强、适合社交传播
- 优先结构：短促现场型 / 热点切口型 / 关系危机型
- 避免：太长、太绕、太像长篇总纲标题

### WebNovel

- 推荐长度：2–6 个实词为佳
- 风格：高概念清楚、剧情入口明确
- 优先结构：高概念型 / 普适危机型 / 跨文化易译型
- 避免：难译文化专名过多、抽象命题过重

### MyFiction

- 推荐长度：3–8 个词
- 风格：人物秘密、关系裂痕、非对称信息
- 优先结构：关系秘密型 / 日常反常型 / 情感动机型
- 避免：冷冰冰纯设定书名

### GoodNovel

- 推荐长度：4–9 个词
- 风格：情绪、命定、人物危险感更强
- 优先结构：危险关系型 / 情绪命运型 / 普适超自然型
- 避免：过冷、过技术、过学术
