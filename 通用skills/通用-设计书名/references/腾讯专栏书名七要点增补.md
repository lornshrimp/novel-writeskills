# 腾讯专栏书名七要点增补

来源：《网络小说入门技巧——小说的表面文章》14.7万阅读，新手专区专栏

---

## P0 核心提炼

### 书名七要点（按重要性排列）

书名是读者决定是否点开的首要依据，也是作品能否被记住的关键因素。

1. **简单**：用尽可能简单的汉字，小学生都能认识，不哗众取宠用生僻字
2. **全汉字组成**：不使用英文、字母、数字、符号，全汉字构成，更规范且有利于读者搜索
3. **响亮好记**：朗朗上口，不能读起来拗口，读者连书名都读不顺就无法记住
4. **不太长**：2–5字内为宜，**四字书名最普遍**（成语四字非偶然，是验证过的记忆规律）
5. **不文艺**：网文面向通俗文学读者，过文艺/哲学气息浓/像言情书名均损人气
6. **适度跟风**（神似形不似）：观察当红作品找到好名字规律，**创新不雷同**；纯跟风会被读者厌恶且被平台限制推荐
7. **体现作品特点**：最好能体现**核心YY点**（书名=快速传递卖点），如穿越背景、特殊异能等

### 关于书名长度的补充规则

- "书名越短越好"是片面说法，短书名有利于排行榜排位，但不能牺牲吸引力换位置
- 综合来看：**吸引力 > 排行榜位置效益**

### 章节名六原则（补充）

章节名的特点是**成事不足、败事有余**——一个好章节名加分有限，一个糟糕的章节名能让人气大跌。

1. **规避违禁内容**：按"书名→简介→章节名→正文"的审查次序，章节名违禁直接删书
2. **规避仆街元素**：如绿帽暗示等，即使正文中没有，标题误导读者也是不可挽回的损失
3. **避免过多剧透**：不要在章节名里提前揭示精心设置的悬念或伏笔结局
4. **控制字数**：过长的标题在部分网站会导致页面变形；长短差异过大也影响目录美观
5. **无论如何都得有**：只有章节序号而无标题的目录会严重影响读者点击欲望
6. **适当体现YY精神**：能为读者提供期待感的章节名，可在作品质量不变的情况下维持老读者
