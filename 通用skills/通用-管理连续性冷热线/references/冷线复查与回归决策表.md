# 冷线复查与回归决策表

## 一、先判断它为什么冷

### 正常冷却

- 正在给别的线让位
- 读者仍记得它
- 动态状态里已写明“暂缓原因”

### 危险失温

- 读者可能已经忘了
- 前次变化太大，但后续没桥接
- 即将回收，却长期没有触碰

## 二、四种决策

### 1. 直接推进

适合：该线本来就该回来了。

### 2. 间接提醒

适合：本章主任务不在它身上，但必须让读者重新记起它，且记起后期待值发生变化。

### 3. 压力转移

适合：本线暂时不回归，但影响转移到另一个人物 / 事件 / 机构身上。

### 4. 合理休眠 / 归档

适合：当前卷确实不该碰，或它已经完成使命。

## 三、禁止的假处理

- 点名式 cameo
- 硬塞一句“我们还没忘记那件事”
- 让角色出来露个脸，但任何压力都没变
