# 章节正文审阅检查清单

## 一、先看抓力，再看文采

章节正文对象的首要判断不是“句子漂不漂亮”，而是：

- 读者能不能被章首拖进去
- 中段有没有明确收获
- 章末有没有下一章动力

## 二、章首抓力

- 前 2–3 段是否至少命中以下两项：
  - 异常 / 冲突触发
  - 情绪锚点
  - 悬念缺口
- 是否避免了背景说明或设定科普压过故事入场

## 三、中段兑现

- 中段是否至少有一次可指认的回报：
  - 新证据落袋
  - 误导被证伪
  - 关系发生变化
  - 规则边界被试出
  - 有效反击成立
- 是否指出：回报类型、落在哪个镜头、是否足够清晰

## 四、章末钩子

- 钩子是否落在具体动作 / 信息 / 危险 / 未完成决定上
- 是否避免了空泛感慨、空问号或假危机

## 四点五、四拍门禁与钩子轮换同构核查（新增）

- 单章是否成立“四拍”最小闭环：
  - 钩子（前200）
  - 蓄力（中段加压）
  - 回报（中段可感兑现）
  - 留钩（后200未闭合）
- 若“四拍”缺失两拍及以上，是否已升级为结构性风险（不得按文风问题轻描淡写）
- 本章钩子类型是否与前两章过度重复（钩子轮换过载）
- 若出现“连续三章同类钩子”风险，是否已给出下一章强制换型补丁

## 五、现实与规则落地

- 文本中是否有可感知的现实锚点或场景支撑
- 规则 / 证据 / 程序 / 代价是否不是只在口头存在
- 现实锚点是否具体到空间、物件、职业流程或日常界面，而不是只剩抽象气氛词

## 六、工程核查

- 本章应落地的卷纲功能点是否都已落地
- 出场人物是否已做传记核查，是否存在身份 / 关系 / 能力边界冲突
- 关键判断是否能回指到证据
- 误导是否可证伪
- 是否出现作者管理编号、具体年份或突然全知解释等出戏红线

## 七、工作文件与读者层边界

- 若章节文件同时包含 `## 作者有话说`、`## 章节后记`、润色说明或回写标记，默认视为作者工作层与正文同文件共存的设计，不因此判为问题。
- 默认放行判断只针对章节正文本体；除非用户明确要求，否则不要把 `## 作者有话说`、`## 章节后记` 是否存在纳入读者出戏项。
- 只有当作者侧内容混入正文、章标题，或用户明确要求审“裸发终稿 / 读者直出稿 / 合刊终稿”，才把这类问题升级为发布风险。
- 若本轮还要判断“原始工作文件是否可直接裸发给读者”，必须把它单列为**附加结论**，不得覆盖默认的正文质量 / 读者可见层放行结论。

## 八、读者端核查

- 每段是否有明确焦点
- 对话是否承担功能，而不只是搬信息

## 九、对话三项专项核查

- 是否存在可感知的**声音差**：删掉提示语，仍能大致听出不是同一口气
- 是否完成**塑人**：对话能否暴露人物在护什么、怕什么、想抢什么，或其职业惯性 / 情绪处理方式
- 是否形成**联结**：读者是否会因为对话而对至少一个人物产生认同 / 同情 / 喜爱 / 好奇
- 若对话承担重大场景推进，是否出现“信息全说了，但人物没立住”的问题
- 若多人同场，是否存在统一作者腔，导致关系温度和位差发白
- 是否至少有一处情绪兑现
- 是否至少有一处贴角色、贴时代的记忆点

## 十、链路失配核查

- 本章当前问题主要是正文表层问题，还是上游供血 / 控制卡 / 场景职责 / 首尾补丁失配问题？
- 若开头已抓人但与本章主线脱节，是否已判为链路失配而非单纯文笔问题？
- 若中段“有内容没回报”，是否已判断回报是否只存在于控制卡里、没进正文？
- 若结尾“有气氛没驱动”，是否已判断最后一场职责与下章接口是否失配？
- 若文本经过润色 / 去AI味后更顺却更弱，是否已判定为补丁遮羞而不是实质修复？
- 若正文在闭环步骤 2 已落稿，是否已复核 `HookType / MidPayoff / TailDriver / RotationCheck / GatePass` 这 5 个日志字段

## 十一、网文付费与追读专项核查

- 本章是否至少有一次主角主动推进 / 有效反制 / 争取筹码，而不是全程承压
- 本章是否至少交付 `1–2` 个微爽点或 `1` 个中爽点；若完全没有，是否已标记为“付费吸附力风险”
- 本章是否存在明显“可以先放一放”的空档感；若有，是否已指出是钩子弱、回报轻、主角主动性不足还是信息增量太虚

## 十二、章节接棒与连续性热度核查

- 章末的新变量是否足够具体，能直接接到下一章第一屏
- 本章是否至少回应了上章或近几章的一个高优先问题，而不是只持续开新口子
- 主要人物 / 关系 / 伏笔 / 卖点链是否已连续 `5` 章以上未有效触碰，本章是否又错过了合理升温窗口

## 十三、平台讨论度与传播切口核查

- 是否存在一句可截图、可转述、可争议、可站队的话或动作
- 是否存在一个会触发段评 / 本章说的细节、态度或认知反差
- 若全章没有可讨论点，是否已指出这会削弱网文平台的互动数据与推荐潜力
