# 问题分级与补丁动作

## P0

- 功能断裂：章首失效、中段无回报、章末无钩子
- 事实 / 规则 / 证据重大冲突
- 人物传记一致性明显冲突
- 关键线索不公平：读者无法回指，或真相只能靠作者解释补出
- 规则 / 能力 / 程序边界被写成万能解题器

处理：必须返工，先修骨架再谈文笔。

## P1

- 节奏偏弱、回报偏埋、钩子发假、现实落地不足
- 证据存在但不可回溯，或误导不可证伪
- 角色声音、段落焦点、情绪兑现明显不足
- 对话高占比场景里，人物声音分不开、塑人无效、读者跟随理由发弱
- 工作文件边界模糊：作者侧说明开始侵入正文或 `## 作者有话说`

处理：给出明确补丁位置与补丁类型。

## P2

- 语言、镜头、细节、张力仍可继续增强

处理：列为优化项，不阻断当前结构判断。

## 常见工伤与补丁优先级

### 一、章首抓不住

1. 提前异常显影
2. 提前现实摩擦
3. 删除暖机说明

### 二、中段空转

1. 补一条新证据
2. 补一处误导被证伪
3. 补一笔关系变化、规则试错结果或代价兑现
4. 若连续调查 / 抒情无回报超过一段，直接建议返工该镜头段

### 三、现实感太空

1. 补一个空间功能细节
2. 补一个职业动作
3. 补一个现代生活针脚

### 四、钩子发假

1. 改成具体动作中断
2. 改成具体新信息入场
3. 改成具体危险落地

### 五、工作文件与读者层混淆

1. 先判断问题是否真的进入正文或 `## 作者有话说`
2. 若只是 `## 章节后记` / 润色说明与正文同文件共存，不单独升格为 P0 / P1
3. 若作者侧标签直接闯入读者正文，再按发布风险处理

### 六、对话只有信息没有人物

1. 先标出哪几句台词只在交代信息，没有关系压力
2. 补一处位差、风险或隐瞒，让对话先长出冲突重心
3. 补一句能暴露人物护点 / 短板 / 代价观的话
4. 若整段对话承担重大推进，建议转 `通用-执行对话冲突` 做专项返修

### 七、人物声音分不开

1. 先指出哪两个人物最像同一作者在轮流说话
2. 要求拉开句长、措辞、回避方式、打断方式或情绪处理方式
3. 强制做一次“删掉提示语后还能否分人”的复检

### 八、读者联结发弱

1. 明确本章目前缺的是认同 / 同情 / 喜爱 / 好奇中的哪一项
2. 补一处具体对话或动作，让人物露出护点、疼点、可爱点或危险秘密
3. 若人物天然不讨喜，至少补出“为什么仍值得继续盯着看”的理由

## 建议评分尺

建议至少评分以下 6 项，每项 10 分：

1. 章首抓力
2. 中段兑现
3. 现实锚点
4. 证据公平 / 规则边界
5. 对话承担度（声音差 / 塑人 / 联结）
6. 章末钩子

### 快速判定口径

- 任一项 ≤ 6：该项视为明显短板
- `章首抓力`、`中段兑现`、`章末钩子` 任一 ≤ 5：建议返工
- `对话承担度` ≤ 5 且本章高占比靠对话推进：建议优先返修对话链
- 总体均分 ≥ 8.5：更适合进入精修层，而不是结构返工层
- 若 `证据公平 / 规则边界` ≤ 5 或人物一致性直接冲突，优先按 P0 / P1 处理，不受均分掩盖
