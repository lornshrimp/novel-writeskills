# 问题分级与补丁动作

## P0

- 功能断裂：章首失效、中段无回报、章末无钩子
- 四拍门禁断裂：同章缺失两拍及以上，或 `GatePass=Fail` 仍尝试放行
- 事实 / 规则 / 证据重大冲突
- 人物传记一致性明显冲突
- 关键线索不公平：读者无法回指，或真相只能靠作者解释补出
- 规则 / 能力 / 程序边界被写成万能解题器

处理：必须返工，先修骨架再谈文笔。

## P1

- 节奏偏弱、回报偏埋、钩子发假、现实落地不足
- 证据存在但不可回溯，或误导不可证伪
- 角色声音、段落焦点、情绪兑现明显不足
- 对话高占比场景里，人物声音分不开、塑人无效、读者跟随理由发弱
- 工作文件边界模糊：作者侧说明开始侵入正文、章标题或其它明确面向读者的正文区块
- 润色 / 去AI味 / 首尾补丁后，问题表面变顺但主抓手、回报、钩子或场景职责被磨平
- 开头 / 中段 / 结尾表面成立，但与控制卡、本章供血线或场景职责明显脱节
- 钩子轮换过载：连续三章同类钩子导致节拍麻木

处理：给出明确补丁位置与补丁类型。

## P1.5

- 中段有内容，但回报强度偏弱，读者容易在本章或本卷中段掉队
- 主角连续承压，但主动推进 / 有效反制 / 抢筹码动作缺位
- 章末表面有钩子，但下章接棒对象发虚，读者会产生“可以先放一放”的空档感
- 主要人物 / 关系 / 伏笔 / 卖点链已逼近回归窗口，本章仍未有效触碰，出现失温征兆

处理：必须补 `1` 个明确微爽点或 `1` 个中爽点、`1` 次主动推进，或 `1` 个更具体的下章接棒对象；补丁后复审，不宜当普通 `P2` 放过。

## P2

- 语言、镜头、细节、张力仍可继续增强

处理：列为优化项，不阻断当前结构判断。

## 常见工伤与补丁优先级

### 一、章首抓不住

1. 提前异常显影
2. 提前现实摩擦
3. 删除暖机说明

### 二、中段空转

1. 补一条新证据
2. 补一处误导被证伪
3. 补一笔关系变化、规则试错结果或代价兑现
4. 若连续调查 / 抒情无回报超过一段，直接建议返工该镜头段

### 三、现实感太空

1. 补一个空间功能细节
2. 补一个职业动作
3. 补一个现代生活针脚

### 四、钩子发假

1. 改成具体动作中断
2. 改成具体新信息入场
3. 改成具体危险落地

### 五、工作文件与读者层混淆

1. 先判断问题是否真的进入正文段落或章标题，而不是只存在于同文件的作者侧区块
2. `## 作者有话说`、`## 章节后记`、润色说明与正文同文件共存，默认属于工作流设计，不单独升格为 P0 / P1
3. 只有作者侧内容实际闯入正文，或用户明确要求按读者终稿口径严判时，再按发布风险处理
4. 不得把“原始工作文件不可裸发”和“正文不建议进入后续发布流程”混写；若用户确实要求前者，必须单列“附加结论”

### 六、对话只有信息没有人物

1. 先标出哪几句台词只在交代信息，没有关系压力
2. 补一处位差、风险或隐瞒，让对话先长出冲突重心
3. 补一句能暴露人物护点 / 短板 / 代价观的话
4. 若整段对话承担重大推进，建议转 `通用-执行对话冲突` 做专项返修

### 七、人物声音分不开

1. 先指出哪两个人物最像同一作者在轮流说话
2. 要求拉开句长、措辞、回避方式、打断方式或情绪处理方式
3. 强制做一次“删掉提示语后还能否分人”的复检

### 八、读者联结发弱

1. 明确本章目前缺的是认同 / 同情 / 喜爱 / 好奇中的哪一项
2. 补一处具体对话或动作，让人物露出护点、疼点、可爱点或危险秘密
3. 若人物天然不讨喜，至少补出“为什么仍值得继续盯着看”的理由

### 九、链路失配

1. 先判断问题主要落在哪一层：故事面供血 / 控制卡 / 场景职责 / 开头补丁 / 章末补丁
2. 不要只建议“润一下 / 顺一下 / 去AI味”，先指出该回写的上游位置
3. 若问题来自补丁遮羞，优先撤回到结构职责层修，而不是继续堆补丁

### 十二、四拍 / 轮换门禁失效

1. 若 `GatePass=Fail`，不得进入终审，先回写步骤2正文
2. 若 `RotationCheck=Yes`，下一章控制卡必须强制换型钩子
3. 若连续三章同类钩子，至少改一章章首钩子类型并重审后200留钩

### 十、网文吸附力不足

1. 补一个小赢、小反击、小揭露或小到账
2. 补一次主动试错、争筹码或翻回主动权
3. 补一句可触发段评 / 本章说的态度句、冲突句或记忆动作

### 十一、下章接棒发虚

1. 把章末的新变量落到具体人、具体记录、具体后果
2. 删掉抽象总结，改成未完成动作或即将落地的危险
3. 若接棒对象与下章首场职责脱节，回写控制卡或最后一场场景职责

## 建议评分尺

建议至少评分以下 10 项，每项 10 分：

1. 章首抓力
2. 中段兑现
3. 现实锚点
4. 证据公平 / 规则边界
5. 对话承担度（声音差 / 塑人 / 联结）
6. 章末钩子
7. 追读反射弧与付费吸附
8. 微爽点 / 中爽点密度
9. 平台讨论度 / 推荐切口
10. 竞对差距与反超路径

### 快速判定口径

- 任一项 ≤ 6：该项视为明显短板
- `章首抓力`、`中段兑现`、`章末钩子` 任一 ≤ 5：建议返工
- `对话承担度` ≤ 5 且本章高占比靠对话推进：建议优先返修对话链
- `追读反射弧与付费吸附` ≤ 6，或 `微爽点 / 中爽点密度` ≤ 5：至少按 `P1.5` 处理
- `竞对差距与反超路径` < 7：综合评分默认不得高于 `8.4/10`
- `竞对差距与反超路径` < 6：综合评分默认不得高于 `7.4/10`
- 若没有量化差距卡（分差 + 证据 + 72 小时补丁），综合评分默认不得高于 `7.0/10`
- 总体均分 ≥ 8.5：更适合进入精修层，而不是结构返工层
- 若 `证据公平 / 规则边界` ≤ 5 或人物一致性直接冲突，优先按 P0 / P1 处理，不受均分掩盖
