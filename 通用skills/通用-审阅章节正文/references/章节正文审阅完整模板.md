# 章节正文审阅完整模板（通用骨架）

适用于：章节正文、正文章节、单章执行层审阅。

## 1. 通用层与题材层的拆分原则（强制）

- 本文件只承载跨题材复用的**章节正文审阅通用骨架**。
- 各题材 Skill 必须在此骨架上补写自己的“题材专项吸引力核查”“题材专项评分项”“题材化结论重点”。
- 题材 Skill **不得删减**本模板的通用主骨架，只能替换或补写题材专项章节。
- 报告默认按“章节正文读者可见层”书写；`## 作者有话说`、`## 章节后记`、润色说明等同文件共存，默认不单列为事故，除非它们越界混入正文或用户要求按终稿严判。
- 默认“可发布 / 可放行 / 不建议直接发布”等结论都只针对读者可见层；若本轮额外检查原始工作文件是否可裸发，必须单列“附加结论”，不得与正文质量结论混写。

## 2. 落盘位置与文件名格式（强制）

- 若作品存在分部：

`审阅意见/第X部/第Y卷/章节审阅报告_第X部_第Y卷_第Z章_[审阅日期].md`

- 若作品没有分部：

`审阅意见/第X卷/章节审阅报告_第X卷_第Y章_[审阅日期].md`
