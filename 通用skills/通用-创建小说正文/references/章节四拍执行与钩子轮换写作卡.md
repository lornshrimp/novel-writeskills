# 章节四拍执行与钩子轮换写作卡

> 来源口径：`网文技法.md`（标准章节模板、钩子轮换）+ `使用说明.md`（前后200字钩子）。

## 写中四拍（正文落稿必过）

```text
钩子（前200字）→ 蓄力（冲突加压）→ 回报（中段可感兑现）→ 留钩（后200字未闭合）
```

## 各拍执行要求

- **钩子**：前 200 字内至少出现冲突/悬念/异常/动作/对话中的 1 种。
- **蓄力**：中段必须可见压力上升，不得只做信息搬运。
- **回报**：本章至少 1 次可感知回报（关系、处境、信息、行动、身份其一）。
- **留钩**：后 200 字保留未闭合问题或新变量，禁止预告腔代替钩子。

## 钩子轮换（跨章）

避免连续 3 章同类钩子，推荐轮换：

```text
悬念钩 → 冲突钩 → 异常钩 → 动作钩 → 对话钩 → 悬念钩...
```

## 章内节奏样式（可直接套）

```text
前200：钩子
中段A：推进主线/支线
中段B：微爽点（小反转/信息差）
中段C：再加压或误判升级
后200：留钩
```

## 写中自检（最小清单）

- 前 200 字是否有效抓人。
- 中段是否有“值回票价”的明确回报。
- 后 200 字是否形成“下一章必须看”的驱动。
- 本章钩子类型是否与前两章过度重复。
