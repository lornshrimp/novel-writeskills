# 写作启动工具与创意阻力处理

适用于已经拿到大纲、设定、人物资料，却迟迟进不了正文；或者刚开写就被“空白页压力、怀疑、拖延、越想越不敢写”卡住的情况。

## 一个底线

- 初稿的第一目标不是完美，而是**离开空白页**。
- 如果你一直想等“最对的第一句”再开始，大概率会把正文卡死在门口。
- 但离开空白页之后，默认要尽快切换到“成稿模式”：继续按节拍往前推进，尽量在首轮把本章写到足够字数，不把“先写一点、后面再扩”当常规策略。

## 常见创意阻力识别

### 1. 拖延者

- 特征：总觉得还没准备够，还想再看资料、再查设定、再等等状态。
- 应对：先写出本章第一个可拍场景，离开空白页；但写出后要立刻顺势推进到后续节拍，不把“只写一个场景”当默认收工点。

### 2. 受害者

- 特征：总觉得限制太多、资料太杂、要求太高，因而迟迟不动。
- 应对：先写“在这些限制下，主角眼前最急的一件事”，把受限本身变成剧情压力。

### 3. 讲述者

- 特征：大脑里全是解释、总结、作者说明，迟迟不肯落镜头。
- 应对：强制把第一段改成动作、物件、对话或感官现场，不许先讲道理。

### 4. 批评家

- 特征：刚写两句就觉得差，边写边删，结果一页都过不去。
- 应对：首轮只准向前，不准回头精修；把评判动作延后到润色阶段。

### 5. 审判者

- 特征：总觉得自己写得不够像“真正的作家”，开写前先给自己定罪。
- 应对：先完成一个可读但不完美的场景，再判断它是不是值得继续改。

### 6. 反复无常的客户

- 特征：脑子里不断改要求，一会儿想写得更文学，一会儿又想更快节奏，导致路线不停切换。
- 应对：先钉死本章最重要的一项任务：抓人、推进、回报、钩子，四者本轮优先其一，其余后补。

## 五种启动工具

### 1. 自由写作

- 连续写 5–10 分钟，只回答一个问题：
  - 这章一开场，主角最先看见 / 听见 / 误判了什么？
- 不管句子漂不漂亮，先把素材倒出来。

### 2. 头脑风暴

- 围绕本章核心问题快速列单：
  - 他最怕什么？
  - 现在最容易出错的是什么？
  - 哪个物件最适合先入镜？
  - 哪句台词最先把关系顶歪？

### 3. 聚合

- 先把一个中心词写在中间，再往外扩：
  - 人物
  - 地点
  - 异常
  - 代价
  - 回报
- 适合卡在“素材很多，但不知道先写哪一个”时。

### 4. 封闭写作

- 在固定时段里只做一件事：写出一个场景单元，不同时打开十个方向。
- 封闭时间里不再追加新要求，只完成当前镜头。

### 5. 剪切—粘贴

- 若开头始终不顺，不要死守第一段。
- 可以先写中段的关键画面、对话热点或章末动作，再回头把真正适合的开头接出来。

## 启动工具的退出条件（强制）

- 启动工具的任务是帮助你离开空白页，不是给“短稿待扩”开绿灯。
- 一旦已经写出第一个稳定场景、一个中段热点或一个章末动作，就应立即回到节拍表，继续把本章推进到完整章节的长度与结构。
- 若此时已经能看出首轮天然撑不起目标篇幅，优先补场景职责、回报位和升级后果，不要等全文写完后再靠反复扩写救场。

## 废料场原则

- 临时删掉的句子、段落、镜头不要立刻丢弃。
- 先丢进“废料场”或临时区，后续可能转化为：
  - 作者有话说素材
  - 下一章伏笔
  - 平台派生版可用段落
  - 后记里的回指说明

  ## 别硬顶空白页：先回到结构底盘

  - 如果你已经不是“没灵感”，而是“有很多东西但不知道先写哪一个”，问题通常不在句子，而在结构定位。
  - 这时不要继续空想，也不要靠反复起稿碰运气，先确认本章 / 本场景在故事整体里的位置。

  ### 写前最低九部件

  先至少回答以下九个部件有没有着落：

  #### 四个连续部分

  1. 布局
  2. 反应
  3. 进攻
  4. 解决

  #### 五个里程碑

  1. 引发性起点
  2. 第一情节点
  3. 中间点
  4. 第二关键点
  5. 第二情节点

  - 你现在要写的这一章，至少要知道自己位于哪一部分、靠近哪个里程碑。
  - 若不知道，正文很容易写成“有内容但没方向”。

  ## 进度节奏表：抗卡顿的工程工具

  - 进度节奏表不是为了把创作写死，而是为了让你知道下一个场景为什么要出现。
  - 与其在正文里硬滚，不如先用 1–2 行写出每个节拍的任务。

  ### 最小用法

  若只写一章，可先压成 3–5 个节拍：

  1. 开场压力 / 异常
  2. 第一次反应 / 试探
  3. 中段回报 / 新信息 / 新阻碍
  4. 代价升级 / 误判修正
  5. 章末驱动 / 下一章接口

  ### 使用原则

  - 一个节拍先只承担一个主任务。
  - 先确认“为什么写这一场”，再决定“这一场怎么写得好看”。
  - 便利贴、卡片、列表、脑图都可以，重点不是形式，重点是顺序看得见。

  ## 什么时候必须先做节拍表

  出现以下任两项，先别继续硬写正文：

  - 写了很多，却说不清每一场的任务。
  - 知道素材多，但不知道先落哪一幕。
  - 中段总在横移，没有明确回报。
  - 章末总收成情绪句，没有真正驱动。
  - 改了很多次，问题却总像在别处。

## 写前最小启动问答

正式落正文前，至少答清以下四问：

1. 本章最先要解决 / 暴露的麻烦是什么？
2. 主角现在最想避免的代价是什么？
3. 第一场里最值钱的异常或动作是什么？
4. 我现在是要先写开头、热点、还是钩子？

答不出来，就先自由写作，不要继续空想。
