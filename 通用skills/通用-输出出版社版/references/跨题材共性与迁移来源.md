# 通用-输出出版社版｜跨题材共性与迁移来源

本文件记录 `通用-输出出版社版` 当前承接的跨题材共性来源，以及无损迁移期间的执行边界。

## 当前来源

- `都市悬疑/.github/prompts/输出出版社版.prompt.md`
- `异能志怪/.github/prompts/输出出版社版.prompt.md`

## 当前可稳定上收的出版社平台共性

- 强调文笔克制、逻辑自洽、出版可读性与审读友好
- 输出结构固定为“正文 + ## 作者有话说”
- 证据、制度与规则应以剧情体裁化方式呈现，不写成说明书
- 平台风格必须先于降相似度，不得先写网文稿再机械去网文腔

## 当前仍保留在题材层的内容

- 各题材的现实落点、民俗落点、人物命运牵引与题材危险感
- 题材专属的审读敏感边界与补丁

## 使用边界

- 命中 `通用-输出出版社版` 时，必须同时加载对应题材的 `题材名-输出出版社版`。
- 迁移期内若仍需回溯细节，以题材 refs 与源 Prompt 为补充。
