# 竞对分析执行清单

以下清单用于把“竞对分析”从模糊任务，收束为可执行的单书报告流程。

## 一、起步参数确认

- [ ] 题材已确认
- [ ] 目标平台已确认（未指定时默认起点中文网）
- [ ] `TopN` 的 `N` 值已确认（未指定时默认 `10`）
- [ ] 当前项目 / 当前作品已确认
- [ ] 输出目录已确认（未指定时默认 `竞对分析/`）
- [ ] 是否只做起点口径，还是要补充其他平台讨论材料，已确认

## 二、工作区维度提炼清单（必须先于外部检索）

### 2.1 必看来源类型

- [ ] 当前题材目录 `写作研究/`
- [ ] 当前题材目录下的相关 Skill / Prompt
- [ ] `通用skills/` 下与题材定位、故事面、人物、设定、章节执行、平台评估相关的 Skill
- [ ] 当前项目自身的大纲 / 分卷 / 正文 / 设定 / 人物 / 审阅报告（若存在）
- [ ] `竞对分析/竞对资料/` 下已有的竞对作品深入研究 / 章节正文摘要 / 资料摘记（若存在）

### 2.2 维度收束模板（建议先填）

- 分析维度：
  - 来自工作区的依据：
  - 为什么本轮必须看：
  - 预计看哪些外部证据：

- 分析维度：
  - 来自工作区的依据：
  - 为什么本轮必须看：
  - 预计看哪些外部证据：

- 分析维度：
  - 来自工作区的依据：
  - 为什么本轮必须看：
  - 预计看哪些外部证据：

> 默认把维度控制在 `8–12` 个。过少容易漏关键差异，过多容易把报告写成材料流水账。
>
> 若 `竞对分析/竞对资料/` 已有历史资料，先提取其中的搜索词、章节定位、问题清单与盲区记录，但后续仍要回到外部来源复核。

## 三、榜单检索清单

- [ ] 已按 `references/榜单检索与样本池确定规则.md` 确定榜单口径
- [ ] 已记录搜索词
- [ ] 已做至少一次交叉搜索
- [ ] 已做至少一次结果核验
- [ ] 已记录榜单时间、来源与不确定点
- [ ] 已写明为什么选择这组 TopN，而不是其他榜单

## 四、单书证据采集最小线

每本样本默认至少覆盖以下证据层：

| 证据层 | 最低要求 | 备注 |
| --- | --- | --- |
| 榜单 / 作品基础信息 | 至少 1 条 | 榜单页、作品页、简介页等 |
| 目录 / 卷名 / 章节命名 | 至少 1 条 | 需要能看出结构承诺与读者预期管理 |
| 正文 / 试读 / 公开章节片段 | 至少 1 条 | 能直接看到开篇、节奏、文笔或信息差经营 |
| 设定 / 世界观 / 机制说明 | 尽量补齐 | 可来自作品页、访谈、总结或读者整理 |
| 书评 / 读者评论 / 讨论 | 至少 2 条 | 尽量覆盖不同来源，避免只采单一口碑 |
| 推荐文 / 盘点 / 拆解 / 读后感 | 至少 1 条 | 用于交叉验证外部认知 |

> 如果某一层证据客观难找，不允许直接跳过；必须写明“搜过什么、没搜到什么、因此不确定什么”。
>
> 若官方站点因验证码、反爬、登录门槛或频控导致正文 / 目录证据不可稳定获取，允许改用**更稳定、可核验的替代来源**（如官方试读、作者公开物料、媒体报道、书评短引文、论坛讨论、读后感、剧情总结等）；**不要把来源不明、不可稳定核验的全文镜像页或转载聚合页当作补正文的核心证据。**
>
> 若工作区 `竞对分析/竞对资料/` 已有该作品的深入研究、章节摘要或历史摘记，应先把其中可复核点转成搜索词 / 证据任务清单；它们是补充参考，不是替代证据。

## 五、单书报告必答问题

每份单书报告至少要回答：

1. 这部作品为什么是强样本，而不是普通热门书。
2. 它的核心竞争优势到底是什么。
3. 这些优势如何同时体现在：
   - 书名 / 简介 / 目录
   - 开篇 / 正文片段
   - 设定 / 机制 / 结构
   - 书评 / 评论 / 读后感
4. 哪些优点值得当前项目学习。
5. 哪些优点只是它的个体条件、时代窗口或作者存量，不应机械照搬。
6. 还有哪些盲区没有证据，不该强下结论。

## 六、证据写作规范

- 所有“强点”判断都要尽量落到具体证据，不写空泛夸奖。
- 所有“可学点”都要写成可执行动作，而不是形容词。
- 所有“不可照搬点”都要写明原因：平台机制、题材差异、作者存量、更新节奏、个人声线、读者预期等。
- 若引用了 `竞对分析/竞对资料/` 中的既有深入研究、章节摘要或资料摘记，要标明它属于工作区补充参考，而不是外部核心证据。
- 尽量区分：
  - **事实**：明确可见的榜单、目录、正文、评论信息；
  - **推断**：基于多条证据形成的判断；
  - **假设**：证据不足但合理怀疑的方向。

## 七、版权与引用边界

- [ ] 不复制大段正文
- [ ] 不整段搬运长评论
- [ ] 只保留必要短引文或改写概述
- [ ] 所有引文都只服务于分析，不替代原文阅读体验
- [ ] 官方源受阻时，已优先查找更稳定、可核验的替代来源
- [ ] 未把来源不明或不可稳定核验的全文镜像页 / 转载聚合页当作核心证据

- [ ] 文件名包含作品名与日期
- [ ] 报告已使用 `references/竞对分析报告模板.md`
- [ ] 每份报告都写明了证据来源类型与判断盲区
- [ ] 每份报告都写明了“值得学什么 / 不该照搬什么 / 对当前项目有什么启发”
