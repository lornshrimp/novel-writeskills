# 通用-输出新浪微博版

这是平台共性本体，负责承接“新浪微博版输出”的跨题材共性规则。

## 适用场景

- 把任意题材章节改写成更适合新浪微博传播语境的版本。
- 需要强化短促抓点、传播性、话题切口与即时情绪反馈。
- 需要为 `题材名-输出新浪微博版` 提供统一的平台骨架。

## 本层职责

- 维护新浪微博平台的跨题材共性规则。
- 统一承接微博版输出的结构约束、标题倾向、传播压缩度与风格门禁。
- 为后续新增题材保留直接复用入口。

## 继续读取的 references

- `references/跨题材共性与迁移来源.md`
- `references/平台共性执行细则.md`
- `references/分节级补救映射与详细规则回填.md`

## 执行时必须先做什么

1. 先读取对应题材的 `题材名-输出新浪微博版` Skill。
2. 先锁定零新增事实、体裁结构、路径映射、字数门禁与标题同步规则。
3. 再执行微博平台的压缩抓点、传播切口与即时情绪强化。
4. 最后才处理降相似度，不得为了降重写成评论帖腔。

## 强制要求

- 命中本技能时，必须同时加载对应题材的 `题材名-输出新浪微博版` Skill。
- 题材特有规则不得回写到本文件中平行维护。
- 在无损迁移完成前，不得删除源 `输出新浪微博版.prompt.md` 中尚未完成映射的内容。
