# 章末钩子检查清单

## 一、范围

- [ ] 只处理最后一段到最后约 150–300 字
- [ ] 没误伤上文主体推进

## 二、具体性

- [ ] 结尾是否落到了具体动作、具体后果或具体新变量？
- [ ] 危险是否有可指认载体，而不是空泛形容？

## 三、驱动性

- [ ] 读者是否能明确感觉到下一章必须承接什么？
- [ ] 这一针是否比本章中段更危险或更尖？
- [ ] 读者是否会自然追问“接下来呢”？

## 四、自然性

- [ ] 钩子是否顺着本章既有信息长出来，而不是硬插进去？
- [ ] 是否保持了人物动机、视角和口径的一致？

## 五、节奏

- [ ] 没有空泛感慨收尾
- [ ] 没有解释性总结收尾
- [ ] 最后一句有落针感，而不是松掉

## 六、红线

- [ ] 没有直接剧透下一章
- [ ] 没有新增关键事实
- [ ] 没有把后记语气混进正文结尾
