# 通用-输出番茄版｜跨题材共性与迁移来源

本文件记录 `通用-输出番茄版` 当前承接的跨题材共性来源，以及无损迁移期间的执行边界。

## 当前来源

- `异能志怪/.github/prompts/输出番茄版.prompt.md`
- `都市悬疑/.github/prompts/输出番茄版.prompt.md`

## 当前可稳定上收的番茄平台共性

- 免费阅读平台语境下的快反馈、强钩子、短段落、高完读率节奏
- 输出结构固定为“正文 + ## 作者有话说”
- 标题需更直给、更具冲突与追读动机
- 字数门禁、作者有话说门禁、反注水门禁、相似度门禁需要与多平台 SOP 协同
- 平台风格必须先于降相似度，不得先写通用稿再改番茄壳

## 当前仍保留在源 Prompt 的详细规则

在无损迁移完成前，以下内容仍以源 Prompt 正文为详细依据：

- 各题材在番茄平台下的具体研究补强条款
- 题材特有的禁行项与补丁口径
- 精细化的章内节拍、标题模板、作者有话说口吻限制

## 使用边界

- 命中 `通用-输出番茄版` 时，必须同时加载对应题材的 `题材名-输出番茄版`。
- 若当前任务仍处于迁移试点阶段，必须继续回看源 Prompt 与试点台账，不得假设本文件已经完全替代旧 Prompt。
