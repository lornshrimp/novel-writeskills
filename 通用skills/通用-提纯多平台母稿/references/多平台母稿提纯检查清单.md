# 多平台母稿提纯检查清单

## 一、骨架保留

- [ ] 事件链完整
- [ ] 角色目标与代价完整
- [ ] 关系变化完整
- [ ] 规则与证据关键点完整
- [ ] 章首抓力、中段回报、章末钩子仍在

## 二、核心稳定性

- [ ] 关键事实是否清楚且不含糊？
- [ ] 关键证据 / 规则关键点是否有载体、有动作、有阻力？
- [ ] 时间、编号、人物关系是否稳定可复用？

## 三、人物压力

- [ ] 主角本章急什么、怕什么、会失去什么，是否足够具体？
- [ ] 对手或阻力是否不是纯概念，而是有实际门槛和口径？

## 四、提纯效果

- [ ] 单一平台腔体下降
- [ ] 重复补偿同一信息的句群减少
- [ ] 解释性外壳变少
- [ ] 文本更适合继续往多平台拉开差异

## 五、可分化空间

- [ ] 哪些层面后续可以安全按平台改，是否已经明确？
- [ ] 当前母稿是否已经被单一平台腔锁死？
- [ ] 有没有被磨成一份过于中性的白稿？

## 六、源头纪律

- [ ] 当前处理对象是否确实是源母稿？
- [ ] 有没有拿平台稿回修母稿，导致口径混杂？
- [ ] 提纯完成后，是否仍然适合作为多平台流程的上游源稿？

## 七、红线

- [ ] 没有删关键剧情
- [ ] 没有删关键证据或规则边界
- [ ] 没有把文本提纯成无味白底稿
- [ ] 没有顺手把它改成某个平台终稿
