# 腾讯专栏：YY"缺什么补什么"原理与爽点增强方法增补

来源：腾讯写作专区「新手专区」专栏  
原文：《写作技巧 | 写小说可以YY什么？》（33.3万阅读）、《网文的爽点有哪些？怎么写才能更爽？》（13.8万阅读）  
落库日期：2025年

## P0 规则

### 1. YY点核心原理：缺什么补什么

> 寻找YY点，用一句话来概括，就是"缺什么补什么"。缺什么，这是针对于读者的，但作者也同样可以把自己当成读者的一员。

**规则：**
- YY点 = 读者现实生活中得不到、满足不了的愿望
- 作者最简单的方法是审视自己的人生遗憾，那里就是最真实的YY点来源
- 不必研究市场数据，自己从小到大的遗憾和渴望，已足以推导出大量YY点

### 2. 基础YY类型（三大普遍需求）

1. **变强**（战力、地位、能力提升）—— 绝大多数男性读者的共同愿望
2. **变有钱**（财富积累、阶层跃升）—— 几乎所有读者的梦想
3. **异性青睐**（校花/男神的关注）—— 得不到所以渴望，是校园类爆款的根源

### 3. 扩展YY类型（不止这三样）

| YY方向 | 对应类型/场景 |
|--------|-------------|
| 尊重与认可 | 都市流、职场流 |
| 长生不老 | 仙侠、修真 |
| 飞行与异能 | 玄幻、都市异能 |
| 奇遇与金手指 | 穿越、系统文 |
| 国家强盛、民族崛起 | 历史文、军事文 |
| 从人人看不起→俯视世界 | 各类型都适用 |

> "这种种的缺憾，才造就了读者们强烈的阅读需求，也造就了一个又一个的经典类型。"

### 4. 爽点原理：把生活中令自己爽的事放大写进书里

> 比如说，你的新书大受欢迎，好评如潮，订阅狂涨，你开心不开心？之前有别的作者怀疑你的能力，现在你成绩远胜于他，你开不开心？这样的剧情，写到书里，当然就成为了爽点。

**规则：**
- 爽点没有固定公式，但原理是：找到自己生活中真实感到爽的瞬间，放大对比，写入书中
- 爽点够不够爽，答案在生活经验里，不需要外部教条

### 5. 四种增强爽度的方法（对比维度）

1. **起点对比**：主角越惨的起点 → 成功时越爽（穷光蛋打赢富豪 > 富豪打赢富豪）
2. **观众见证**：单人偷着乐 < 无数人羡慕嫉妒恨围观（加观众倍增爽感）
3. **期待反转**：之前所有人看不起主角 > 大家觉得主角成功理所当然（逆袭 > 顺势）
4. **相对坐标**：同样的成就在不同时代/难度背景下价值不同（含金量影响爽度）

### 6. 设计口径总结

> 不要只想到"让主角越来越强"——那只是最基础的一种YY。  
> 真正的题材卖点 = 锁定一个读者普遍"缺"的点 → 通过主角的经历持续满足这个"缺"。
