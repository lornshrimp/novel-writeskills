# 创新组合写法模板

## 组合来源层（新增）

- `职业切口层`：这本书为什么必须由这个职业 / 身份去碰这件事。
- `社会压力层`：这种异常或冲突为什么只能在这种城市 / 制度缝隙里长出来。
- `规则发动层`：主角一旦处理它，靠什么方法推进，又会付出什么代价。

```markdown
## 创新组合说明

- 职业切口：
- 社会压力源：
- 母体来源：
- 规则发动方式：

### 组合逻辑
1. 为什么这个职业最容易撞上这种异常：
2. 为什么这种异常只能在这种城市 / 制度缝隙里长出来：
3. 为什么主角处理它时必须付出代价：
4. 这套组合与常见模板的最大区别：
```

## 最小创新工件（新增）

```markdown
【职业切口层】
【社会压力层】
【规则发动层】
【当前判定】Green / Yellow / Red
```

## Green / Yellow / Red 创新判定（新增）

- `Green`：职业切口、社会压力、规则发动三层咬合，创新不是空拼盘。
- `Yellow`：组合思路在，但某一层仍偏抽象，离可持续供血还差一口气。
- `Red`：只是把几个标签堆在一起，没有形成真正独有的持续冲突发动机。
