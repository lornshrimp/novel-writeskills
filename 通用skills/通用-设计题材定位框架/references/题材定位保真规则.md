# 题材定位保真规则

## 一、先回答四个问题，再谈大纲

1. **第一眼新鲜点**：读者为什么会点开。
2. **主角非入局不可的原因**：不是因为剧情需要，而是职业、现实处境、缺陷或旧债逼他下场。
3. **主线谜团的耐久度**：为什么足以撑长篇，而不是 10 章就泄底。
4. **持续回报**：中后段回报不是同一种刺激重复，而是信息、规则、关系、代价中的至少两类轮换升级。

## 二、定位的五个钉子

### 1. 平台主口径

必须先判断主平台骨架是什么，以及辅平台只借什么、不借什么。

### 2. 主角抓手

主角必须至少有一项“别人不会、他会”的抓手，而且抓手必须能转化成方法，而不是只有头衔。

### 3. 母体来源

机制 / 超常 / 核心异常不要从真空里掉下来，至少锁定一个主母体，必要时允许一个辅母体，但不宜过多。

### 4. 现实锚点

每条核心线至少落一项现实锚点：职业流程、制度摩擦、地域空间、家庭 / 债务 / 名誉压力、现代生活针脚等。

### 5. 代价锚点

每次关键推进至少要能落到：身体、关系、职业、名誉、法律 / 制度、心理中的一类或多类。

## 三、创新组合写法

避免空拼盘，改写成：

`职业切口 × 社会压力 × 母体来源 = 本书独有的持续冲突发动机`

## 四、总承诺口径

定位时必须能说清：

- 现实锚点可感
- 规则 / 能力 / 机制可复盘
- 成长 / 升级 / 供血有回报

## 五、卷首承诺口径

若定位涉及开书、卷首或新卷启动，必须额外回答：

- 前几章用什么异常 / 冲突 / 关键困境抓住读者
- 第一条可追线索是什么
- 第一种可见回报是什么
- 这一卷如何连续 3–5 章维持吸引力

## 六、现实议题显影与说教隔离

- 现实议题可以存在，但必须通过事件链、证据链、人物命运与制度摩擦显影。
- 禁止把题材定位写成观点先行、议题先行的提纲。

## 七、边界与禁区

以下任一项出现，说明定位仍未钉死：

- 机制像万能工具，没有触发条件、代价和边界
- 职业只是头衔，不提供方法论抓手
- 场景只有地名，没有制度摩擦和生活针脚
- 平台定位含糊，导致节拍、回报、钩子无法统一

## 八、最终验收口径

一个合格的题材定位，至少应让后续总纲与正文都能明确回答：

- 为什么读者会点开
- 为什么主角必须入局
- 为什么这个故事只能长在这种职业 / 城市 / 制度缝隙里
- 为什么机制既危险又可复盘
- 为什么读者追到中后段仍能持续获得回报
