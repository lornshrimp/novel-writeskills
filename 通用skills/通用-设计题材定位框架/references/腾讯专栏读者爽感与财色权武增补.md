# 腾讯专栏：读者爽感与财色权武增补规则

来源：《三个问题：读者喜欢什么？爽么？爽在哪里？》
作者：阅文历史频道编辑 徐徐
阅读量：23.8万

## P0规则：读者爽感三问框架

定义题材爽点前必须先走完三问：

**第一问：读者喜欢什么？**
- 先分析目标读者群的属性和标签
- 不同分类读者口味截然不同，必须针对具体分类具体分析
- 不要凭感觉，要从读者身上找答案

**第二问：爽么？**
- 喜欢的才会觉得爽；想要的得不到的，才会YY
- 如果连"什么是爽"都没弄清楚，无法写出让人觉得爽的内容
- 爽不爽，读者说了才算；答案要从读者身上找

**第三问：爽在哪里？**
- 两层自问：①我写的，爽在哪里？②别人（火书）写的，爽在哪里？
- 对比分析才能知道自己写的爽点是否到位

## P0规则：历史读者群五属性（可迁移到同类分析）

| 属性 | 描述 | 对创作的要求 |
|------|------|-------------|
| 成熟 | 对人和社会有自己见解，重视逻辑和人设合理性 | 情理之中，意料之外；无脑流很难起来 |
| 情怀 | 华夏情怀/人物情怀/主义情怀/热血情怀 | 要有能激活情怀的核心元素 |
| 考究向往 | 对特定朝代有深度代入感，考究党居多 | 适合专项定位某朝代深度写 |
| 男性主导 | 男性读者偏好"财色权武"四字 | 选准对应爽感写法（见下） |
| 部分女读者 | 以情感和人设为主 | 与"成熟"属性相辅相成 |

## P0规则：财色权武四类爽点写法

**财**（财富/事业爽）
- 核心YY：穿越后以现代手法在古代经商，获利并获得达官贵人认可
- 爽感来源：才智变现、被同行推崇、打破原有秩序

**色**（情感/风月爽）
- 核心YY：以才博取佳人芳心；欲拒还迎、调戏互动中产生爽点
- 限定：不是黄色的色，是男女暧昧关系中足智多谋的"谓爽"
- 延伸：女神分类（高冷/萝莉/外冷内热等）对应不同写法

**权**（地位/权势爽）
- 核心YY：从草根到高官/大将军/皇帝
- 常见路径：科举做文官、行兵打战、独得皇上恩宠
- 爽感写法：微服私访记的打脸套路；弄掉把自己当傀儡的奸臣；扩张疆土

**武**（武力/绝对实力爽）
- 核心YY：绝对武力值镇压一切，"既然能用绝对实力镇压的，那就别废话了"
- 代表类型：无敌流
- 延伸：自己牛或麾下跟随者牛，都是实力展示

## P0规则：理论与落地的关系

- 理论终究是理论，写得出来效果才是重点
- 同样的打脸，用不同手法爽感不同；找到更顺手的方式更重要
- 看书看少了、或者看书只是在看书而不思考，是大多数作者缺少判断力的根源
