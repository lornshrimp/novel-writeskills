# 腾讯专栏：文风与平台匹配增补规则

来源：《嘿，编辑，你跟我说说，为啥我的书没人看啊》
作者：作家助手（多编辑圆桌：木棉/北河/夜宵/虎牙/灰灰/松露/熊猫/星辰/麒麟/晴天/汤圆/水荇）
阅读量：38万

---

## P0规则：文风与所发站点必须匹配

**核心诊断路径（编辑松露）：**
- 先检查：文风是否与所发站点读者偏好匹配？
- 文艺风、成熟风放到爽文站 = 天然劣势，成绩不行是正常的
- 不要在错误的平台用错误的风格写作；改风格或换平台，二选一

**具体诊断顺序（编辑星辰）：**
1. 检查文风是否与站点匹配
2. 检查书名+简介+开篇（第一眼印象三件套）
3. 对比分析同类型热销书

---

## P0规则：成绩不好时的自查清单（多编辑共识）

| 自查项 | 说明 |
|--------|------|
| 稳定更新 | 第一要务，几乎所有编辑都强调 |
| 文风与站点匹配 | 不匹配是天然劣势 |
| 书名+简介+开篇 | 第一眼印象，最容易影响点击 |
| 爽点是否到位 | 剧情设置是否合理，爽点是否把握到位 |
| 多看火书/热销书 | 扫榜单、扫火书、扫红书，学习成功案例 |

---

## P0规则：质量提升三步公式

**多看 + 多写 + 多总结**

- 多看：扫榜单，扫火书，扫红书（热销书）
- 多写：坚持，新人成绩差是正常的，靠量换质
- 多总结：从看书到思考的升级；"没有不会总结的成功者，只有不断原地摔倒的失败者"

---

## P0规则：平台风格匹配前的判断

- 题材/文风定位后，应先确认目标平台读者偏好是否一致
- 若流行题材和自己书的风格不匹配，优先建议写流行题材（编辑松露建议）
- 不能在读者基础少的平台发自己特定风格的书，再抱怨没读者
