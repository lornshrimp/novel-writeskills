# 场景去模板腔清单

## 一、使用目的

场景最常见的模板腔，不在结构，而在句子表面：

- 抽象词堆太多
- 形容词一排排铺
- 所有段落都像在提醒读者“这里很紧张”
- 角色总在感受，几乎不做事

## 二、高危模板腔信号

### 1. 抽象压迫词重复出现

### 2. 空气句太多

### 3. 角色总在“感到 / 觉得 / 意识到”

### 4. 句长过于平均

### 5. 结尾总是概括式总结

## 三、替换原则

### 原则 1：一个抽象词，尽量换成一个物理细节

### 原则 2：一个情绪判断，尽量换成一个动作停顿

### 原则 3：一个结论句，尽量换成一个新变量

## 四、推荐保留的人味来源

- 具体生活痕迹
- 人物的职业反应
- 说话方式与小动作习惯

## 五、快检问题

- 如果删掉“压抑 / 紧张 / 不安 / 危险”这些词，这场还成立吗？
- 如果把“他感到 / 她觉得”删掉，这场还看得懂吗？
- 这一场有没有一个读者能记住的具体物理点？

## 六、验收口径

一个去掉模板腔的场景，应满足：

- 少一点空词
- 多一点现实针脚
- 少一点“作者告诉你很紧张”
- 多一点“读者自己看出局势正在变坏”
