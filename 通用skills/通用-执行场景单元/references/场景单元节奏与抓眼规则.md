# 场景单元节奏与抓眼规则

## 抓眼入口优先级

优先用以下入口之一开场：

1. 反常细节：日常场景里出现不该出现的信号。
2. 迫近风险：再不动作就会错过窗口。
3. 情绪缺口：角色刚失去、刚发现、刚误判。
4. 未完成动作：角色带着半截计划入场。

若是新章 / 新场景起手，优先遵循：

- 前 7 行内至少给出一个“我为什么现在必须看下去”的理由。
- 前 300 字内必须把第一处有效刺激抛出来，不能一直热机。
- 若当前场景承担章首功能，优先写成“日常常态 + 第一处反常 + 立刻动作”。

## 节奏稳定规则

- 前 300 字应尽快抛出当前场景的核心矛盾。
- 关键紧张场景优先短句与动作句，尽量减少长段解释。
- 感官描写优先服务危险感、空间压迫感与状态变化，不为堆辞藻而堆。
- 限知视角优先：尽量让读者只知道角色此刻能看到、听到、摸到的东西。
- 若场景带职业性或行规，优先把专业判断压成一句短判断，再用动作链兑现。
- 快节奏场景优先使用“动作 → 反馈 → 新动作”短回路，不要在高压段落中穿插大段背景史。
- 若当前场景是单元案入口，最好在本场结尾前留下“更大问题”的影子，而不是只结掉本场小麻烦。

## 现实锚点补充

- 异常最好长在熟悉空间里：楼道、码头、值班室、出租屋、便利店门口、旧车库。
- 场景里尽量保留 1 个日常噪声或现实器物，用来抵住悬浮感。

## 信息差抓点

- 旁人看见了怪，主角看懂了规矩，是高价值抓点。
- 信息差必须落到“先判断 → 再动作 → 再后果”，否则只会显得主角在说教。
- 若主角看懂规则的代价过低，读者会觉得廉价；最好让信息差先救一命，再引出新麻烦。

## 场景推进四件套

- 目标
- 阻力
- 动作
- 后果

缺任何一项，场景都容易发飘。

## 章首常用公式

### 公式 A：日常被打断

`角色在做熟悉动作 → 某个细节不对 → 角色复核 → 本场冲突成立`

### 公式 B：危险已经在倒计时

`角色晚一步就会错过窗口 → 立刻行动 → 在行动里发现新异常`

### 公式 C：职业判断先压场

`旁人慌乱 / 误判 → 主角一句话点破规矩 → 按流程做第一步 → 局势暂稳但更大问题浮出`
