# 场景单元节奏与抓眼规则

## 抓眼入口优先级

优先用以下入口之一开场：

1. 反常细节：日常场景里出现不该出现的信号。
2. 迫近风险：再不动作就会错过窗口。
3. 情绪缺口：角色刚失去、刚发现、刚误判。
4. 未完成动作：角色带着半截计划入场。

若是新章 / 新场景起手，优先遵循：

- 前 7 行内至少给出一个“我为什么现在必须看下去”的理由。
- 前 300 字内必须把第一处有效刺激抛出来，不能一直热机。
- 若当前场景承担章首功能，优先写成“日常常态 + 第一处反常 + 立刻动作”。

## 节奏稳定规则

- 前 300 字应尽快抛出当前场景的核心矛盾。
- 关键紧张场景优先短句与动作句，尽量减少长段解释。
- 感官描写优先服务危险感、空间压迫感与状态变化，不为堆辞藻而堆。
- 限知视角优先：尽量让读者只知道角色此刻能看到、听到、摸到的东西。
- 若场景带职业性或行规，优先把专业判断压成一句短判断，再用动作链兑现。
- 快节奏场景优先使用“动作 → 反馈 → 新动作”短回路，不要在高压段落中穿插大段背景史。
- 若当前场景是单元案入口，最好在本场结尾前留下“更大问题”的影子，而不是只结掉本场小麻烦。

## 现实锚点补充

- 异常最好长在熟悉空间里：楼道、码头、值班室、出租屋、便利店门口、旧车库。
- 场景里尽量保留 1 个日常噪声或现实器物，用来抵住悬浮感。

## 信息差抓点

- 旁人看见了怪，主角看懂了规矩，是高价值抓点。
- 信息差必须落到“先判断 → 再动作 → 再后果”，否则只会显得主角在说教。
- 若主角看懂规则的代价过低，读者会觉得廉价；最好让信息差先救一命，再引出新麻烦。

## 场景推进四件套

- 目标
- 阻力
- 动作
- 后果

缺任何一项，场景都容易发飘。

## 行动—反应—继续行动短回路

场景不要只堆动作，还要有基本韵律：

1. 角色采取行动
2. 遭遇阻力 / 得到反馈
3. 出现情绪或判断反应
4. 角色决定下一步继续行动

高压场景里，这个回路可以很短；低压场景里，也不能整段只停在反应而不往前走。

## 场景热点规则

每个场景都应有一个**热点**：

- 可能是一句对话
- 可能是一个动作瞬间
- 可能是一次发现 / 暴露 / 失手

若连热点都找不出来，这场戏大概率不值得保留。

若热点是一句对话，这句对话至少应同时承担以下两项：

- 把关系顶歪：让读者马上感觉谁压谁、谁躲谁、谁在抢口径
- 把人物露出来：让说话方式暴露职业惯性、熟悉度、脾气或代价观
- 把读者钩住：让读者立刻产生认同 / 同情 / 喜爱 / 好奇中的至少一种

不要把场景热点写成只有信息、没有人物压力的“通知句”。

修剪方法：

1. 先圈出热点
2. 再回看热点前的铺垫是否每句都必要
3. 能删的热机就删，直到热点前只剩真正需要的台阶

## 紧张程度表（0–10）

写场景时可粗估紧张度：

- `0–2`：过于平，通常不宜单独成场
- `3–5`：过渡 / 铺陈 / 关系缓冲区
- `6–8`：大多数有效场景的工作区
- `9–10`：重大场景或高潮区，不宜滥用

常规模式：

- 从较低值进入
- 逐步抬升
- 在结尾前达到本场高点

## 展示区与说明区

- 当紧张度越过 `5`，优先进入**展示区**：动作、对话、体感、即时后果都应更具体。
- 当紧张度低于 `5`，可以保留少量说明，但若整场长期停在说明区，说明这场戏要么写虚了，要么该删。

## 对话链接入规则

- 若本场核心推进靠对话完成，默认按 `通用-执行对话冲突` 的标准施工，而不是把对话只当成解释工具。
- 对话应至少交出三样东西：
  - **声音差**：删掉提示语，也能大致分出谁是谁
  - **塑人**：能听出人物在护什么、怕什么、想抢什么
  - **联结**：能让读者对至少一个人物形成跟随理由
- 对话之后最好立刻接动作、反馈或后果，避免整段只剩口头拉扯。
- 若场景里有三人以上同场说话，优先拉开他们的句长、措辞、回避方式与发火方式，不要满场统一作者腔。

## 推手收尾

场景结尾不要无声散场；尽量留下至少一项：

- 预告中的危险
- 重大决定
- 未答问题
- 新变量
- 尚未落地的动作

读者翻下一段，往往不是因为场景“结束得完整”，而是因为它结束得“不让人甘心”。

## 章首常用公式

### 公式 A：日常被打断

`角色在做熟悉动作 → 某个细节不对 → 角色复核 → 本场冲突成立`

### 公式 B：危险已经在倒计时

`角色晚一步就会错过窗口 → 立刻行动 → 在行动里发现新异常`

### 公式 C：职业判断先压场

`旁人慌乱 / 误判 → 主角一句话点破规矩 → 按流程做第一步 → 局势暂稳但更大问题浮出`
