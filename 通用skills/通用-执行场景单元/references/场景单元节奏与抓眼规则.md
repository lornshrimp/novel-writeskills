# 场景单元节奏与抓眼规则

## 抓眼入口优先级

优先用以下入口之一开场：

1. 反常细节：日常场景里出现不该出现的信号。
2. 迫近风险：再不动作就会错过窗口。
3. 情绪缺口：角色刚失去、刚发现、刚误判。
4. 未完成动作：角色带着半截计划入场。

## 节奏稳定规则

- 前 300 字应尽快抛出当前场景的核心矛盾。
- 关键紧张场景优先短句与动作句，尽量减少长段解释。
- 感官描写优先服务危险感、空间压迫感与状态变化，不为堆辞藻而堆。
- 限知视角优先：尽量让读者只知道角色此刻能看到、听到、摸到的东西。
- 若场景带职业性或行规，优先把专业判断压成一句短判断，再用动作链兑现。

## 现实锚点补充

- 异常最好长在熟悉空间里：楼道、码头、值班室、出租屋、便利店门口、旧车库。
- 场景里尽量保留 1 个日常噪声或现实器物，用来抵住悬浮感。

## 信息差抓点

- 旁人看见了怪，主角看懂了规矩，是高价值抓点。
- 信息差必须落到“先判断 → 再动作 → 再后果”，否则只会显得主角在说教。

## 场景推进四件套

- 目标
- 阻力
- 动作
- 后果

缺任何一项，场景都容易发飘。
