## P0 规则：对比是最强情节设计技巧

### 核心定论

所谓"笔力不足"，本质上是**情节设计问题**，而不是文字功底问题。

对比是网文中最重要的情节设计技巧：**几乎所有情绪效果都能通过对比设计来实现**。

### 对比的两种类型

| 类型 | 说明 | 典型效果 |
| --- | --- | --- |
| 正衬 | 凸显相同方向（A强→B更强） | 震撼感、层层递进 |
| 反衬 | 突出对立差异（弱小↔坚决，贫穷↔拒绝作恶） | 热血、感动、守诺、善良 |

### 情绪目标 → 对比设计速查

| 情绪目标 | 对比设计方法 |
| --- | --- |
| 热血 | 敌人强大（正衬铺垫）+ 主角弱小但意志坚决（反衬）；多层对比叠加效果更强 |
| 搞笑 | 高端/严肃场合 + 极度反差的低端行为（如《修真聊天群》用电饭锅炼丹） |
| 出风头 | 众人轻视（铺垫）+ 主角展现绝对实力（落差产生爽感） |
| 泄愤 | 对方嚣张/霸道（铺垫）+ 主角完全碾压打脸（落差释放愤怒） |
| 守诺 | 放弃滔天利益（代价）+ 仍赴约（凸显守诺的价值） |
| 感情 | 艰难环境（铺垫）+ 相互扶持（反衬温情） |
| 善良 | 极度贫困（反衬）+ 拒绝不义之财（凸显善良含金量） |

### 多层对比案例（《修真四万年》火车情节）

火车情节用了多层嵌套对比实现超强热血感：
1. 强敌（三级/四级）vs 主角（底层一级）→ 基础压迫感
2. 同伴/民众的恐惧绝望 vs 主角无惧向前 → 反衬主角意志
3. 逐级升级对比：一胜 → 更强敌出现 → 再胜 → 更强 → 循环拉升 → 最终爆发

→ 多层对比叠加时，情绪效果呈几何倍增长。

## 场景设计时的对比使用清单

执行场景单元前，检查是否充分使用了对比：

- [ ] 本场景的情绪目标是什么（热血/搞笑/煽情/出风头/守诺）？
- [ ] 是否设计了足够的铺垫层（正衬/反衬），为情绪高点蓄力？
- [ ] 如果是热血/出风头类场景，是否增加了多层嵌套对比而非单次落差？
- [ ] 搞笑场景：是否利用了情境反差（高端场合 + 低端行为）？
- [ ] 是否避免了"直接写情绪"（如"他热血沸腾"）而改用对比来让读者自己感受到？
