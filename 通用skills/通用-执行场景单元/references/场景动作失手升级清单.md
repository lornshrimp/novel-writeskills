# 场景动作失手升级清单

## 一、使用目的

动作失手不是拖时间，而是升级器。

真正有效的场景里，失手之后局势必须更坏、更紧或更偏。

## 二、四类高效失手

### 1. 看了，但没看清

适合升级成：

- 信息缺口
- 误判

### 2. 试了，但没反馈

适合升级成：

- 风险升级
- 规则显影

### 3. 留了证，但证不稳

适合升级成：

- 证据入场
- 新风险

### 4. 退了，但没退干净

适合升级成：

- 关系张力
- 风险升级

## 三、失手后的标准路径

推荐节拍：

`动作失手 → 判断卡住 / 判断出错 → 新后果出现 → 被迫换动作`

## 四、失手与下一动作速配

| 失手类型 | 最适合转出的后果 | 最适合逼出的下一动作 |
| --- | --- | --- |
| 看不清 | 信息缺口 / 误判 | 再对照 / 换角度 / 求证 |
| 没反馈 | 风险升级 / 规则显影 | 再试一次 / 改用别的方法 |
| 留证不稳 | 新风险 / 证据入场 | 备份 / 隐藏 / 补证 |
| 退不干净 | 关系张力 / 风险升级 | 解释 / 掩饰 / 立刻返回 |

## 五、红线

- 不要让失手只是拖时间
- 不要连续多次失手却没有局势变化
- 不要失手以后马上靠作者解释兜住
- 不要把失手写成主角降智

## 六、验收口径

一个合格的动作失手，应满足：

- 失手有原因
- 失手带来更坏局势
- 角色不得不换动作或换判断
- 场景因此更紧，而不是更水
