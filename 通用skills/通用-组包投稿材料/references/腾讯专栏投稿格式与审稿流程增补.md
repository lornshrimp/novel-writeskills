# 腾讯专栏：投稿格式与编辑审稿三环节增补

来源：腾讯写作专区「新手专区」专栏  
原文：《入门指导——如何打动编辑》（22.4万阅读）  
落库日期：2025年

---

## P0 规则

### 1. 编辑审稿三环节（从外到内，越往前越重要）

> 实际的流程很简单：首先是观察作品的各种外观，包括书名笔名、作品简介、类型题材、版面段落、基本数据；然后是作品的表面文章，主要是文字水准和作品开局；最后才是正文内容的速读。

**三环节优先级（做好表面文章比正文内容更重要！）：**
- **环节1：外观表面**（书名、简介、类型、排版段落）→ 最先判断，最影响结果
- **环节2：文字开局**（文字水准 + 开局吸引力）→ 环节1平平时才认真看
- **环节3：正文速读**（人物、情节、创意、节奏）→ 最能代表真实力，但判断权重最低

**关键机制：**
- 环节1特别好 → 粗扫环节2，无明显问题直接签约
- 环节1较差 → 粗扫环节2，不亮眼直接放弃
- 环节1平平 → 认真看环节2；环节2好/差都能快速决定，不上不下才进环节3

### 2. "表面文章"的核心意义

> 作者只要能做好"表面文章"，作品离签约就不远了。

**组包材料最重要的表面层：**
- 书名（含笔名）
- 作品简介（是否吸引人、不云山雾罩）
- 排版整洁（有分段，有标题层次）
- 类型题材清晰

### 3. QQ主动投稿规范（四条基本建议）

**（1）先查资料，不莽撞**
- 知道编辑的名字、所在网站、负责组别（主编还是责编）
- 必做：点开编辑QQ资料查个人签名和空间，可能直接写明了投稿须知

**（2）保持礼貌，不啰嗦**
- 错误方式："你好，请问在吗？"（打断工作后还要多一来回）
- 也错误：二话不说直接传稿（不礼貌）
- **正确格式：** 在一条消息中，搭礼貌称呼 + 完整自我介绍 + 作品核心资料

**标准QQ投稿模板：**
> "XX编辑好，我是新人作者YY，希望能与贵站签约。我的作品为《书名》，说的是【一句话故事】。全书预计XXX万字，已完成X万字，同时还有一份详细的大纲与设定文档。请问是否方便现在接收文件？"

**（3）不卑不亢，不露怯**
- 不要在投稿中反复强调"我是新人"、"我写得不好"
- 这会让编辑下意识去寻找稿件缺陷，而非发现亮点
- 平和平等的心态 = 正确投稿态度

**（4）规范格式，亮卖点**

**投稿文档的标准结构：**
1. 作品/作者基本资料：书名 + 笔名 + 作者QQ + 类型 + 预计/已完成字数
2. 作品介绍：100字内简介 + 金手指设计 + 作品特色 + 主要卖点
3. 作品大纲及相关设定
4. 作品正文（全部，或前3万字+全稿另附）

**格式要点：**
- 优先使用 .doc 格式（而非 .docx，兼容性更好）
- 全部内容放一个文档；超过10万字正文可截前3万 + 另附全稿
- 文档标题 = 作品书名
- 每部分加粗标题，合并成一行的内容不要分两行
- 若文档较长，顶端加一行说明包含哪些内容

**扬长避短原则：**
- 投稿文档是主动展示机会，完全可以"报喜不报忧"
- 如有某部分写得不好，可以省略掉；如有特别精彩的部分，可以标红/加粗
- 如有其他加分信息（如有其他平台签约经历），主动添加

---

## 适用场景

- 组包投稿材料时，按三环节优先级确保"表面层"质量不低于正文质量
- 生成投稿文档时，严格按四段结构输出
- 评估投稿材料质量时，环节1（外观表面）作为首要判断维度
