# 场景单元拆分法

## 一、不要按“段落”拆，要按“压力变化”拆

一个有效场景单元，至少包含：

- 目标
- 阻力
- 小结果

只换了地点、没换压力，不一定值得拆成新单元。

## 二、常用 4 单元结构

### 单元 1：晚进入场

开在压力已经在场的位置，而不是暖机。

### 单元 2：试探 / 误判

角色以为自己知道问题，但其实还在错路上。

### 单元 3：中段回报

必须让读者“拿到东西”。

### 单元 4：代价 / 钩子

局部完成，但更大的债务立起来。

## 三、五单元扩展结构

当一章较长或任务较重时，可扩成：

1. 入场压力
2. 局部推进
3. 中段回报
4. 误判修正 / 代价升级
5. 章末钩子

## 四、拆分失败信号

- 每个单元只是“继续聊 / 继续走 / 继续想”
- 说不出第三单元的中段回报
- 最后一个单元只有感慨，没有动作 / 信息 / 危险
