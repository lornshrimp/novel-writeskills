## 控制卡新增字段（P0）

每章控制卡默认新增两项：

1. `动作-反应链`：至少两拍
   - 第一拍：动作（制造不平衡） -> 反应（质疑/压制/震惊）
   - 第二拍：动作（反击/推进） -> 反应（打脸/收益/局势变化）
2. `不平衡设计说明`
   - 本章通过哪种信息差、身份差或场景差打破平衡
   - 打破后如何回到新的阶段平衡并引出下一拍

## 章节施工硬要求（P0）

1. 任何章中小剧情都要有“动作 + 反应”；缺一视为空转。
2. 动作必须影响主角主线（名誉、地位、财富、关系、目标）之一。
3. 反应必须可见，不能只在主角内心里“自我感动”。
4. 本章中段回报要写成“读者可见变化”，禁止写成抽象推进。

## 控制卡禁行项（P1）

1. 禁止把“铺垫”当作无目标慢叙；铺垫必须抬高预期。
2. 禁止把“伏笔”当装饰；伏笔要绑定回收窗口。
3. 禁止连续多场“主角动作但无人反应”的独角戏。
4. 禁止只做情节罗列，不写每拍的情绪功能。

## 自检清单（控制卡交付前）

- [ ] 是否写清了两拍动作-反应链？
- [ ] 是否明确了本章不平衡来源？
- [ ] 中段回报是否可见、可量化、可复述？
- [ ] 本章是否至少回收或推进一条既有线索？
