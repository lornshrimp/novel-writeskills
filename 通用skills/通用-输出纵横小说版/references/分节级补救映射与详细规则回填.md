# 通用-输出纵横小说版｜分节级补救映射与详细规则回填

## 一、开头（前 300 字）补救

### 问题识别

| 问题类型 | 症状描述 |
|---|---|
| 氛围堆砌型 | 前 3 段全是环境描写，主角未出现，无事件切入点 |
| 背景热机型 | 用冗长背景说明代替直接事件入场 |
| 情绪化直给型 | 直接写"他很紧张""气氛很恐怖"，无细节支撑 |
| 视角越界型 | 开头即出现上帝视角，补充主角不知道的信息 |

### 补救操作

1. **氛围堆砌型**：保留 1–2 个最有现实锚点的感官细节，其余环境描写压缩或后移至主角进入场景时逐步展开。
2. **背景热机型**：把背景信息切碎，用主角的行动、对话或内心闪回自然带出，禁止整段说明。
3. **情绪化直给型**：把情绪词替换为对应的具象动作或环境细节（参见执行细则第四节心理刻画规则）。
4. **视角越界型**：删除超出主角感知边界的信息，加上主角感知锚点短语。

## 二、中段（30%–70%）补救

### 问题识别

| 问题类型 | 症状描述 |
|---|---|
| 节奏失衡-拖沓型 | 连续 5 段以上无新线索、无新压力、无人物互动推进 |
| 节奏失衡-仓促型 | 关键线索未经铺垫直接揭晓，主角"脑子一转就懂了" |
| 线索释放过快 | 前半段就解开主要谜团，后半段无悬念可维系 |
| 感官描写缺失 | 关键场景无视觉/听觉/嗅觉/触觉中任何一项感官细节 |
| 人性落点缺失 | 中段全程推进剧情，无一处人物情感温度或现实映射 |

### 补救操作

1. **拖沓型**：识别水段，压缩或删除与悬念推进无关的段落；在被删段落的位置插入一个新线索抓点或关系张力点（零新增事实原则：从原章其他部分调取信息重组，不凭空创作）。
2. **仓促型**：在揭晓关键线索前，补入主角的感知过程（看到 → 联想 → 推断 → 结论），让读者跟上推理链。
3. **线索释放过快**：将已解开的谜团重新拆分，把一部分答案后移至章末附近，章末只给半个答案留下追读钩子。
4. **感官描写缺失**：在关键场景（悬念发现点、对峙点、危机升级点）各补入 1–2 种感官细节，优先使用嗅觉 + 听觉组合（最能强化悬疑感）。
5. **人性落点缺失**：在中段最适合的人物交互或独处时刻，补入一个侧面动作细节（非直接情绪词）反映人物内心温度或人性矛盾。

## 三、章末（后 15%）补救

### 问题识别

| 问题类型 | 症状描述 |
|---|---|
| 完全闭环型 | 章末把所有谜团解开，无悬念留到下章 |
| 钩子虚弱型 | 章末有悬念，但力度不足（"总感觉有哪里不对"等模糊表述） |
| 情绪化结尾型 | 章末用情绪化语言渲染，而非用事件/细节/悬念支撑钩子 |
| 反派视角型 | 章末切换到反派视角透露过多关键信息，削弱主线悬念 |

### 补救操作

1. **完全闭环型**：把最后 1–2 个谜团解答退后半章，章末只到"主角即将发现"或"关键证据出现但尚未读懂"的节点停止。
2. **钩子虚弱型**：把模糊表述替换为具体异常细节或具体行动停滞点（"他刚想开口，电话突然断了""脚印在这里消失了——但门是锁着的"）。
3. **情绪化结尾型**：删除情绪渲染段落，用环境异常细节或主角感知的具体反常来承载悬念感。
4. **反派视角型**：压缩反派视角篇幅，仅保留一个侧面行动细节，不披露计划、动机或关键信息。

## 四、常见全篇问题快速映射

| 症状 | 根因 | 处置优先级 |
|---|---|---|
| 整章无感官描写 | 跳过了氛围营造规则 | 高 |
| 主角"神推理"频繁出现 | 未执行"推理链透明化"规则 | 高 |
| 节奏全程绷紧，没有缓冲段 | 未执行"快慢交替"规则 | 中 |
| 章末已完全解谜 | 未执行"悬而未决钩子"规则 | 高 |
| 人性落点为零 | 仅关注情节推进，忽略纵横读者核心需求 | 中 |
| 视角频繁切换 | 未执行"限制性视角"规则 | 高 |
| 术语堆砌，不融入剧情 | 未执行"场景化使用"规则 | 中 |
| 超自然元素比例过重 | 跑偏玄幻赛道 | 高 |
| 正文字数超 5000 CJK | 未控制中等节奏字数上限 | 中 |
| 标题是纯情绪词 / 抽象词 | 未执行纵横标题规则 | 低 |

## 五、标题重写规则

纵横标题倾向：

- **优先**：行动 + 异常物件组合（"锁着的门里传出脚步声"）、代价型标题（"遗书上多了一个名字"）、悬念型短句（"他认出了那双鞋"）。
- **禁止**：纯情绪词（"恐惧""黑暗来临"）、标题党夸张（"震惊！真相竟然是…"）、纯符号或 emoji。
- **字数建议**：10–20 字，简洁清晰，能从标题推断出场景基调。

---

## 六、`## 作者有话说` 规则

- 以轻互动和追读牵引为主，不讲平台、不讲写作过程、不复盘剧情、不补设定。
- 适合：小剧场式温暖一句、与读者互动的推理提问、下一章气氛暗示（不剧透）。
- 禁止：主题升华式总结、章节内容复盘、设定科普。
- 字数目标：220–280 CJK（硬下限 200 CJK）。
