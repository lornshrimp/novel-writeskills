# 通用-执行对话冲突

好的对话冲突不是说得热闹，而是每一句都在争夺位置、信息或主动权。

## 常见触发词 / 用户说法速查

- 帮我把这段对话写得更有冲突
- 这段对白太像轮流发言
- 做一场盘问 / 谈判 / 争执戏
- 这段信息出来了但人物没立住
- 帮我把潜台词拉出来

## 何时使用

- 设计盘问、问责、谈判、争执、试探、威胁、翻脸等高压对话。
- 修复“对白很多，但冲突不成立”的场景。
- 需要通过对话完成信息释放、关系重排或风险升级。

## 默认执行顺序

1. 先写明双方显性诉求与隐性目的。
2. 再写双方不能退让的边界与可交易筹码。
3. 再安排试探、误导、卡口径、说 A 指 B、知而不言、假让步、反击或失控节点。
4. 最后确认对话结束后局面真的发生了变化。

## 对话基础规则（新增）

**对话占比与密度**：
- 单章对话占比应在 **25%-55%** 范围内（过少显得叙述冗长，过多显得空洞）
- 长篇累积对话超过 5 轮应插入**动作或环境穿插**，防止纯对话无立足点
- 高压对话中，话多的一方应该是想掌控全局的人物；话少但犀利的一方应该是握有主动权的人物

**对话意图与冲突**：
- **禁止信息问答式对话**（A 问、B 答、纯信息传递）；即使必须传递信息，也应该夹带立场对立或态度不同
- **每段对话的双方必须有不同的诉求或看法**，否则视为伪对话（不成立的对话）
- 对话里应该看见：谁在回避、谁在试探、谁在让步、谁在底线

**对话的三段式结构**（实战推荐）：
```
[动作或环境引导] + [对话内容] + [身体反应或后续动作]
```

示例：
```
✅ "真要挑这个走？"阿彪蹲在门槛上，叼着烟，"你不怕别人看见说你神经病？"
   陈勉拍了拍筐沿："看见了又怎样，我又没犯法。"
   "你是不怕，但你老婆呢？"
   陈勉的手停了一下。（身体反应）

❌ "你真的要去吗？"
   "是的。"
   "那你要小心。"
   "我知道。"
```

三段式的好处：
- 让对话有场景感和身体感，不显得浮空
- 对话中的停顿、反应、动作都在暗示人物的真实想法和当下状态
- 读者可以同时从语言和肢体语言读出冲突的深度

## 对话落地补丁（新增）

- 对话不只承担信息释放，还应尽量承担：关系推进、权力位移、误解加深、信任修复、情绪升温或情绪决裂中的至少一项。
- 重要对话里默认要避免整段口气齐平；句长、停顿、躲闪、反问、答非所问都应服务人物声口差。
- 若一段对话只能靠人物把心里话完整说出来才能成立，通常说明潜台词、位差或冲突筹码还没写出来。
- 读者最好能从对话里同时听见“表面在聊什么”和“真正争的是什么”。

## 最低交付

- 双方都想从对话中拿到东西。
- 至少有 1 层潜台词与 1 次立场升级。
- 对话结束后，信息、关系或风险至少变化一项。

## 默认输出口径

- 默认输出一段或一场可直接落稿的高压对话冲突方案。
- 默认保留原有信息目标，但拉开位差、潜台词与冲突升级。

## 硬规则

- 禁止角色轮流当喇叭解释设定。
- 禁止所有人一个口气，听不出身份与位置差异。
- 禁止冲突只停留在口头态度，没有后果。
- 禁止靠直说把所有底牌一次讲完。
- 高压对话必须至少承担两项功能：试探 / 误导 / 交换筹码 / 改写关系 / 升级风险。
- 高压对话最好至少争到一项：口径、权限、时间窗、位置差或证据解释权。
