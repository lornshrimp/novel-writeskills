# 对话真实感与声口分化校准卡

适用于对话已经有冲突轮廓，但仍显得书面、同腔、假交流的场景。

## 一个底线

真实对话不是现实录音逐字稿，也不是语法课示范句。

对话要做到的是：

- 听起来像这一类人在这一刻真的会这样说
- 同时比现实废话更少、冲突更集中、信息更有效

## 先校准四个维度

### 1. 年龄与代际

- 年轻角色更可能跳词、省略、抢话、临时改口
- 年长角色未必慢，但更可能带经验判断、习惯性定义和省力表达

### 2. 文化与地理

- 不同成长环境的角色，用词密度、脏话方式、幽默点、绕弯方式不同
- 不要只靠方言词装口音，重点是句法习惯与反应方式

### 3. 关系与位置

- 上下级、恋人、陌生人、熟人，说话的冒犯额度完全不同
- 有权的一方更容易切断话题、定义局面
- 没权的一方更容易试探、回避、拖延或换问题回答

### 4. 场景压力

- 高压时，句子通常更短、更偏目标导向
- 压力越高，越少有人完整交代理由
- 情绪失控者不一定说更多，可能说得更碎、更硬、更绕

## 不像活人的常见警报

- 双方都用完整句、完整逻辑、完整礼貌地轮流发言
- 明明彼此熟悉，却还互相解释早就知道的信息
- 明明局势危险，却还有闲心做完整背景说明
- 不同角色只是换名字复述同一层信息
- 一段对话听不出谁更强、谁更怕、谁在抢位置

## 声口分化的四个抓手

### 1. 省略度不同

- 强势方可以少说，因为默认别人会补全
- 弱势方更可能解释、修补、求对方理解
- 熟人之间常省主语和背景，陌生人通常不会

### 2. 攻防方式不同

- 有人直接逼问
- 有人装糊涂
- 有人转开话题
- 有人先认一半再偷换另一半

把同一冲突写出不同攻防手感，人物才会分开。

### 3. 职业惯性不同

- 警察先问证据与时间点
- 律师先卡定义和责任
- 销售先稳情绪和可谈空间
- 医护先问状态和风险
- 物业先问流程和责任边界

### 4. 脸面管理方式不同

- 有人靠沉默保面子
- 有人靠冷笑
- 有人靠抢话
- 有人越慌越爱说完整句掩饰自己

## 把对话从“写出来”改成“听出来”

改写时优先做三件事：

1. 删掉人物为了作者交代背景而说的句子。
2. 给每个人补一句：他真正想拿到什么，最怕漏出什么。
3. 让一句台词同时承担信息与位置，而不是只传资料。

## 快速修正动作

- 把一句太完整的话拆成：半句 + 动作 + 补半句
- 把解释改成反问、打断、顺嘴带出
- 把“我很生气”改成口气变化或句式突然变硬
- 把统一的礼貌表达拉开成不同人物的防守方式

## 声口校准工件（新增）

```markdown
【人物 A】
- 位置 / 权力感：
- 护点：
- 怕点：
- 抢点：
- 常用句长：短 / 中 / 长
- 常用防守方式：打断 / 装糊涂 / 反问 / 硬顶 / 顺水推舟

【人物 B】
- 位置 / 权力感：
- 护点：
- 怕点：
- 抢点：
- 常用句长：短 / 中 / 长
- 常用防守方式：打断 / 装糊涂 / 反问 / 硬顶 / 顺水推舟

【这场对话里最该拉开的点】
- 词汇偏好：
- 句式习惯：
- 情绪失控时的变化：
- 动作锚点：
```

## 对话成活标准

- 遮住人名后，大致还能听出是谁在说
- 角色的社会位置和关系温度能从句法里听出来
- 对话结束后，局面至少变化一项：信息 / 关系 / 风险 / 主动权
- 读者更像在偷听，而不是在听作者安排讲解

## Green / Yellow / Red 声口门禁（新增）

- `Green`：遮住人名后仍能大致分人，且位置、关系温度、情绪滤镜都能从句法与动作里听出来。
- `Yellow`：人物有轻微差异，但换给别人说也不算太违和；通常需要再补动作锚点或防守方式。
- `Red`：删掉提示语后几乎全员同腔，或一遇高压就统一变成完整句、书面句、作者句。
