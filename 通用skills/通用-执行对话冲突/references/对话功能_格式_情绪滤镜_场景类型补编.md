# 对话功能_格式_情绪滤镜_场景类型补编

> 吸收外部 `dialogue-writing` 的通用工法，用于把对话从“轮流发声”补强成“带功能、带身份、带节奏、带潜台词的行动单元”。

## 一、对话首先必须“做事”

每段对话至少承担一项：

- 推进情节
- 揭示人物
- 制造冲突
- 改写关系
- 营造氛围

若删掉一段对话，对故事几乎没有影响，这段对话大概率不该存在。

## 二、简洁性原则

真人说话通常：

- 更短
- 更碎
- 更不完整
- 会被打断
- 会迟疑
- 会转移话题
- 会暗示而不是讲解

所以默认优先：

- 删啰嗦书面语
- 删无功能客套
- 删角色互相重复已知信息

## 三、对话格式底线

- 每个说话人起新段
- 标签能省则省
- 默认优先用中性标签或动作标签
- 不为避免重复“说”而滥用花哨替代词
- 情绪优先落在动作、停顿、声音状态，而不是副词修饰

### 推荐习惯

- `“……”他说。`
- `“……”她问。`
- `“……”他停了一下。`
- `她把杯子推远。"……"`

## 四、语言指纹

每个角色默认至少应有以下几项差异：

- 词汇偏好：粗粝 / 文雅 / 术语化 / 口语化
- 句式习惯：短促 / 迂回 / 反问 / 祈使
- 信息表达方式：直说 / 暗示 / 反话 / 沉默
- 口头小习惯：少量、自然，不当标签贴纸

目标：遮住名字，也能大致猜出谁在说话。

## 五、身份映射

职业、阶层、时代、教育、文化背景会改变说话方式：

- 医生偏症状 / 预后 / 判断
- 军人偏简短 / 指令式
- 商战角色偏试探 / 利益交换
- 底层人物可能更口语、更省略
- 贵族 / 上位者更可能迂回、压迫、设定坐姿与口气

不需要每句都强调身份，但关键对话必须透出身份感。

## 六、情绪滤镜

同一角色在不同情绪下，说话应发生可辨变化：

- 愤怒：短句、打断、命令句增多
- 恐惧：不完整、重复、省略、结巴
- 傲慢：反问多、回避正答
- 悲伤：停顿多、答非所问、节奏变慢
- 伪装：过于礼貌、措辞精确、自然感降低

## 七、节奏工具

### 呼吸控制

可用以下方式制造停顿与压力：

- 动作插入
- 沉默留白
- 环境打断
- 话题岔开再拉回
- 反问替代回答

### 冲突升级阶梯

默认优先按：

`试探 → 暗示 → 直面 → 对抗 → 爆发`

不要一上来就直接开到最大音量，除非场景本身就是崩盘瞬间。

### 认知错位与信息差

- A 知道，B 不知道
- 读者知道，角色不知道
- 双方都在试探对方到底知道多少

这类信息差本身就是张力源。

## 八、潜台词工具

### 常见方法

- 表里不一
- 回避焦点
- 反问替代回答
- 说别的事来回应真正的问题
- 用沉默或动作代替语言
- 用道具动作借代情绪

### 道具借代示例

- 摔杯子 = 愤怒
- 转戒指 = 犹豫
- 整理衣领 = 紧张
- 把茶杯推远 = 拒绝

## 九、动作与对话结合

动作与对话可以：

- 一致：强化情绪
- 矛盾：揭示真相

比起“他说他支持”，更有力量的常常是：

- 嘴上支持，手却握紧
- 嘴上平静，身体却后退

## 十、常见场景类型

### 争吵

- 短句
- 打断
- 重复关键词
- 逐步升级
- 真正价值在于暴露真实诉求，不是比谁嗓门大

### 告白

- 迟疑
- 停顿
- 寻找措辞
- 脆弱感比华丽措辞重要

### 审讯

- 节奏由提问者主导
- 回答越来越短
- 压力来自已知信息逐步释放

### 谈判

- 条件交换
- 试探底线
- 选择性说话
- 真交易往往在言外完成

### 调情

- 双关
- 试探
- 暗示
- 身体距离变化参与潜台词

## 十一、风格适配

### 时代语感

- 古风：可文白夹杂，但别全篇翻古文壳
- 现代：口语化，允许不完整句
- 未来 / 科幻：术语要自然，不大段解释

### 体裁差异

- 言情：潜台词、试探、空间距离
- 悬疑：信息博弈、回避、问答节奏
- 喜剧：错位、节奏、反差
- 军事 / 商战：短句、判断、决策导向

## 十二、外部经验吸收的两条补丁

- 情感向章节里，角色观察优先触发情感反应，不优先触发“侦探式理性分析”
- 同一能力或同一种观察力不要连续重复展示；换切口、换表现层、换情绪结果

## 红线

- 不得让角色用对话直接讲设定讲义
- 不得让所有人一个声带
- 不得把长篇信息播报误当高质量对话
- 不得过度礼貌到不像真人
- 不得用花哨标签替代真正的动作和情绪表现
