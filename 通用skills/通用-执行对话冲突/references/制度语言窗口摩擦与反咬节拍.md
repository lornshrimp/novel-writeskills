# 制度语言、窗口摩擦与反咬节拍

## 适用场景

- 盘问 / 问责 / 质询
- 物业 / 医院 / 学校 / 公司窗口沟通
- 熟人对峙
- 主角与反派的试探性交锋

## 三种高价值话术

1. **制度语言**：用规则、流程、权限把人卡住。
2. **窗口摩擦**：用拖延、转述、踢皮球、模糊口径制造压迫。
3. **反咬节拍**：让被盘问的人突然把问题折返给主角。

## 节拍设计

- 先试探
- 再卡口径
- 再给一点假让步
- 最后反咬或翻面

## 三个常用施压动作

1. **说 A 指 B**：表面在解释流程，实际在逼对方认责、表态或自己露馅。
2. **知而不言**：故意缺半句、留半拍，让对方为了补缝隙自己多说。
3. **情绪错频**：一方越冷静、越公事公办，另一方越显得失控、心虚或位置更低。

## 自检

- 这段对话里，谁掌控话语节奏？
- 有没有至少一次口径变化或位置变化？
- 对话结束后，局面是不是更难了？
