# 对话冲突失败信号

## 常见失败信号

- 对话很长，但没有任何地位变化。
- 大家只是把已知信息再说一遍。
- 每个人都像作者的同声传译器。
- 冲突点只在情绪词，不在利益、代价与立场。
- 人物开口太规范，像在写申请书，不像在活着。
- 一个人连续说太久，其他人只负责当听众。
- 对话里硬塞背景资料，像把说明书撕碎后喂给人物。
- 场景没有保留，所有底牌都亮完，后文没得追。
- 该快的场景被说明拖慢；该慢的场景又只剩吵架加速。

## 来源层定位（新增）

| 失败信号 | 更可能出问题的层 | 第一补法 |
| --- | --- | --- |
| 只有态度，没有利害 | `诉求与筹码层` | 先补双方想拿什么、怕失去什么 |
| 有争执感，但局势没改写 | `博弈推进层` | 补一次翻面、让步失败或主动权变化 |
| 谁都像一个作者在轮流发言 | `声口与动作层` | 拉开句式、回避方式、动作锚点 |
| 解释很多，偷听欲很低 | `诉求与筹码层` + `博弈推进层` | 把背景硬塞改成试探、遮掩、顺嘴带出 |
| 情绪词很多，但没有压力 | `博弈推进层` | 把情绪落回风险、代价、位置争夺 |

## 修正优先级

1. 先补双方的显性目标与隐性目标。
2. 再补不能退让的边界。
3. 再让一句话带来新的信息、误会或伤害。
4. 最后检查对话结束后谁失去、谁占优、谁暴露。

## 快速排错表

- **听起来像作者**：删说教，删背景硬塞，改回人物自己的词。
- **听起来像公文**：增加缩略、打断、回避、犹豫、动作承接。
- **听起来太满**：删掉一部分直说，留下未说完的部分。
- **听起来太平**：补风险、补欲望、补一处关系或局势变化。
- **听起来太乱**：检查上下句是否真在互相回应，而不是各说各话。

## Green / Yellow / Red 快判（新增）

- `Green`：至少能清楚回答“双方想拿什么、谁在失去什么、这轮对话后谁的位置变了”。
- `Yellow`：冲突存在，但某一层仍像占位符；例如知道在吵，却说不清他们到底在争什么。
- `Red`：诉求说不清、局面没变化、人物分不开，或删掉整段后剧情几乎不受影响。

若 `诉求与筹码层` 与 `博弈推进层` 同时失效，默认直接按 `Red` 处理，不要试图只靠修句子蒙混过关。
