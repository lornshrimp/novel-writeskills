# 对话冲突结构与潜台词规则

## 四步基础结构

1. 试探：先抛软问题、假让步或伪信息。
2. 碰撞：双方底层诉求开始正面冲突。
3. 翻面：某一方暴露真实意图、误判或弱点。
4. 余震：对话结束，但局面被改写。

## 潜台词检查

每段关键对白至少回答：

- 他说出口的是什么？
- 他真正想逼对方承认什么？
- 他最怕被对方发现什么？

## 值得偷听原则

判断一段对话该不该保留，可以先问：

- 旁边的陌生人会不会想偷听？

若不会，通常有几种原因：

- 只是交换背景资料
- 只是重复读者已经知道的事
- 只有态度，没有利害
- 有信息，但没有变化

值得偷听的对话通常至少具备以下之一：

- 话里有风险
- 话里有秘密
- 话里有位置争夺
- 话里有人快要失去什么

## 对话的四类功能

一段有效对话最好至少承担两项功能：

1. 塑造人物
2. 释放信息
3. 推动情节
4. 制造张力 / 悬念

若只承担“解释设定”这一项，通常不够格。

## 对话推动故事的最低门槛

每个有效对话场景至少应完成以下之一，最好同时完成两项以上：

- 带来新信息
- 呈现新障碍
- 让人物的目标更清楚
- 让风险更大
- 让关系位置发生变化
- 让主题被人物以自身立场碰出来

如果一段对话删掉后，故事的局势几乎没变化，这段对话大概率还不够格。

## 张力的三种常见形态

- **心理冲突**：彼此怀疑、试探、压迫、忍着不说破
- **言语冲突**：反问、刺探、顶撞、逼问、挖坑
- **肢体冲突前夜**：动作克制、距离变化、压迫感上升

并非每段对话都要吵，但至少要让局面一直在绷紧。

## 动机显影

对话不只是在传递态度，也是在暴露人物真正想要什么。

优先检查：

- 人物嘴上在谈什么？
- 实际在护什么、抢什么、躲什么？
- 这一轮话说完后，谁更想继续，谁更想撤退？

若对话只听得到观点，听不到欲望和代价，人物就容易发空。

## 人物声音差异不是口头禅堆砌

人物声音差异，优先来自以下几项，而不是先给每个人硬塞一句口头禅：

- **位置差**：谁有权决定、谁在求、谁在挡、谁在审。
- **熟悉度**：陌生人、熟人、上下级、亲密关系，说话的省略度和冒犯度都不同。
- **职业惯性**：医生、警察、老师、销售、律师、物业、家长，会优先使用不同的词。
- **受教育与生活环境**：句式长短、抽象度、措辞精细度会不同。
- **情绪处理方式**：有人硬顶，有人阴着绕，有人先沉默再补刀。
- **眼下目标**：想压住场子的人，和想尽快脱身的人，不会说成一个口气。

实用检查：

- 临时遮掉人名和提示语，还能不能大致分出是谁在说？
- 同一句信息，强势方、弱势方、老手、新手会不会说成完全一样？
- 这句台词如果换给另一角色说，违和感够不够强？

如果答案都是否，说明人物声音还没立住。

## 刹车与油门

对话天然比纯叙述更快，但快慢仍可控制：

- **踩油门**：短句、打断、追问、少解释、少背景、少动作停顿。
- **踩刹车**：插入短动作、观察、犹豫、试探、局部说明，让读者消化压力。

提醒：

- 高压场景要防止“全是快句，结果只剩吵”。
- 慢节奏场景也不能慢成闲聊茶话会，仍要保留潜在张力。

## 用对话补少量场景 / 背景信息

对话可以顺手带出地点、职业、关系和处境，但只能少量、顺手、带冲突地带：

- 不要让人物为了作者方便而解释他们彼此早就知道的事。
- 优先把背景信息包在试探、争执、遮掩、解释失败或顺嘴带出里。
- 若信息量太大，改用叙述或动作承担，不要全塞进嘴里。

一条实用判断：

- 这句背景信息如果删掉，人物还会这样说话吗？
- 若不会，说明它更像作者塞给人物的任务。

## 有所保留比全说更有效

人物不该把自己知道的一切一次性交代完。

更有效的做法：

- 先试探
- 再松一点口
- 留下没说完的部分
- 让对方或读者意识到“还有东西没被说出来”

保留能同时增强：

- 真实感
- 张力
- 读者继续往下看的欲望

## 语体差异提醒

- 强势角色更常用切断、定义、命令。
- 弱势但聪明的角色更常用反问、拖延、转移或借题施压。
- 专业角色说话应体现职业语汇，但不能堆成说明书。

## 对话中的人物塑造

对话不是给人物配字幕，而是让人物自己露馅。

优先让读者从对话里听见以下内容：

- 他习惯怎么争夺主动权
- 他最先护的是面子、利益、秘密还是关系
- 他一急会变得更硬，还是更碎、更乱、更爱解释
- 他有没有职业本能，是否会先问程序、证据、价格、后果
- 他身上有没有旧伤口，一碰就改口、沉默、炸开或过度防御

一段能塑人的对话，最好至少露出以下之一：

- **立场**：他站哪边，护什么。
- **短板**：他哪里会乱，会怕，会露破绽。
- **习惯**：他惯用的攻击、防守、回避方式。
- **代价观**：他认为什么值得，什么不能失去。

常用补强动作：

- 把“角色直接说结论”改成“先护一下，再说”。
- 把“情绪说明”改成“口气变化 + 微动作 + 半句停住”。
- 让人物在一句话里同时暴露身份和欲望，而不是只传信息。
- 让人物对同一信息作出不同反应，而不是统一口径复述。

## 对话与读者联结

读者愿意继续跟一段对话走，往往不是因为台词漂亮，而是因为他开始站队、担心、好奇，或者觉得人物“像活人”。

对话里最常用的四个联结入口：

- **认同点**：他说出了读者熟悉的窘境、判断、职业压力或生活惯性。
- **同情点**：他明明在扛，却已经快失去什么。
- **喜爱点**：他有分寸、担当、幽默、专业、克制，或在坏处境里仍替别人留余地。
- **好奇点**：他明显知道更多，却没全说；或他这样说，背后一定另有原因。

如果人物天然不讨喜，不要急着洗白，优先补以下支撑：

- 他至少在乎一件真的东西
- 他至少有一个清楚代价正在逼近
- 他至少在某个点上足够专业、有效或敢扛
- 他至少留下一点没说透的深层动机

提醒：

- 读者联结不等于让人物讨好读者。
- 更有效的是让读者知道：这人有毛病，但这场里他真的会疼、会怕、会输、会咬牙撑。

## 常见失败写法

- 反复直呼对方姓名，像在演肥皂剧
- 用副词和夸张提示语替代真实情绪
- 借人物之口替作者说教
- 让人物只说，不行动、不反应、不思考
- 对话结束后，局面毫无变化
- 让人物重复彼此早就知道的信息，只为了交代背景
- 让人物把所有底牌一次说完，没有任何保留
- 让对话过度规范、像公文，不像活人
- 让人物长篇演讲，却没有被打断、反驳或转换局面

## 修正动作

当对话发假、发平、发直时，优先：

1. 删掉不必要的称呼
2. 把“他愤怒地说”改成动作 / 停顿 / 口气变化
3. 给每个人物补一句“他真正想拿到什么”
4. 在结尾处加一个局面变化，而不是温吞散场
5. 把作者替人物说的话撤掉，让人物只说他们真会说的话
6. 检查是否有一部分信息应改由叙述承担，而不是对白硬扛
