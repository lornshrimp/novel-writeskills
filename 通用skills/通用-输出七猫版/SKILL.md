# 通用-输出七猫版

这是平台共性本体，负责承接“七猫版输出”的跨题材共性规则。

## 常见触发词 / 用户说法速查

- 输出七猫版
- 改成七猫能发的版本
- 做七猫平台改写
- 帮我把这章改得更免费阅读平台一点

## 何时使用

- 把任意题材章节改写成更适合七猫平台读者的版本。
- 需要强化直给反馈、强情节推进、免费阅读追读力与高密度抓点。
- 需要为 `题材名-输出七猫版` 提供统一的平台骨架。

## 不适用情形

- 只做题材边界补充，不做平台输出本体。
- 只做多平台编排、落盘和终检，而不处理七猫平台风格本身。

## 本层职责

- 维护七猫平台的跨题材共性规则。
- 统一承接七猫版输出的结构约束、标题倾向、节奏压力与风格门禁。
- 为后续新增题材保留直接复用入口。

## 继续读取的 references

- `references/跨题材共性与迁移来源.md`
- `references/平台共性执行细则.md`
- `references/分节级补救映射与详细规则回填.md`

## 默认执行顺序

1. 先读取对应题材的 `题材名-输出七猫版` Skill。
2. 先锁定零新增事实、体裁结构、工作目录、章节层级、字数门禁与标题同步规则。
3. 再执行七猫平台的直给反馈、强推进与追读压力强化。
4. 最后才处理降相似度，不得为了降重破坏平台抓力。

## 默认输出口径

- 默认输出一版可直接继续落盘或进入平台终检的七猫派生正文。
- 默认保留原章核心事件链、抓点节拍与章末驱动，不新增关键事实。

## 最低交付

- 有明确的七猫平台直给反馈与强推进感。
- 保住原章核心事件链、读屏顺滑度与追读驱动。
- 读起来不像泛免费平台稿，也不像硬压缩摘要稿。

## 硬规则

- 绝对禁止输出“抽象自述 + 动词模板 + 口径名词”的垃圾句群；凡出现类似“对照项落在… / 先把同一句话拆碎 / 先把顺滑的解释拆开 / 只求能追溯（复核、对得上）”等模板化短语，必须判定为污染并整稿回炉。
- 严禁使用或间接调用以下脚本对正文做生成、扩写、拼接或降重：`scripts/append_cn_unique_monologue.ps1`、`scripts/append_cn_unique_narration.ps1`、`scripts/append_cn_unique_thirdperson.ps1`、`scripts/rephrase_cn_body.ps1`、`scripts/rephrase_en_body.ps1`、`scripts/cn_lexicon_profile_transform.ps1`。这些工具不得进入平台正文生产链路。
- 平台稿新增“模板句污染清零门禁”：正文不得出现连续 2 句及以上“我/我这边/我心里… + 把/将… + 对齐/校准/锁死/拆开… + 连接词 + 口径名词 + 只求/不求/先把…”结构；命中即失败，不论字数与相似度是否通过。

- 命中本技能时，必须同时加载对应题材的 `题材名-输出七猫版` Skill。
- 题材特有规则不得回写到本文件中平行维护。
- 平台派生正文默认落在本 Skill 的工作目录 `七猫/` 下。
- 章节级派生正文目录统一按小说结构决定：若作品有分部，则位于 `七猫/第X部/第Y卷/`；若作品无分部，则位于 `七猫/第X卷/`。
- 平台派生正文文件名默认不带日期，沿用既有章节号与平台标题同步规则；日期只用于配套审阅报告、书评等派生产物。
- 章节派生正文必须以 `.md` 文件形式落盘：若对应章节目录不存在，执行前必须先创建完整目录路径，再将改写后的章节内容写入该目录下的 `.md` 文件，不得只输出聊天稿而不写入文件。
- 落盘路径的完整推断优先级：① 用户显式指定 chapterPath → ② 从源文件名与分部/卷信息自动推断 → ③ 若无法确定分部/卷，暂停并向用户确认，不得乱推断后静默落盘。
- 平台稿落盘后必须显式运行 `scripts/count-chapter.ps1 -MinCJK 3500 -MaxCJK 6500` 核正文，并运行 `scripts/count-afterword.ps1 -MinCJK 200 -MaxCJK 300` 核 `## 作者有话说`。
- 生成阶段必须先达到字数保底目标再允许落盘：正文目标 `4200–5800` CJK（硬下限 `3500`），`## 作者有话说` 目标 `220–280` CJK（硬下限 `200`）；未达保底目标时必须先回炉扩写，禁止“先落盘再补字数”。
- 在无损迁移完成前，不得删除源 `输出七猫版.prompt.md` 中尚未完成映射的内容。

## 与其他 Skill / Prompt 的边界

- 本 Skill 只负责七猫平台的跨题材共性骨架。
- 题材边界、题材禁行项与题材特有口径，继续由对应 `题材名-输出七猫版` 承接。
