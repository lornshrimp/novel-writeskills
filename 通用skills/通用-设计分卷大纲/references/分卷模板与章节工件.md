# 分卷模板与章节工件

```markdown
# 第X部第Y卷：《卷标题》- 分卷大纲

## 本卷承接清单（强制）

## 本卷目标字数与目标章数来源（强制）

## 卷概览

## 本卷5/15节奏布点图

## 章清单（Chapter Beat Sheet）

## 卷级启动表（前五章｜强制）

## 爽点布点表（卷内｜强制）

## 本卷核心人物推进表

## 伏笔管控台（卷级映射）

## 本卷上游任务落地核对表（强制）

## 具体章节规划
### 第X章：[标题]
- 故事梗概：
- 场景设置（WHEN / WHERE / WHO）：
- 核心冲突（WHAT / WHY）：
- 镜头分解（HOW-1）：
- 关键对话片段（HOW-2）：
- 动作节点（HOW-3）：
- 证据清单：
- 人物状态变化：
- 悬念钩子：
- 反向引用：
```

## 强制口径

- 章清单是唯一蓝图
- 分卷章数默认按每章约 3000 字换算
- 每章必须明确承接哪项上游任务
