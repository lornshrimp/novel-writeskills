```markdown

## 卷概览

## 故事面承接与卷章落点表（强制）

## 本卷5/15节奏布点图

## 本卷竞对对照与超越动作（强制）

- 本卷主对标竞对（1–3 本）：

  - [竞对作品] → [可借鉴强点] → [映射位置：5/15节拍 / 章清单 / 卷级启动表 / 钩子策略]
- 落后点（每本至少 1 条）：
  - [我方维度] 当前弱于 [竞对作品] 的 [具体表现]

  - P0：

## 章清单（Chapter Beat Sheet）

## 本卷上游任务落地核对表（强制）

## 具体章节规划

- 故事梗概：

- 核心冲突（WHAT / WHY）：

```

## 强制口径

- 章清单是唯一蓝图
- 分卷章数默认按每章约 3000 字换算
- 每章必须明确承接哪项上游任务
- 每卷必须明确承接哪些故事面，以及这些故事面分别落到哪些章群
- 每章至少要能说明：自己是显性落点章、保温章，还是交接章
