# 分部与分卷设计规则

## 一、上一级内容全量落地

- 分部 / 分卷设计首先要确认：上一级已经分配给本层的内容，是否被切实、全量、保质落地。
- 落地至少满足：
  - 全量承接
  - 结构展开
  - 可写、可拆、可验收
  - 可反向追溯

## 二、分层承接口径

- 分部大纲：承接总纲层文件组分配给本部的全部内容
- 无分部小说的分卷：直接承接总纲层分配给本卷的全部内容
- 有分部小说的分卷：承接本部分配给本卷的全部内容

## 三、目标字数与章数来源

- 本层目标字数必须以上一级分配为准
- 本层目标章数必须由上一级分配字数换算得出
- 默认换算口径：每章约 3000 字

## 四、三层拆解定义

- 情节块：战役，跨卷承接战略故事面
- 故事线：战斗，构成情节块的具体情节链条
- 故事点：场景，最小可写单位，通常对应一章

## 五、最终验收口径

- 是否写出了本层承接清单
- 是否能回指上一级分配来源
- 是否已落到情节块 / 故事线 / 章清单 / 工件
- 是否不存在遗漏、缩水与悬空
