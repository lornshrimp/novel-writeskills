# 卷级四段推进与关键点布置表

适用于把单卷从“很多章、很多事件”校准成“有卷内起承转合、又能继续给总纲供血”的可拆卷结构。

## 一、先明确：单卷不是缩小版总纲，但也不能没有结构骨架

一个卷纲不必把全书九部件完整重演一遍，但至少要具备卷级四段推进：

1. 卷内布局
2. 卷内反应
3. 卷内进攻
4. 卷内解决 / 卷尾缺口

## 一点五、卷级来源层速判（新增）

- `卷段职责层`：布局、反应、进攻、解决 / 卷尾缺口四段是否各司其职。
- `关键点布置层`：本卷扰动点、第一情节点、中间点、卷尾前最后关键信息是否明确。
- `卷尾交接层`：本卷如何阶段闭环、同时把主线和下一卷缺口交出去。

如果单卷只有“持续调查 / 持续升级 / 持续推进”，读起来往往会散。

## 二、卷级四段的默认职责

### 1. 卷内布局

任务：

- 建立本卷主问题
- 让本卷新资源 / 新人物 / 新局面入场
- 明确这一卷与上卷相比新增了什么压力

这一段最怕：

- 重复回顾上卷
- 迟迟不给本卷独立抓手

### 2. 卷内反应

任务：

- 主角对本卷新局面先试、先躲、先摸索
- 暴露本卷阶段短板
- 让局面看似可解，实则越走越窄

这一段最怕：

- 主角刚入卷就全面掌控
- 线索只增不变，没有误判和代价

### 3. 卷内进攻

任务：

- 主角开始主动布局
- 卷内主要攻防在此真正成型
- 卷级卖点从“看见”转为“兑现 / 转化”

这一段最怕：

- 和反应段只是同一件事做得更久
- 没有明显主动权变化

### 4. 卷内解决 / 卷尾缺口

任务：

- 本卷阶段性问题得到处理
- 主线拼图更新
- 留下逼下一卷接棒的行动性缺口

这一段最怕：

- 卷内问题没解决，硬拖到下卷
- 卷尾只有抽象悬念，没有新驱动

## 三、卷级关键点至少要看四个

### 1. 本卷扰动点

- 这一卷最先把读者拽进去的事件 / 异样 / 代价是什么？

### 2. 卷内第一情节点

- 什么时候本卷主问题真正成立？
- 主角从“接到新情况”变成“必须处理本卷核心麻烦”的点在哪里？

### 3. 卷内中间点

- 哪个新认知、新误差或新压力，把本卷从反应推向进攻？

### 4. 卷尾前最后关键信息

- 哪个信息让卷内解决成为可能，同时把主线推进到更深一层？

## 四、卷纲最小布置表

| 卷级位置 | 主角主要在做什么 | 本卷阻力 | 卷内回报 / 坏消息 | 与主线关系 | 卖点作用 |
| --- | --- | --- | --- | --- | --- |
| 卷内布局 | 进入新局面 | 新资源 / 新敌意 / 新压力 | 第一刺激 | 接上卷余波 | 重新显影 |
| 卷内第一情节点 | 本卷主问题成立 | 本卷核心麻烦浮现 | 被迫入局 | 主线再咬合 | 转成任务 |
| 卷内反应 | 试探、误判、摸索 | 外部阻力 + 卷内短板 | 连续受挫 / 小发现 | 为主线探路 | 带出代价 |
| 卷内中间点 | 新认知 / 新误差 | 压力改向 | 方向改变 | 主线拼图更新 | 二次升级 |
| 卷内进攻 | 主动布局与反击 | 阻力升级 | 扳回一城 / 更险 | 对主线施加主动影响 | 卖点兑现 |
| 卷尾前关键认知 | 最后一批卷内关键信息 | 真阻力显影 | 决战前置 | 主线缺口扩大 | 卖点冲顶 |
| 卷内解决 / 卷尾缺口 | 阶段性收束 | 终局阻力 | 卷内结果 + 下一卷缺口 | 把主线送进下卷 | 卖点留钩 |

## 五、章清单如何配合卷级四段

章清单至少要看出：

- 哪几章属于卷内布局
- 哪几章属于卷内反应
- 哪几章属于卷内进攻
- 哪几章负责卷内解决与卷尾缺口

如果章清单从头到尾都是同一强度、同一类型的推进，说明卷级结构还没站起来。

## 六、与卖点保真的关系

每卷必须写明自己属于哪一种：

- 强化卖点
- 兑现卖点
- 转化卖点
- 为下一次卖点爆发铺垫

并进一步说明：

- 本卷前段怎样让读者重新认出这本书
- 本卷中段怎样避免卖点断供
- 卷尾怎样把卖点送进下一卷

## 七、和总纲层的接口

- 总纲层负责全书九部件与长线推进。
- 分卷层负责把其中一段长线切成可读的卷级闭环。

所以卷纲必须回答：

- 本卷承接的是总纲哪一段压力？
- 本卷结束后，主线局势改变了什么？
- 下一卷接手的是哪一种新缺口？

## 八、硬性警报

出现以下任两项，应判卷纲未达标：

- 卷内主问题说不成一句话
- 中段没有明确改向点
- 卷内反应段和进攻段不分
- 卷尾只留氛围，不留行动性缺口
- 本卷读完对主线局势几乎没改变
- 卖点在本卷只出现一次，后面长期失踪

## 九、一句话收口

**卷纲不是把总纲切小，而是把总纲的一段压力切成可读、可回报、可交接的阶段性故事。**

## 十、最小卷级工件（新增）

```markdown
【卷段职责层】布局 / 反应 / 进攻 / 解决-缺口
【关键点布置层】扰动点 / 第一情节点 / 中间点 / 卷尾前关键信息
【卷尾交接层】本卷闭环 / 下一卷缺口 / 主线局势变化
【当前判定】Green / Yellow / Red
```

## 十一、Green / Yellow / Red 卷级判定（新增）

- `Green`：四段清楚、关键点明确、卷尾交接可追，可支撑真正的卷级闭环。
- `Yellow`：卷段方向大致成立，但中间点、卷尾缺口或主动权变化仍偏虚。
- `Red`：只有“很多章很多事”，没有卷级结构、关键点或交接缺口，读起来像流程堆叠。
