# 标题规范与章节验收增强

## 标题创作专业要求

### 分部 / 分卷标题创作要求

标题必须让读者一眼识别到：这一卷 / 这一部承诺的是**危险、秘密、代价、升级或真相**。理想状态是既能形成系列口味，又能单独成立。

#### 标题特征要求

- **阶段性明确**：体现该卷 / 该部在全书中的阶段任务
- **戏剧张力**：带出冲突、缺口、反常或代价
- **可连载**：系列内统一风格，但单卷也能抓人
- **强钩子**：制造问题与缺口，而不是把结果讲完

#### 禁忌标题类型

- 过于抽象：如“终局”“革命”“永生”等不落地概念词
- 纯口号 / 纯情绪：没有事件与冲突指向
- 过度内部化：像作者工程标签，而不是读者点击诱因

### 章节标题创作要求

#### 章节标题特征要求

- **情节推进性**：能暗示该章的核心事件或转折
- **情感共鸣**：触动读者的关注点
- **简洁有力**：优先短句与强动词，建议 6–12 字，尽量别超过 14 字
- **系列感**：与同卷其他章名保持风格统一
- **强吸引力**：让读者在目录页 1 秒内感到“这章有事发生”

#### 章名高点击句式（优先用）

- **命令句 / 禁令句**：如“别点”“先留证”“别回头”
- **审判 / 羞辱句**：如“你被分流了”“默认死亡”
- **代价 / 选择句**：如“救谁”“违令要命”

#### 强制禁忌

- 禁止工程标签化：不要用“剧情转折”“秘密揭露”“证据推进”这类内部分类词做章名
- 避免说明书式专业词堆叠：除非能翻成读者一眼懂的威胁句
- 避免把哲学和主题词直接当章名

## 标题创作流程要求

### 第一步：要素提取

- 从该卷 / 该章的核心事件中提取关键词（案由、地标、证据、程序、禁令、代价）
- 从主要冲突中提取情绪关键词（恐惧、羞辱、代价、背叛、救赎）
- 从主线谜团中提取缺口关键词（谁在掩盖、为什么要封口、证据去哪了）

### 第二步：组合创新

- 将“证据 / 程序”与“代价 / 禁令”组合
- 将“地标”与“危险 / 缺口”组合
- 将“口供 / 关系”与“反转”组合

### 第三步：风格检验

- 是否让读者一眼看出“本章 / 本卷有事要发生”
- 是否避免油腻玩梗和过度内部化表达
- 是否与当前作品整体语气一致

### 第四步：读者测试

- 是否能在 1–3 秒内传达危险、悬疑、冲突或代价
- 是否能激发对内容的好奇心
- 是否避免了过于小众、难懂的专业术语

## 章节节奏管理

### 三段式结构

- **悬念开头**：快速入场，不绕背景
- **冲突深化**：中段至少一次实质推进
- **悬念结尾**：结尾必须留下转折 / 揭露 / 危机之一

### 节奏搭配

- 动作章节与思辨章节要穿插，避免连续空转
- 每章至少植入一个伏笔点或推进一个既有伏笔线
- 每章都要有明确的情感张力和读者体验回报
- 每 5 章左右安排一个小高潮，每 15 章左右安排一个大高潮

### 章末钩子设计

每章结尾至少命中以下 3 类中的 1 类：

- **剧情转折钩子**：突然变化或计划被打乱
- **秘密揭露钩子**：部分真相暴露，激发追问
- **新危机钩子**：新威胁、新倒计时、新困境入场

## 章节验收增强清单

### 章节微型剧本最小验收

- [ ] 粒度达标：完整填写微型剧本模板全部字段
- [ ] 时间地点：至少 1 处精确到具体日期 / 时段 / 地点
- [ ] 三类冲突：人物-人物 / 人物-制度 / 人物-自我，至少命中 1 类
- [ ] 镜头分解：3–5 个镜头，具备感官细节与功能注释
- [ ] 对话片段：3–6 句关键对白，含潜台词标注
- [ ] 动作节点：≥3 个“起因 → 动作 → 结果”的微因果链
- [ ] 证据清单：≥2 项可审计证据载体
- [ ] 状态变化：人物 / 关系 / 资源 / 空间至少一项变化
- [ ] 悬念钩子：明确指向下一章的问题句或条件触发
- [ ] 反向引用：绑定上级故事线与台账编号

### 分卷层额外验收

- [ ] 卷级启动表（前五章）已完成
- [ ] 5/15 节奏布点图已完成
- [ ] 爽点布点表已完成
- [ ] 章清单与具体章节规划一致
- [ ] 标题与钩子分布没有同质化到发板

## 反面信号

- 章名像作者工程标签
- 章节只有“有事发生”，没有具体证据、动作与后果
- 结尾只有抽象感慨，没有下一章触发条件
- 节奏表和章清单写了，但不能真正拆章执行
