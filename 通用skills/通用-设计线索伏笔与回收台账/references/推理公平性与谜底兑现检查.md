# 推理公平性与谜底兑现检查

## 何时必须加做这一检查

当台账里存在以下任一情况时，不要只做普通回收管理，还要加做“公平性与谜底兑现”检查：

- 存在核心谜底、真凶揭露、身份翻案、旧案反转；
- 存在长线误导，且读者会把它当成真答案；
- 终局回收不只是解释，而是要改写人物关系或主线认知版图。

## 公平推理四问

1. **读者有没有提前见过关键拼图**
   - 终局所依赖的关键事实，前文是否已有痕迹、细节或可回指载体。

2. **误导能不能被拆回去**
   - 假答案是否依赖真实线索的错误组合，而不是纯作者遮挡。

3. **主角靠什么赢到答案**
   - 是补做了验证、重排了证据、对质出裂口，还是承担了某个行动代价。

4. **谜底揭示之后改变了什么**
   - 至少改变一项：人物关系、风险等级、主线认知、代价分配、下一轮缺口。

## 台账字段补强建议

对承担谜底功能的关键条目，建议额外补齐以下字段：

- **谜面类型**：事实谜面 / 动机谜面 / 身份谜面 / 手法谜面
- **首个可见信号**：读者第一次能看到它的场景或细节
- **验证动作**：主角最终靠什么动作把它坐实
- **末次翻案证据**：真正推翻旧解释的证据载体
- **揭示后余波**：谜底兑现后新增的代价、缺口或卷级压力

## 谜底兑现最低要求

一条合格的谜底兑现，至少要完成以下五件事中的四件：

1. 让旧解释失效；
2. 让前文信号重新排序并显得“原来早就在那里”；
3. 让人物做出新的关系判断或行动选择；
4. 把责任、代价或真相重新归位；
5. 留下一点新的不安，而不是所有问题一起归零。

## 常见不公平信号

- 末章突然出现新人物、新规则或新能力，独占解题权限；
- 真相主要靠反派自白成立，前文几乎没有足够证据支撑；
- 误导只有遮蔽，没有真实可误判依据；
- 谜底解释得很完整，但读者回看前文时找不到可验证落点。
