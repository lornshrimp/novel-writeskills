# 人性锚点与社会议题耦合

## 核心结论

- 超长篇故事面若只靠机制或主题标签维持，中段必塌。
- 合格故事面必须把**主线悬念**与**主角执念 / 创伤 / 亏欠**绑定。
- 社会议题应作为冲突燃料，而不是观点口号。

## 人性锚点写法

- 明确主角最怕失去什么。
- 明确主角为什么必须查下去，而不是“因为正义感”。
- 明确某个旧痛 / 旧债如何在关键节点反咬主角。

## 社会议题耦合方式

- 议题先进入人物处境，再进入案件机制。
- 优先落到：职业压力、制度缝隙、城市生存摩擦、亲密关系撕裂。
- 让议题通过具体事件、证据和后果显影，不要用大段观点代替剧情。

## 自检问题

- 拿掉社会议题标签后，故事面是否还剩可执行冲突？
- 拿掉主角的人性锚点后，故事面是否还能稳定供血？
- 议题是否改变了人物选择、案件方向或代价级别？
