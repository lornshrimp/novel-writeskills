# 爽点模式库与三阶节拍

> 来源口径：`网文技法.md` 与 `修订体系.md` 的爽点分布规则。

## 六种基础爽点模式

| 模式 | 结构 | 适配场景 |
| --- | --- | --- |
| 装逼打脸 | 轻视→展示实力→震惊 | 都市 / 职场 / 玄幻 |
| 扮猪吃虎 | 表面弱→关键爆发→反差 | 隐藏身份 / 逆袭 |
| 越级反杀 | 悬殊对抗→逆转 | 玄幻 / 竞技 |
| 打脸权威 | 权威质疑→证明→反转 | 学院 / 职场 |
| 反派翻车 | 反派得意→计划崩盘 | 全题材 |
| 情感升温 | 平淡→惊喜→关系推进 | 言情 / 日常 |

## 三阶布局（写作侧）

- 微爽点：每章至少 1 个（信息差 / 小成功 / 小反转）。
- 中爽点：每卷 5-8 个（阶段突破、身份揭示、关键胜负）。
- 大爽点：全书 3-5 个（终局级或卷终级兑现）。

## 卷级密度参考（3卷/25万字）

| 卷 | 大爽点 | 小爽点 | 密度参考 |
| --- | ---: | ---: | --- |
| 卷一 | 3-4 | 8-10 | 约每1.5章1个小爽点 |
| 卷二 | 4-5 | 12-15 | 约每章1个小爽点 |
| 卷三 | 2-3 | 5-6 | 约每章1个爽点 |

## 爽点黄金公式

```text
铺垫（期待）→ 展示（行动）→ 反馈（他人/环境反应）→ 余韵（代价与收获）
```

> 缺“反馈”会显著削弱爽点体感。

## 审计信号

- 若连续 4 章无中小爽点，判定为“节拍失温风险”。
- 若爽点只靠旁白总结、无动作与后果，判定为“空爽点”。
- 若爽点与主卖点无关，判定为“噪点爽点”。
