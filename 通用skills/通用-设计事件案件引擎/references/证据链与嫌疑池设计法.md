# 证据链与嫌疑池设计法

## 一、证据链至少分三层

### 1. 入场证据

负责把事件从“怪事”拉进“可调查对象”。

### 2. 误导证据

负责把主角和读者带偏，但必须能被拆回去。

要求：

- 看起来像真证据
- 能解释眼下现象
- 最终能被更高质量证据推翻

### 3. 翻案证据

负责把嫌疑池重新洗牌，或把主线拼图推进一格。

这类证据最好能同时带来：

- 一个新答案
- 一个更大的新问题

## 二、嫌疑池不要只会“加人减人”

嫌疑池的变化至少有四种：

- 排除：某人被证伪
- 升温：某人的危险性突然上升
- 转移：嫌疑从个人转向机构 / 群体 / 规则漏洞
- 伪洗白：暂时看似清白，后面才知道洗白本身就是误导

## 三、常见假嫌疑池信号

- 嫌疑人只是名单，没有动机差异
- 所有人都“看起来有问题”，但没有不同层次的压力
- 关键对象只因作者不想暴露真相，就被刻意写得模糊

## 四、实用检查

- [ ] 每个重要嫌疑对象都有不同的“为什么像问题源”
- [ ] 至少有一个嫌疑对象的危险性来自制度 / 关系 / 利益，而不是脸谱化坏人味
- [ ] 翻案后，主线拼图前进了，不只是事件结束了
