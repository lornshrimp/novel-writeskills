# 证据链与嫌疑池设计法

## 一、证据链至少分三层

### 1. 入场证据

负责把事件从“怪事”拉进“可调查对象”。

### 2. 误导证据

负责把主角和读者带偏，但必须能被拆回去。

要求：

- 看起来像真证据
- 能解释眼下现象
- 最终能被更高质量证据推翻

### 3. 翻案证据

负责把嫌疑池重新洗牌，或把主线拼图推进一格。

这类证据最好能同时带来：

- 一个新答案
- 一个更大的新问题

## 二、嫌疑池不要只会“加人减人”

嫌疑池的变化至少有四种：

- 排除：某人被证伪
- 升温：某人的危险性突然上升
- 转移：嫌疑从个人转向机构 / 群体 / 规则漏洞
- 伪洗白：暂时看似清白，后面才知道洗白本身就是误导

## 三、常见假嫌疑池信号

- 嫌疑人只是名单，没有动机差异
- 所有人都“看起来有问题”，但没有不同层次的压力
- 关键对象只因作者不想暴露真相，就被刻意写得模糊

## 四、实用检查

- [ ] 每个重要嫌疑对象都有不同的“为什么像问题源”
- [ ] 至少有一个嫌疑对象的危险性来自制度 / 关系 / 利益，而不是脸谱化坏人味
- [ ] 翻案后，主线拼图前进了，不只是事件结束了

## 五、谜团推进五步法

当案件承担“解谜”职责时，最好让推进链条至少经过以下五步：

1. **先立第一问**：不是急着回答“谁干的”，而是先把“到底发生了什么”钉清。
2. **旧解释先失效**：主角必须先把当前最像答案的解释拆坏，谜团才会真的前进。
3. **嫌疑池重排**：推进后要么换重点对象，要么把嫌疑从个人转移到关系 / 机构 / 旧案。
4. **行动验证**：关键推进尽量通过跟踪、对质、复盘、复勘、试探等动作完成，而不是旁白总结。
5. **真相同时回答三件事**：事实是什么、为什么会这样、谁从遮蔽中获益。

## 六、解谜公平性检查

### 公平性最低要求

- 终局答案依赖的关键信息，前文必须已有可回溯信号，不能整包末章空降。
- 误导必须像真答案，但必须能够被更高质量证据拆回去。
- 关键翻案证据最好不是单一口供；至少再有动作、载体或旧细节呼应。
- 主角的最后判断，应建立在“他做成了什么验证”上，而不是“作者终于告诉他了”。

### 常见失真信号

- 前文没有足够铺垫，终局只能靠新设定兜底。
- 假答案只负责拖时间，不负责逼出更好的问题。
- 所谓推理，其实只是角色在最后一章突然想明白。
- 真相解释很完整，但对人物关系、代价和主线拼图没有产生新后果。
