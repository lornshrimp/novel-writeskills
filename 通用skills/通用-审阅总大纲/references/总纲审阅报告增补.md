# 总纲审阅报告增补

## 报告里必须写重的部分

### 1. 范围说明

- 覆盖了哪些总纲层文件
- 哪些未覆盖
- 由此带来的判断风险

### 2. 机制递进与供血诊断

- 四阶段机制递进卡在哪里
- 当前最强的主线里程碑锚点是什么
- 单元案矩阵有没有持续供血
- 终局闭环条件是否提前埋好

### 3. 具体修改建议

建议以“插入位置 + 结构补丁 + 预期收益”格式书写，例如：

- 在【某故事面 / 某矩阵位置】补写【机制或拼图】，让【某阶段】不再空转。
- 将【某单元案】明确绑定到【某块主线拼图】，避免主线断供。
- 在【某阶段结尾】增加【阶段胜利 / 反噬 / 代价】，形成卷群级闭环。
