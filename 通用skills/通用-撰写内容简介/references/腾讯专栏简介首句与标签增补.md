## 简介硬规则（P0）

1. 简介作用只有一个：强化吸引力，不是讲完整梗概。
2. 首句必须承担“书名补充 + 题材识别 + 入口钩子”三职能中的至少两项。
3. 简介应明确：人物处境、核心冲突、持续悬念。
4. 简介须与书名同向，不得吸引错位读者。

## 结构建议（P1）

1. 可加入标签词（题材、关系、风格）帮助快速筛选读者。
2. 先亮卖点，再控信息；不要先讲背景百科。
3. 可用小剧场（偏人物关系文）或剧情概述（偏强剧情文），但语气须与正文一致。
4. 设悬念可以，但不能把后续关键反转讲完。

## 简介禁行项

1. 把大纲写进简介。
2. 首句空泛，不给读者可见钩子。
3. 只堆设定名词，不落人物困境。

## 自检清单

- [ ] 首句是否可独立成为点击理由？
- [ ] 是否明确“谁在什么困局里”
- [ ] 是否写出了读者想继续看的问题？
- [ ] 是否与书名承诺一致？
