# 腾讯专栏：简介多编辑规范增补

来源：
- 《看了这之后，你肯定知道简介该怎么写了》（作家助手，8.8万阅读）
- 《【新人】签约冲刺04：签约冲刺—再查一遍自己的书名和简介》（阅文起点创作学堂，7万阅读）

---

## P0规则：简介格式硬规定

- **禁止一大段**：千万别写成一大段，读者直接略过，多分几段
- **禁止云山雾罩**：简介是为了让读者了解你的书，不是猜谜
- **字数：百来字左右**（少了没内容，多了看着烦）
- 格式上的简洁和文字上的简洁同等重要：段落分明 > 揉成一团

---

## P0规则：简介功能定位

- **简介不是大纲**：大纲是给人看故事内容，简介的最大任务是吸引读者来看正文
- **简介是餐厅招牌菜**：只展示勾起兴趣的部分，不把味道全写出来
- **最终目标**：简洁有力，突出最大卖点和特色，让读者眼前一亮

---

## P0规则：简介四步骤（编辑丹青/夜宵综合）

1. **确立类别/风格/卖点**：明确受众，不同类别写法不同（军事≠都市≠仙侠）
2. **体现背景**：几个关键词暗示即可（玄幻、校园、都市等），不必详述
3. **主角形象**：一两句话概括主角最重要的身份信息
4. **留噱头/伏笔**：让读者有阅读正文一探究竟的欲望

---

## P0规则：简介主线公式（编辑汤圆/虎牙）

**人物 + 时间 + 地点 + 做什么**

示例：《大国重工》简介——"国家重大装备办处长冯啸辰穿越到了1980年，看他如何与同代人一道，用汗水和智慧，铸就大国重工。"
- 时间：1980年
- 人物：国家重大装备办处长冯啸辰
- 地点：我国
- 做什么：与同代人一道，铸就大国重工

在主线公式基础上，可加入：金手指、最终目标、主要对手。

---

## P0规则：简介禁用表达

| 禁用 | 原因 |
|------|------|
| "她将何去何从" | 导语式，无实质信息 |
| "她将如何抉择" | 导语式，读者无法判断故事是什么 |
| "因机缘巧合两人相识" | 平淡叙述，无冲突无吸引力 |
| "一天晚上她遇见了他就此着迷" | 平淡叙述，建议直接展现戏剧性 |
| 长篇累牍（把大纲搬来） | 乏味，失去简介精髓 |
| 含糊不清、故作高深 | 读者看完不知道讲什么，劝退 |
| 千篇一律的套路简介 | 让读者认为是随大流之作 |

---

## P0规则：让简介有冲突和反差（编辑妖妖灵）

- 简介要有冲突有爆点或者有反差反转
- 放弃平淡叙述，直接将戏剧性展现出来
- 现言常见形式：①小场景对话体（对话中隐藏信息，展现男女主性格）②叙述体中展现强烈反差

---

## P0规则：简介内容取舍原则（编辑远征/丹青）

- 从剧情入手、或从金手指入手、或着重突出噱头——选一个切入点，不要三都要
- 抓住故事中最吸引人的**一个点**来写
- 不可能面面俱到，取舍是必然
- 以突出作品最大卖点和特色为主的简介，都是好简介

---

## P0规则：二次元/特定品类调性要匹配（编辑丹青）

- 二次元读者更喜欢段子手式抖机灵，或能体现核心的关键对白片段
- 常规的苦大仇深不适合二次元
- 简介风格最好与作品风格搭

---

## P0规则：好简介的最高目标

- 好简介 = 突出特色，让读者眼前一亮，产生"跟别的妖艳贱货不一样"的新鲜感
- 让读者记住，简介就成功了
- 简洁有力，用词准确不晦涩，让读者迅速抓住关键信息
