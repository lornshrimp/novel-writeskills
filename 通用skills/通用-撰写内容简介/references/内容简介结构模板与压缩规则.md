# 内容简介结构模板与压缩规则

## 标准结构

1. 主角是谁，当前处境如何。
2. 什么反常事件 / 关键选择把他拖进局里。
3. 眼前冲突是什么。
4. 更大的承诺是什么：真相、升级、代价、关系撕裂中的哪几项。

## 任务类型

1. 系列全书简介
2. 分部 / 分卷简介
3. 章节简介

## 输入最小集

- 标题
- 当前书名 / 投放名（如有）
- 关键人物
- 一句话差异化卖点
- 明线冲突
- 暗线冲突
- 情感核心
- 关键象征意象

## 版本输出最低要求

- 至少 1 个主版简介
- 若涉及平台或投放差异，至少给：长版 / 短版 / 平台重点版
- 每个版本都必须明确自己的主钩子

## 一版一主钩规则

- 标准长版：主钩 1 个 + 辅钩最多 2 个
- 平台短版：主钩 1 个 + 辅钩最多 1 个
- 引流 / 超短版：只允许 1 个主钩

若读完后说不出“这一版最先抓我的是什么”，则判定失败。

## 压缩提醒

- 第一段优先放“角色 + 处境 + 反常”。
- 第二段优先放“当前冲突 + 更大承诺”。
- 句子要能独立成立，避免一长串逗号拖死节奏。

## 失败信号补充

- 一段里硬塞四五个卖点，像大纲压缩块
- 首句像另一个更长的书名，而不像简介开头
- 读完只记住设定很多，却说不出人物当下在怕什么

## 失败信号

- 简介看完还不知道主角在跟什么对抗。
- 只有设定酷点，没有故事承诺。
- 语言过空，像所有书都能套。
