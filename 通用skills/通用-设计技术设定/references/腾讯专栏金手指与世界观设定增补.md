## 金手指 P0 规则

### 金手指的定义

金手指 = **主角独有的、能够帮助主角完成主线目标的东西**。

- 主角要提升社会地位 → 能帮助他提升社会地位的就是金手指
- 主角要破案 → 能帮助他破案的就是金手指
- 主角要谈恋爱 → 对他谈恋爱有帮助的就是金手指

### 金手指的基础功能（必须满足）

金手指必须**为主角完成主线目标提供帮助**。这是金手指的不可或缺的基础功能。

### 金手指的附加功能（尽量实现）

好的金手指还应**为小说凸显卖点提供便利**。常见卖点-金手指组合：

- 热血卖点 → 代价类金手指（付出越大、威力越大，凸显"虽千万人吾往矣"的热血）
- 搞笑卖点 → 反差类金手指（系统、特殊能力带来出人意料的喜剧效果）
- 悬疑卖点 → 信息差类金手指（额外信息来源，加速调查同时制造悬念）

### 经典案例参考

| 作品 | 金手指 | 基础功能 | 附加功能 |
| --- | --- | --- | --- |
| 《诡秘之主》 | 灰雾之上 | 提供信息帮助破案/完成主线 | 虚张声势、配角脑补迷化 |
| 《大王饶命》 | 系统 | 修行途径，提高社会地位 | 主动怼人、花式怼人 |
| 《我师兄实在太稳健了》 | 洪荒了解 + 超高天赋 | 苟全性命、延续宗门 | 稳健隐忍的戏剧感 |
| 《大奉打更人》 | 警察经验 + 地书碎片 | 破案提升社会地位 | 吹牛+配角脑补+社会性死亡 |

## 世界观 P0 规则

### 世界观的本质定义

世界观 ≠ 静止的升级体系或势力分布。世界观 = **这个世界的历史（世界剧情）**。

- 世界以前是什么样的
- 世界是如何发展的
- 世界未来会走向何方

### 为什么需要世界观（适用场景）

**适用范围**：幻想类小说（升级文、无限流、权谋文、探索类悬疑文等）

**不适用范围**：偏现实背景的小说（创业文、文娱文、体育竞技、职业文），不会涉及高层次世界改变，不需要构造世界剧情。

**核心作用**：解决审美疲劳——避免读者在长期阅读后对"强敌-升级-败敌-夺宝-强敌"的单调螺旋升级感到倦怠。

### 世界观的最简构造方法

**聚焦两个势力间的单一矛盾冲突**，是构造世界观的最高性价比捷径：

1. 确定两个核心势力（如人族 vs 禁区、命运 vs 诸神）
2. 明确这两个势力之间的核心矛盾
3. 基于这一矛盾推演过去历史
4. 随主角社会地位提升，主角逐步介入世界剧情并开始影响走势

**扩展路径**（可选）：增加势力个数，各有立场，剧情复杂度几何倍增长。但矛盾点不建议增加，会模糊主角主线的终极目标。

### 世界观在故事中的作用节奏

1. 开篇：主角处于世界剧情的旁观者或边缘参与者位置
2. 中篇：主角随社会地位提升，逐步了解并介入世界剧情
3. 高潮：主角成为能够影响、改变世界走势的关键人物

## 设计技术设定时的检查项

设计金手指前检查：

- [ ] 这个金手指是否直接服务于主角的主线目标？
- [ ] 这个金手指是否还能服务于作品的主要卖点（热血/搞笑/悬疑）？
- [ ] 金手指是否与主线卖点精准匹配，而非随意安插？

设计世界观前检查：

- [ ] 这是幻想类小说吗？（是→需要世界观；现实类→可跳过）
- [ ] 是否能提炼出两个核心势力之间的单一矛盾冲突？
- [ ] 这一矛盾是否能推演出足够支撑全书的历史背景？
