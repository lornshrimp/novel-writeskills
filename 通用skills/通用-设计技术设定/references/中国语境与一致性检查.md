# 中国语境与一致性检查

## 中国城市语境检查

- 基层与社区结构：派出所 / 社区警务、网格员、居委会、物业与业委会之间的张力是否可感。
- 平台与生活链路：外卖、快递、网约车、短租民宿、直播带货等是否成为合理证据源或风险源。
- 空间特征：老旧小区、城中村、写字楼园区、医院学校、地铁与地下空间是否写得具体而非悬浮。
- 人情与面子：熟人社会残留、关系链协查、灰度成本是否写出来。
- 信息与舆论：群聊扩散、偷拍视频、热搜与删帖、匿名爆料的双刃剑是否纳入冲突。

## 叙事伦理边界

- 隐私与伤害控制：高风险细节必须降维处理，避免形成教程。
- 法理与常识边界：不要把办案写成无视程序的爽文推土机，摩擦必须显性。
- 社会议题嵌入：用人物命运与案件后果呈现，不用口号替代戏剧。
- 读者信任维护：所有反转都应“意料之外、情理之中”，并可回指前文。

## 跨文档一致性检查

- 字段口径统一：同一证据 / 要素在不同文件中的字段、编号格式、时间口径一致。
- 时间线协调：首次出现、回收、反转、失效窗口在所有文档中统一。
- 命名规范：人名、地名、单位名、物证名、黑话保持一致。
- 关系映射：人物关系、组织规矩、地点规则与证据链互相支撑，不打架。
- 影响范围：同一要素对剧情的影响强度不能忽强忽弱。

## 与大纲设定对接

- 情节支撑：要素能支撑关键情节与反转节点，尤其是证据闭环点。
- 人物互动：要素必须符合人物能力范围与认知水平。
- 社会背景：手法 / 程序与城市生态相匹配。
- 冲突合理：冲突来自线索不够、证据被拿走、程序卡点、人性代价，而不是作者强按。
- 演进逻辑：从异常 → 怀疑 → 证据战 → 真相阶梯的推进保持一致。

## 最后四问

- 这个设定放进中国城市日常里，像不像真的会发生？
- 它会被谁卡住？
- 它最怕哪条口径不一致？
- 它有没有和别的设定打架？
