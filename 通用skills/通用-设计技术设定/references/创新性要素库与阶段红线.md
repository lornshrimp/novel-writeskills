# 创新性要素库与阶段红线

本节用于提升“手法 / 证据载体 / 程序冲突”的新鲜度与现实触感；引入时必须同时遵守可复盘、可证伪、成本可见与伦理边界。

## A. 日常物品犯罪化

原则：选读者熟悉的日常物品或流程，把“正常功能”异化为冲突工具，但必须保留专业逻辑与可证伪破绽。

推荐字段：

- 物品 / 流程
- 正常功能
- 异化用途（非教程化描述）
- 留下的痕迹 / 凭证（至少 2 类）
- 最容易露馅的破绽
- 证伪路径（谁 / 用什么 / 多久 / 代价）

## B. 民用科技异化

原则：优先使用家庭 / 社区 / 平台已普及的民用科技与记录系统，让恐怖感来自“我也在用”。

落地时必须写清：

- 数据从哪来
- 被谁拿到
- 如何进入冲突
- 如何留下可核查记录
- 隐私边界、权限边界、程序摩擦与社会后果

## C. 本土化社会工程学

原则：利用社区、熟人关系、基层服务信任形成误导路径，把悬疑落在熟悉的社会结构里。

落地时注意：

- 重点写“信任如何被借用 / 伪造 / 反噬”
- 不写具体违法教程
- 必须配套可证伪破绽：口径不一致、台账缺页、时间窗冲突、点位对不上、回执空白等

## D. 心理恐怖的证据化

原则：把“细思极恐”从氛围落回证据链，让气味、声响、光源、回避动作都能回指某个机制或真相碎片。

落地时注意：

- 写清感官细节对应的现实来源 / 触发条件
- 若涉及创伤记忆具象化，要写清创伤事件、形成时间与触发节点

## 阶段红线速记

- 开篇阶段：不得直接给决定性证据，不得让万能监控覆盖一切。
- 单元案升级阶段：允许误导，但必须能证伪；不得用神谕式直觉直接锁定真相。
- 黑幕逼近阶段：不得让关键证据只靠作者安排到手，不得无成本解决舆论或权力阻力。
- 终局阶段：不得靠大段自白替代证明，不得临时空降关键设定或关键证据。

## 使用提醒

- 先选阶段，再选要素。
- 任何创新要素都必须服从“可证据化 + 有边界 + 有代价 + 有程序摩擦”。
