# 通用-输出今日头条版｜分节级补救映射与详细规则回填

## 适用场景

本文档用于在执行今日头条版改写时，若章节某处不符合平台风格、钩子密度、互动驱动或中等节奏的要求，进行局部补救的查表与方案。

## 一、开篇钩子密度不足补救

### 症状
- 开篇 500 字内无信息增量或悬念。
- 钩子间隔过长（超过 250 字无新钩子）。
- 读者在开篇可能"失焦"，不知所云。

### 补救方案

| 问题位置 | 原因 | 补救策略 |
| ------- | ---- | ------ |
| 开篇第 1-3 段 | 背景交代过多 | 砍掉背景，直接进"已发生或即发生"的事件；背景通过"主角思考"零散穿插 |
| 开篇第 2-5 段 | 信息密度过低（纯景物、纯心理） | 交织"信息 + 冲突"，每一句都要有指向（线索指向谎言、动作指向情绪、对话指向真相） |
| 中段（500-1500 字） | 冗余对话或思考 | 对话直奔主题，删除"闲聊"；心理转为"动作 + 细节"（如想到恐怖的事，用"身体发抖"替代"心里很害怕"） |
| 章尾（1500 字后） | 没有悬而未决的钩子 | 在最后 100-200 字补一个"新线索 / 异常发现 / 人物突然反应"，让读者"不得不追更" |

## 二、情节与冗余比例不适

### 症状
- 情节推进速度慢（应 50% 情节但实际 35%）。
- 冗余描写过多（过多心理独白、景物堆砌、对话闲聊）。
- 读者"感觉进度慢"，完读率可能下降。

### 补救方案

| 问题处 | 原始结构 | 今日头条改法 |
| ---- | ------ | --------- |
| 心理独白段落 | 整段呈现主角想法 | 拆散成"动作 + 对话 + 环境细节"，同时体现心理 |
| 景物描写段落 | 连续 3-5 句描写同一场景 | 提取 1-2 个最关键的感官细节，删除其余；景物改为"主角发现异常"的方式呈现 |
| 人物对话 | 日常闲聊或解释过多 | 对话变成"双方都在隐瞒"的紧张感；每句话都透露线索或推动事件 |
| 过渡段 | "两天后，林宇又来到了案发现场" | 直接从"林宇推开门"开始，时间点在对话或动作里自然显露 |

## 三、互动驱动点不足

### 症状
- 章节内容不容易激发"想评论"的欲望。
- 没有"你怎么看？"的话题点。
- 分享欲低，评论数预期不达标。

### 补救方案

| 问题 | 补救策略 |
| ---- | ------ |
| 悬念不引发讨论 | 悬念改为"有多种可能的解读"而非"单一谜底"；如 "这个纸条是威胁还是求救信？" 而非 "这个纸条是谁留的？" |
| 人物选择无道德两难 | 每个重要选择都要设置"各有道理的两条路"，让读者想评论"我会怎么选" |
| 线索无"反转空间" | 设置"表面意思 vs 真实含义"对立的线索，如"他的不在场证明看起来完美，但仔细想想..." |
| 结尾无悬而未决 | 章末最后 2-3 句务必留一个"新信息 / 新疑惑"，让读者"必须评论讨论" |

## 四、视角锚点缺失

### 症状
- 出现"上帝视角"信息（主角不可能知道的事）。
- 感知锚点模糊（描写没有明确说"是谁感知的"）。
- 沉浸感降低，代入感消失。

### 补救方案

| 问题 | 改法 |
| ---- | ---- |
| "房间里有一股怪味" | 改为"他推开门，一股不对劲的气味扑面而来" |
| "远处的灯塔在夜色中显得诡异" | 改为"他透过黑暗，看到远处灯塔闪烁的光，突然有种不祥的感觉" |
| "他没注意到角落里的监控摄像头" | 保留"他没注意到"的认知边界，后续才让他发现；不要在这一章就告诉读者有摄像头 |
| 叙述性过去式（"之前…"） | 改为"他回忆起…"或"他突然想起…"，加感知主体 |

## 五、中等节奏把握不住

### 症状
- 全章高度紧张（读者容易疲劳）。
- 节奏起伏不足（平淡一整章）。
- 快慢段比例失衡。

### 补救方案

| 问题 | 补救策略 |
| ---- | ------ |
| 全章都是快节奏 | 在 40%-60% 处插入一个"对话缓冲"或"回忆反思"段，让读者喘气；但这个缓冲段仍需透露线索 |
| 全章都是慢节奏 | 在 20%、50%、80% 处各放一个"冲突升级"或"新线索出现"的快速推进 |
| 节奏跳跃不自然 | 快段和慢段之间用"主角的动作反应"或"环境变化"自然过渡 |
| 中段掉速（疲软期） | 中段务必设置"新悬念抛出"或"前一段的反转"，重新拉起张力 |

## 六、人性落点不够突出

### 症状
- 故事变成单纯的"案件推理"。
- 人物显得"为了破案而破案"，无血无肉。
- 读者没有情感共鸣，只有"解谜"快感。

### 补救方案

| 问题 | 补救策略 |
| ---- | ------ |
| 反派过于绝对的坏 | 在反派的动机或选择里加一丝"可以理解的理由"，即使最后被定罪也不是"傻坏蛋" |
| 正派过于绝对的好 | 在主角的选择里加入"两难"或"缺陷"，如"为了破案牺牲了什么？" |
| 案件推理无人情味 | 在线索的发现或验证过程里，加入"这与某个人的情感 / 记忆 / 选择"的关联 |
| 故事无社会现实感 | 将某个案件细节与现实社会议题（如信息泄露、职场压力、网络暴力）勾连，增加代入感 |

## 七、非正文区块污染（模板句清零）

### 硬禁止规则
- 禁止在正文中出现"对照项落在…""先把同一句话拆碎…""只求能追溯…"等模板化元话术。
- 禁止出现连续 2 句"我这边…+把…+对齐…+连接词…+口径名词"的垃圾句群。
- 禁止出现"校准、锁死、拆开、同步"等企业行话。

### 清零方案
若发现污染，按如下策略整理：

| 污染段落类型 | 清零法 |
| -------- | ----- |
| "对照…口径…对齐…" | 全删，改为叙述性表达或对话 |
| "先把…再把…" 的拆碎式 | 合并成一句或一段自然叙述 |
| "我心里…我这边…我的感受…" 堆积 | 改为"动作 + 环境细节"，剔除"我"字重复 |
| "只求…不求…" 的保留 | 删除此类元话术，改为具象的选择行为 |
