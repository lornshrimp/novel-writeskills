# 通用-输出今日头条版｜平台共性执行细则

## 适用目标

将章节改写成更适合今日头条平台的版本，同时保持：

- 主要信息不丢失
- 强情节、弱冗余的紧凑表达
- 输出为纯小说正文结构 + 互动驱动口径

## 一、平台共性任务定义

- 面向今日头条都市悬疑读者（以 25-40 岁社会中坚、35 岁以上高粘性用户为核心），强调开篇钩子密集、强情节推进、互动驱动与中等节奏。
- 保留事件链、关键对话、关键设定、伏笔与章末悬而未决钩子。
- 不允许新增事件链、证据、设定、人物、地点、时间点或因果。

## 二、平台生态特征

### 读者画像

- 核心群体：25-40 岁社会中坚（职场人、专业人士），35 岁以上用户粘性最强（日均 78 分钟使用时长）。
- 核心需求："有逻辑、有代入感、能持续追更"，既有碎片阅读速度需求，也有深度内容吸附力。
- 核心厌恶：拖沓冗余、信息密度过低、无互动话题点的"孤立"内容。

### 内容分发逻辑

- 平台采用"个性化智能分发 + 分级流量池"机制：冷启动流量池（500-1000 人）→ 中级流量池（1-10 万人）→ 高级流量池（10 万 +）。
- 关键数据权重：阅读量 15% + 收藏量 15% + 评论率 15% + 分享量 10% + 完读率 10% = 65%（互动与深度阅读占 65%）。
- **开篇特别权重**：3000 字开篇权重占 45%，直接决定作品生死。
- 原创内容有明显倾斜：获得原创标签、优先推荐通道、更高广告分成。
- 都市悬疑已成为第二大热门题材（仅次于玄幻仙侠），在平台拥有独立冷启动流量池。

### 平台主流赛道

- 规则怪谈 / 中式悬疑是 2026 年顶流赛道，搜索量暴涨 300%。
- 爆款公式：**强逻辑智斗 + 规则反杀 + 中式克苏鲁氛围 + 群像塑造**。
- 主流子类型：规则怪谈（机制解谜）、民俗悬疑（民俗元素 + 现代反转）、AI 犯罪（新型作案手法）、时空交错（穿越/循环设定）。

## 三、跨题材共同硬约束

### 1. 零新增事实

- 所有情节与信息必须来自原始章节正文。
- 允许的扩写仅限于等价重述、感官细节丰富化、心理动作细节化。
- 精确数值、时间戳、ID、回执编号、界面字段必须原样保留。
- 叙事人称与视角必须与原文一致。
- 钩子增强只能靠呈现方式，不得新增"下一章事件"。

### 2. 沉浸式小说体裁

- 输出文件只能写成小说正文 + `## 作者有话说`。
- 正文中禁止教程、复盘、清单、方法论、额外小标题、列表、表格、代码块、引用块。
- 证据材料只能以剧情内短插入形式出现，并紧接人物反应与下一步行动。

### 3. 共同门禁

- 正文字数以 CJK 统计，必须在 3500–6500 之间。
- 单章字数参考区间：3500–5500 字（今日头条中等节奏偏快的单章）。
- 必须满足足够比例重写，不得直接复用大量原句与原段落结构。
- 必须去除模板腔、企业行话、说明书式段落。
- 标题必须重写，且同步写入文件名与正文标题。
- **开篇钩子密度**：每 200 字必须有一个可感知的信息增量或情绪波动；禁止开篇 500 字内无钩子。

## 四、今日头条平台节奏共性

### 1. 开篇钩子密集策略（平台特殊权重）

开篇 3000 字在算法权重中占 45%，是作品成败关键：

- **第 1 句务必直达冲突**：不要背景交代，要直接进"已发生或即将发生的事件"。
- **每 150-200 字一个信息增量**：关键线索、人物矛盾、环境异常、时间压力等。
- **开篇 3 章节奏公式**：
  - 第 1 章：绝境 + 金手指觉醒 / 关键线索出现（4000-5000 字）。
  - 第 2 章：初步反击 / 线索印证 + 新悬念抛出（4000-5000 字）。
  - 第 3 章：推理深化 / 误导显现 + 强钩子收尾（4000-5000 字）。
- **钩子类型多样化**（避免单调）：信息钩子（新线索）+ 情绪钩子（情绪爆发）+ 认知钩子（颠覆认知）。
- 章末钩子必须留下"必须追更才能解开的悬疑"，而非"自成闭环就断更"。

### 2. 强情节、弱冗余的表达模式

今日头条读者平均阅读时长 2.3 分钟，但深度内容阅读增长超 150%——意味着"高信息密度但需要沉浸感"：

- **内容比例基准**：50% 情节推进 + 25% 人物互动 + 15% 细节铺垫 + 10% 情绪爆发。
- **语言风格**：直奔主题、信息密度高、句式精炼。
  - 不写"房间很冷"，写"房间温度突然降了 10 度，冷气机却没开"（信息 + 异常点）。
  - 不写"他很紧张"，写"他的手开始发抖，手里的遗书被捏出褶皱"（动作 + 细节）。
- **段落结构**：短段为主，频繁换行。每个段落一个信息点，快速推进。
- **对话设计**：每句对话都要有作用——传递信息、推动情节或塑造人物，禁止"闲聊"。

### 3. 中等节奏的互动驱动

- **章内节奏**：快慢交替，但快段比例应高于纵横（约 60% 快段 + 40% 缓冲段）。
- **互动话题点预埋**：每章至少设置 2-3 个"读者想评论、想讨论"的点——悬疑线索、人物选择、伦理困境等。
  - 例如：主角的决定是否正确？隐藏的真凶是谁？这个线索有何含义？
- **评论/分享驱动**：章末悬念必须能驱动"迫不及待评论讨论"的欲望。
- **完读率优化**：中间不设节奏下跌点，全程信息密度不低于 50%。

### 4. 视角选择与感知锚点（强沉浸感）

- **首选**：第一人称或严格限制性第三人称，完全锁定主角感知边界。
- **感知锚点规则**（强制）：所有描写必须经由主角的感官、记忆或推理。
  - 不写"房间里一股死人的气味"，写"他闻到房间里传来一股熟悉而令人作呕的气味——那是死亡的味道"。
  - 不写"隐约可见的塔影在暮色中显得格外诡异"，写"他透过暮色朦胧的光线，看到塔影在远处显得有些不对劲"。
- **视角切换禁忌**：短暂切换到其他角色视角时，不能让读者掌握主角未知的核心真相；切换后快速回到主角。

### 5. 线索铺陈与悬念管理

- **3:2 原则**（相比纵横的 3:1）：每 3 个误导性线索，搭配 2 个关键性线索（互动驱动要求更多解谜爽感）。
- **间隔释放**：关键线索均匀分布，避免前半无线索后半全爆的局面。
- **误导设计**：线索本身真实，但指向谎言——读者"发现真实线索"但推理错误，增加转折冲击。
- **章末悬而未决**：不全部解开，留下 1-3 个新伏笔，吸引追更。

### 6. 人性落点与社会现实映射（核心吸附力）

今日头条读者对"人性复杂"的关注度高于单纯"悬疑诡计"：

- 每章至少保留一处体现人性温度或社会现实的内容。
- 反派也要有"可理解的动机"，而非单纯的坏人设定。
- 正面角色要有缺陷与两难的选择，而非纸片人英雄。
- 故事最终指向应该是"人的复杂"而非"案子的复杂"。

### 7. 社交媒体属性（平台特有）

- **话题感**：章节内容要能引发"这事怎么看？"的讨论。
- **分享欲**：优先设置令人惊讶、恐怖或感动的节点，激发"转发给朋友"的冲动。
- **互动语气**：作者有话说里可适度加入"你怎么看？""你猜下一步？"等互动问题。
- **评论友好**：在正文中适度埋入"容易被圈评"的细节，如"这个细节说明了什么？""他这样选择对吗？"。

### 8. 原创声明与平台激励意识

- 确保所有内容符合原创要求，不使用大段摘抄或明显抄袭。
- 作品需考虑长期数据留存，而非短期流量爆发。
- 与大号创作者、平台编辑的互动也是间接的流量扩大手段。

## 五、平台审核红线（内容硬约束）

- 不涉及政治敏感内容：不攻击国家制度、不损害国家形象、不煽动民族仇恨、不宣扬恐怖主义或宗教极端主义。
- 不宣扬负面价值观：不渲染血腥暴力色情，正面角色价值观符合主流，反派最终必须被绳之以法。
- 不引用现实真实案件、地名或国家机构名称，避免对号入座；可创作虚构案件与地理背景。
- 章节标题不得为纯符号、emoji 或"标题党"式夸张表述。
- 专业知识（法医学、刑侦学等）表述必须准确，不得误导。
- 原创性：内容必须自创，不得复制粘贴他人作品；改写须达 50% 以上重写比例。

## 六、今日头条 vs 纵横版对标

| 维度 | 今日头条 | 纵横小说 |
|------|--------|--------|
| **开篇权重** | 45% 在 3000 字开篇 | 标准匀分布 |
| **钩子密度** | 每 150-200 字一钩 | 每 250-300 字一钩 |
| **情节比例** | 50% 情节 + 25% 互动 | 40% 情节 + 30% 互动 |
| **用户画像** | 25-40 岁社会中坚，中短长混读 | 80、90 后男性，深度长读 |
| **互动驱动** | 评论、分享、话题讨论优先 | 完读率与长期留存优先 |
| **线索比例** | 3:2（误导:关键） | 3:1（误导:关键） |
| **节奏感** | 快段 60% + 缓冲 40% | 快段 50% + 缓冲 50% |
| **社交属性** | 强（内容需有讨论话题） | 弱（内容专注解谜） |
