# 感官链与生理反应补强

## 基本原则

人物情绪优先落到：

- 呼吸
- 喉咙
- 指节
- 后颈
- 胃部
- 皮肤触感
- 错误动作

不要优先落到抽象判断。

## 用生理反应替代抽象情绪

少写：

- 他很害怕
- 她很崩溃
- 气氛很压抑

多写：

- 指节发白
- 后颈一凉
- 胃里往下坠
- 喉咙发紧
- 手上的动作停半拍
- 该拿的东西没拿稳

## 感官链的推荐顺序

### 危险场景

- 先触觉 / 听觉
- 再嗅觉 / 温度
- 最后才给视觉总述

### 压抑场景

- 先空气与声音
- 再人物的呼吸、衣料、椅背、门把手等触感
- 再补空间的光线与距离

## 一段内的最低感官密度

需要强化沉浸时，尽量做到：

- 至少 2 种感官
- 至少 1 个身体反应
- 至少 1 个具体动作后果

## 五感不是平均分配，而是按场景取舍

- 先决定这场最该让读者**看到 / 听到 / 闻到 / 摸到 / 尝到**什么。
- 不必五感平均铺满；平均反而容易发花。
- 同一段里优先让 1–2 种感官担主轴，其余感官作为补刀。

## 展示优先，说明兜底

- 优先让读者通过动作、表情、体感、声音去感到情绪。
- “他很害怕 / 她很压抑 / 气氛很怪”这类说明，只有在展示已给足仍需提纯时才少量保留。
- 紧张段落越高压，越应少说抽象话、多给可感细节。

## 多余法则

不能直接推动情节、人物、气氛或主题的描写，都要警惕是否多余。

优先删：

- 只为好看而好看的形容词堆叠
- 同义重复的修饰词
- 对已经明显可感的情绪再解释一遍
- 只在炫观察力、却不改变阅读体验的细枝末节

## 感官描写的额外职责

感官不只是“让句子更漂亮”，还应至少承担其一：

- 建背景
- 立基调
- 映照人物情绪
- 放大冲突
- 暗示主题

若完全不承担这些职责，就容易沦为漂亮噪音。

## 克制陈词滥调

- 少用现成俗语替代真实感受。
- 优先寻找本场、本人、本物件才成立的感官细节。
- 同一类惊艳表达在一章内不要反复使用，第一次叫新鲜，第二次就容易发腻。

## 常见改法

- 把“环境很怪”改成“哪里怪、怪到身体哪儿了”
- 把“他愣住了”改成“动作停半拍 / 呼吸断一下 / 视线错开”
- 把“她很紧张”改成“重复一个无意义的小动作”

## 合规与克制

- 恐怖不等于血腥
- 优先用体感、异味、声音、温差制造压迫
- 感官链要服务情节，不为辞藻堆砌

## 读者类型平衡提醒

- 若目标是快节奏连载 / 商业正文，感官描写应更短、更准、更靠近行动。
- 若目标是情绪浓度或文学质感较高的段落，可适度延长描写，但仍要服从“多余法则”。
