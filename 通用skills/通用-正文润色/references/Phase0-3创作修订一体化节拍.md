# Phase0-3 创作修订一体化节拍

> 来源口径：`经验总结.md`（改进版流程）+ `修订体系.md`（A/B/C 批次）。

## 四阶段总览

```text
Phase 0 设定与规划 → Phase 1 初稿写作 → Phase 2 三批次修订 → Phase 3 终审输出
```

## Phase 0（设定与规划）

- 完成总纲（世界观 / 人物 / 主线）。
- 完成分卷目标与冲突层级。
- 逐章规划同步写：开场钩子、章末留钩、爽点位、伏笔位。

## Phase 1（初稿写作）

- 按章纲推进，不在初稿阶段过度打磨。
- 写一章登记一章伏笔。
- 每 5 章快扫一次：字数、对话比、前后 200 字钩子。
- 卡文时允许占位推进，避免全局停摆。

## Phase 2（三批次修订）

### A 批次：人物 / 钩子 / 对话

- 前 200 字钩子覆盖目标：100%。
- 后 200 字留钩覆盖目标：90%+。
- 对话占比目标：25%-55%。
- 配角需有独立动机，不得工具人化。

### B 批次：伏笔 / 金手指 / 精简

- 伏笔清单完整且可追踪。
- 金手指具备“成长 + 触发 + 代价”。
- 连续 200 字解释段必须打散。
- 高频冗余词、拟声词、重复意象做控频处理。

### C 批次：网文化 / 道具呼应 / 风格

- 章末钩子类型明确（悬疑 / 选择 / 危机）。
- 关键道具形成跨章呼应弧线。
- 清理元叙述与 AI 万能句式。
- 同概念命名口径统一。

## Phase 3（终审输出）

- 全文通读（重点首尾与卷切节点）。
- 伏笔终审：确保无遗漏、无伪回收。
- 格式统一（标点、分段、排版）。
- 输出终稿并落盘审阅记录。

## 正文润色对接规则

- `通用-正文润色` 默认工作位在 Phase 2 与 Phase 3 之间。
- 若发现结构性问题（钩子缺失 / 回报缺失 / 伏笔断链），应回写上游，不可仅句级抛光。
- 若发现问题属于阶段错序（例如把 C 批次问题放到 A 批次解决），应先纠正批次再润色。

## 一句话执行原则

先把“讲什么”修对，再把“怎么讲”修好；先过门禁，再追文采。
