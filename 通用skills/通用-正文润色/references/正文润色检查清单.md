# 正文润色检查清单

## 开工前

- [ ] 范围已确认：整章 / 场景 / 段落
- [ ] 强度已确认：轻润 / 中润 / 重润
- [ ] 不能动的钉子已确认

## 第一遍：内容与功能

- [ ] 事实、时间、称谓、因果一致
- [ ] 本章主承诺明确
- [ ] 章首异常 / 冲突 / 悬念已立住
- [ ] 中段至少一次回报清晰可指认
- [ ] 章末留下更具体的新缺口 / 新危险 / 未完成动作

## 第二遍：语言与镜头

- [ ] 抽象判断已尽量落回动作、物件、反应
- [ ] 长短句节奏已经拉开
- [ ] 对话不再是说明书式问答
- [ ] 去掉提示语后，关键对话仍大致听得出人物声音差异
- [ ] 段落适合移动端阅读

## 第三遍：弱项路由

- [ ] 若主要问题在对话，已补停顿、回避、错位、潜台词
- [ ] 若主要问题在对话，已检查对话是否同时承担推进、塑人与联结，而非只传信息
- [ ] 若主要问题在对话，已至少补出一种读者入口：认同 / 同情 / 喜爱 / 好奇
- [ ] 若主要问题在动作，已把推进改成可见动作链与后果
- [ ] 若主要问题在心理，已把抽象情绪尽量改成身体反应与环境刺点
- [ ] 若主要问题在场景，已补空间、物件、界面、秩序失真的可感细节

## 最终验收

- [ ] 更顺，但没有更虚
- [ ] 更抓人，但没有更花
- [ ] 中段回报与章末钩子仍然有效
- [ ] 没有为了“好看”而误伤事实、规则、证据或人物口径
