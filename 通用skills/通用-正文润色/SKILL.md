# 通用-正文润色

正文润色的目标不是单纯把句子写漂亮，而是把**已有章节**修到“更顺、更准、更抓人、但不失真”。

当题材 Skill 与本 Skill 组合调用时：

- `通用-正文润色` 负责共性流程、默认交付、弱项路由、句级动作与沉浸控制
- 题材 Skill 负责题材边界、项目特例与裁判源补充

## 润色前裁判源复核（强制）

- 动手润色前，必须认真仔细阅读与本章实际出场人物对应的人物传记，以及与本章背景、技术、规则、证据、程序链、世界观有关的技术设定文档。
- 不得只凭现成正文、上一轮润色痕迹或审阅摘要直接下刀；凡涉及人物动机、能力边界、关系状态、设定代价、证据机制的句段，都必须回到作者侧裁判源核对后再改。
- 若同一人物传记或技术设定存在多个版本，必须先确认当前可执行的最新版本；若关键裁判源缺失，必须先补齐、定位或明确阻断，不能边猜边润。
- 若本轮润色会触碰人物弧线、规则代价、证据链、关系位置或背景事实，必须在回写前复核一次相关裁判源，确认没有把正文润偏。

## 共性执行顺序

1. 先认真仔细阅读本章相关人物传记、技术设定 / 背景设定、控制卡与最新审阅结论，确认不能动的钉子、可增强处与不可越界处。
2. 再确认范围、强度与不能动的钉子。
3. 先保信息、事实、因果、视角与结构功能。
4. 先修章首抓力，再修中段回报，再修章末钩子。
5. 再按弱项路由处理对话、动作、心理、肖像、景物、静态或场面。
6. 最后用“润色后链路职责保真复核卡”确认卖点、回报、钩子与场景职责没有被润平，再做句级精修、沉浸复检并直接回写。
7. 若本轮改动影响正文篇幅，回写前必须重跑 `scripts/count-chapter.ps1 -MinCJK 3500 -MaxCJK 6500`；若本次连同 `## 作者有话说` 一并处理，再补跑 `scripts/count-afterword.ps1 -MinCJK 200 -MaxCJK 300`。
	- 正文门禁只看脚本输出的 `BodyCJK / Meets3500 / WithinRange`；`Len` 只作参考。
	- 不得用“去空白统计”“NoWhitespaceLen”或编辑器字符数代替 `BodyCJK`。

## 本轮精修新增关注点（新增）

- 默认优先判断前 `300` 字是否已经交付人物抓手、第一压力与一个读者会继续看的问题。
- 中段润色不能只让句子更顺；必须确认本章至少有一次情绪、关系、信息、行动或身份层面的实质回报。
- 精修时优先把抽象判断改写成动作、感官、物件、停顿、具体选择与可见后果，减少作者替角色总结。
- 默认把章节自指、连载导览腔与作者讲评口吻视为沉浸事故：凡出现“这几章”“前几章”及其“前面几章 / 前面这几章”变体、“后几章”及其“后面几章 / 后面这几章”变体、“整章”及其“这整章 / 这一整章”变体、“上一章”“下一章”“本章 / 这章”“后文会”“我们会看到”“读者会发现”等，应优先改写为人物当下可感的后果、记忆、关系余波或具体动作。
- 若原稿大段平均、句群同宽、段落过厚，默认要顺手做手机端友好压缩，但不能压掉信息与节拍。
- 重要人物的魅力与脆弱应同时可见；润色不能把真实短板洗成“无瑕高光”。

## 弱项路由规则

- 对话发直、口气混、说明腔重 → 走“语言类”修法
- 动作发虚、推进靠概括 → 走“行动类”修法
- 心理发空、情绪只会直说 → 走“心理类”修法
- 人物脸模糊、神态无抓手 → 走“肖像类”修法
- 环境不立、氛围不入戏 → 走“景物类”修法
- 房间器物证物写死 → 走“静态类”修法
- 多人现场不热不紧、没有焦点 → 走“场面类”修法

## 极其快的节奏与极其强烈的冲突——正文润色规则（新增）

### 快节奏润色的核心方向

**1. "句长与段长的多样化"**
快节奏的正文应该在句长和段长上制造变化，避免齐平带来的单调感：
```
原稿：陈勉走进房间。房间里很暗。他看到了一个人影坐在对面的椅子上。陈勉的心跳加速了。他慢慢走向椅子。那个人影突然站了起来。

润色方向：句长变化 + 短句密集
陈勉推门。房间黑洞洞的。暗处，一个身影站了起来。不对。那不是站，而是突然从黑暗里扑出来。陈勉的脚步一滞。

[短句-短句-中长句-短句的变化制造了加速感]
```

**2. "动词的活跃化"与"时间信号的即刻化"**
快节奏的正文应该尽量用"即刻发生的动词"代替"正在发生的心理"或"可能会发生的推测"：
```
❌ 陈勉想起了上次见到这个人的时候。那次他说过如果再见面就不是朋友。陈勉意识到自己可能要面临一些危险。

✅ 陈勉认出了他。"如果再见面就不是朋友。"上次的话浮现在脑子里。危险已经近在咫尺。
```

快版本用"认出、浮现、近在咫尺"这些即时动词代替了"想起、意识到、可能会"。

**3. "转折点与节奏断层"的有意设置**
快节奏的正文应该有意识地制造几个"断层"或"转折"，让读者心理节奏跟着波动：
```
原稿：陈勉走进去，发现房间里有人，他很惊讶，但他想到了应该怎样应对。

润色：
陈勉推门进去。
房间里有人。
那是——不可能。
他应该在医院里。
```

短断句制造的断层让节奏产生了波动。

### 强冲突润色的核心方向

**1. "对立的具体化"与"立场碰撞的即刻化"**
强冲突的正文应该把所有的"对立"都化成具体的、可看见的、即刻发生的碰撞，而不是抽象的威胁或虚空的紧张：
```
❌ 陈勉和对面的人对立了。他们想要的东西不一样。陈勉很害怕，但他不想退缩。

✅ 陈勉盯着对面的人。"把钱放下。"那个人说。"除非你先把人放了，"陈勉说。两个人的目光在半空中撞在一起。无法妥协的两条线。
```

强版本让"对立"有了身体、有了语言、有了可感知的碰撞。

**2. "代价的物化"与"后果的身体化"**
强冲突的正文应该把所有的代价都改写成"可看、可听、可感"的具体后果：
```
❌ 她意识到自己犯了一个巨大的错误。错误会导致一些可怕的后果。

✅ 她按下的按钮没有回弹。灯灭了。楼下传来了巨大的爆炸声。烟灰透过窗缝吹进来。
```

强版本用具体的动作、声音、视觉来呈现"巨大的错误"。

**3. "权力与位差"的动作化表现**
强冲突的正文应该通过角色的动作、姿态、空间位置来表现权力的不对等，而不是直白地说"他掌握了主动权"：
```
❌ 绑匪掌握了绝对的主动权。他可以决定人质的生死。陈勉感到自己很无力。

✅ 绑匪靠在椅子上，腿搭在桌子上，手里转着一把刀。陈勉站在他五步远的地方，双手垂在身体两侧。"你说。"绑匪说。
```

强版本通过座椅、腿、刀、距离、指令来呈现权力的不对等。

**4. "信息的非对称性"的对话与细节中体现**
强冲突的正文应该充分利用"不同人物掌握不同信息"这一点，在对话和细节中体现这种非对称：
```
陈勉不知道警察已经在楼外围好了。所以他还在和绑匪谈判。而绑匪知道警察在外面，所以他的每一句话都在试探："你告诉我，有多少人在外面？"

[对话中透露出来的：陈勉在猜，绑匪在确认。这就是信息非对称。]
```

### 快节奏 + 强冲突润色的检查清单

[ ] 句长是否有明确的变化（不是齐平的节奏）？
[ ] 段长是否有意识的起伏（短段制造加速，长段制造沉重）？
[ ] 动词是否都是即时发生的（而不是"想象""推测""可能会"）？
[ ] 时间信号是否明确且推进（什么时刻、多少时间、倒计时）？
[ ] 对立是否具体化了（不只是"对立"，而是具体的行为、语言、空间对抗）？
[ ] 代价是否身体化了（不只是"有后果"，而是可看、可听、可感的具体后果）？
[ ] 权力分布是否通过动作和姿态体现（而不是直白叙述"谁掌权"）？
[ ] 信息非对称性是否在对话和细节中有所体现（让读者能察觉到"这个人知道但那个人不知道"）？
[ ] 是否避免了心理总结、作者旁白和抽象描写？
[ ] 整章是否维持了"快速+冲突"的双重强度，而不是某段掉回原来的节奏？

## 硬规则

- 不新增关键事实，不删减核心信息。
- 未认真仔细阅读本章相关人物传记与技术设定 / 背景设定文档前，不得直接改写人物动机、关系状态、规则表现、证据位置或背景事实。
- 不得把正文润成说明书、散文或空抒情。
- 不得把中段回报和章末钩子润没。
- 不得保留或新增“这几章 / 前几章 / 前面几章 / 前面这几章 / 后几章 / 后面几章 / 后面这几章 / 整章 / 这整章 / 这一整章 / 上一章 / 下一章 / 本章 / 这章 / 后文会 / 我们会看到 / 读者会发现”这类章节自指 / 导览式话术；若原稿已有，必须改写成场景内表达，而不是原样擦亮。
- 不得为了顺口删掉关键证据、编号、时间点、规则边界。
- 不得在不必要时整章推倒重写。
- 不得用“看起来差不多够长了”代替脚本校验；凡本轮改动影响正文或 `## 作者有话说` 篇幅，都必须显式重跑对应字数脚本。
- 若润色说明、日志或后续审阅报告需要记录字数，必须直接引用脚本字段 `Len / CJK / BodyCJK / Meets3500 / WithinRange / MeetsAll`，不得改写成“去空白统计”。
