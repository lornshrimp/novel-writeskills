# 三批次修订 SOP 与门禁节奏

> 来源口径：`修订体系.md` / `经验总结.md`。

## 批次顺序（不可逆）

```text
A（人物/钩子/对话）→ B（伏笔/金手指/精简）→ C（网文化/道具/风格）→ 终审
```

> 禁止跳批次：A 未完成不得进 B；B 未完成不得进 C。

## A 批次门禁

- 前200字钩子覆盖率：目标 100%。
- 后200字留钩覆盖率：目标 90%+。
- 对话占比：25%-55%。
- 配角动机：每个常驻配角需有独立动机信号。

## B 批次门禁

- 伏笔清单完整（埋设/状态/回收窗口）。
- 金手指弧线存在“升级 + 代价 + 触发器”。
- 连续 200 字纯解释段应打散。
- 高频冗余词与拟声词需控频。

## C 批次门禁

- 章末必须有悬疑钩 / 选择钩 / 危机钩之一。
- 重要道具需形成跨章呼应弧线。
- 清理元叙述与 AI 万能句式。
- 同概念用词统一。

## 执行节拍建议

- 每 3-5 章做一次快审（字数 / 对话比 / 钩子覆盖）。
- 连续多章回炉时，先修“结构门禁”再修“语言质感”。
- 修订日志必须记录：本轮批次、未过门禁项、下一步补丁动作。
