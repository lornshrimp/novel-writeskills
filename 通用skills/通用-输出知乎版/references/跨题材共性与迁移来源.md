# 通用-输出知乎版｜跨题材共性与迁移来源

本文件记录 `通用-输出知乎版` 当前承接的跨题材共性来源，以及无损迁移期间的执行边界。

## 当前来源

- `异能志怪/.github/prompts/输出知乎版.prompt.md`
- `都市悬疑/.github/prompts/输出知乎版.prompt.md`

## 当前可稳定上收的知乎平台共性

- 知乎语境下的高概念、高现实映射、逻辑链可追与克制锋利感
- 输出结构固定为“正文 + ## 作者有话说”
- 标题更接近规则句 / 代价句 / 冲突句，但仍保持小说感
- 证据必须可复盘、信息必须可验证、推进点需高密度可追
- 平台风格必须先于降相似度，不得把知乎版写成泛中文平台稿

## 当前仍保留在源 Prompt 的详细规则

在无损迁移完成前，以下内容仍以源 Prompt 正文为详细依据：

- 各题材在知乎平台下的研究补强条款
- 题材特有的现实议题绑定、规则链、民俗/调查边界
- 精细化标题类型、作者有话说口吻、章内节拍补丁

## 使用边界

- 命中 `通用-输出知乎版` 时，必须同时加载对应题材的 `题材名-输出知乎版`。
- 若当前任务仍处于迁移试点阶段，必须继续回看源 Prompt 与试点台账，不得假设本文件已经完全替代旧 Prompt。
