# 章节开头检查清单

## 一、范围

- [ ] 只处理前 150–300 字或前 2–3 段
- [ ] 没有误伤后文主体结构

## 二、30 秒测试

- [ ] 前 2–3 段里，是否已有异常 / 冲突 / 悬念缺口中的至少两项？
- [ ] 读者是否能立刻感觉到“这事现在就有问题”？

## 三、动作测试

- [ ] 主角是否在做一个眼下必须处理的动作？
- [ ] 压力是否跟动作绑在一起，而不是抽象悬着？

## 四、代入测试

- [ ] 情绪是否落在身体反应、微动作、物件或环境错位上？
- [ ] 是否避免了纯背景说明式开头？
- [ ] 若用对话开场，前几句是否已经能分出身份 / 熟悉度 / 位差？
- [ ] 若用对话开场，是否至少有一句暴露人物眼下在护什么、怕什么或想抢什么？

## 五、抓力与节奏

- [ ] 第一波动作是否早于大段解释？
- [ ] 是否存在至少一个可拍出来的反常细节？
- [ ] 句长有变化，不像平推说明文？
- [ ] 开头最后一句是否能把读者推向下一段？

## 六、留存测试

- [ ] 看完这 150–300 字，读者会不会自然想问“为什么会这样？”
- [ ] 这个问题是否足够具体，而不是大而空？
- [ ] 读者是否已经通过对话或动作，对人物形成至少一种联结：认同 / 同情 / 喜爱 / 好奇？

## 七、红线复检

- [ ] 没有提前讲透答案
- [ ] 没有新增关键事实
- [ ] 没有把后文回报偷跑掉
- [ ] 没有把章首润成和其他章节一个腔
