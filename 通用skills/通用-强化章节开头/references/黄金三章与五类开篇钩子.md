# 黄金三章与五类开篇钩子

## 黄金三章最小任务

- **第一章**：立主角抓手 + 抛第一异常 / 第一压力。
- **第二章**：让主角进入第一个挑战 / 案件动作。
- **第三章**：给一次小高潮或更强悬念升级，让追读理由成立。

## 五类常用钩子

1. 绝境钩子
2. 反常钩子
3. 物品钩子
4. 身份钩子
5. 信息差钩子

## 使用原则

- 先用钩子把读者拽进来，再用动作和代价把读者按住。
- 钩子不能只是怪话或怪景，必须尽快引出主角反应。
- 钩子之后要接冲突与动作，不能只接解释。

## 扰乱事件与第一扇门的区别

章首强化时要特别避免把两者写糊：

- **扰乱事件**：先打乱日常，让读者感到事情不对劲了。
- **第一扇门**：把主角真正推进主冲突，回不去原样生活。

章首通常先需要扰乱事件，不一定立刻需要第一扇门；但如果两者都不到位，开头就容易只有气氛没有抓力。

## 六类高效开场法

除已有五类钩子外，还可按以下切法快速判断开头是否有效：

1. **一句异常先落地**：先给人物名 + 即将发生的坏事。
2. **从事件中段切入**：先让人已经在挨打、被追、被问、被撞见。
3. **真实情感入场**：先给一个能立刻建立联结的情感痛点。
4. **回到过去式预告**：先让读者知道这件事后来会很糟，再回到起点。
5. **态度型开场**：靠强口气、强观察、强立场抓住读者。
6. **对话型开场**：问题、试探、威胁、口径差先于背景说明出现。

选择哪一种，取决于这章最值钱的东西到底是：危险、关系、情绪还是秘密。

### 对话型开场最低要求

- 前几句不要只是打招呼，要先把关系顶出偏差
- 让读者听出人物声音差，而不是统一作者腔
- 至少露出一项：位差 / 风险 / 隐瞒 / 旧账 / 误会
- 对话之后要立刻接动作或后果，不能接成长段解释
- 开场里的这波对话，最好直接贴住本章卖点：危险、关系、秘密或压力中的一种

## 第一屏常见失败信号

- 背景先于人物
- 人物先于动作，但没有困境
- 对话有声音，没有冲突
- 情绪有了，但没有具体威胁
- 开头整段都在解释，读者仍不知道为什么必须继续看

## 第一屏修正优先级

1. 先把“有事要发生”前置
2. 再把人物眼下会失去什么说具体
3. 再把动作提前到能看见的地方
4. 最后再压缩背景与说明

## 章首自检

- 前 150–300 字是否出现至少两项：异常 / 冲突 / 情绪锚点 / 悬念缺口？
- 开头是否已经像故事开始，而不是作者热身？
- 章首是在交代世界，还是已经打乱了主角的世界？
