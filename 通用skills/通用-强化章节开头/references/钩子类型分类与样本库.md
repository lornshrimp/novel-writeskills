# 钩子类型分类与样本库

> 基线口径：默认起点中文网（未标平台同口径）。

## 使用说明

- 先判断本章职责（信息投放 / 冲突升级 / 危机推进 / 情绪转折）。
- 再从六类钩子中选 1 类主钩子，必要时叠加 1 类副钩子。
- 相邻章节尽量避免同类钩子连续重复。

## 六类钩子

### 1) 冲突钩

**定义**：角色目标直接对立。

**章首样例**：

```text
“这房子你不能动。”
“凭什么？”
```

**章末样例**：

```text
门开了，屋里那人没有让路。
```

### 2) 悬念钩

**定义**：抛出自然问题但暂不回答。

**章首样例**：

```text
门上贴着三根鸡毛。
```

**章末样例**：

```text
桌上多了一碗温水，没人承认放过它。
```

### 3) 异常钩

**定义**：违背常态或规则的细节出现。

**章首样例**：

```text
下山三次，都回到同一棵树前。
```

**章末样例**：

```text
口袋里的石头自己翻了个面。
```

### 4) 强情绪钩

**定义**：高压情绪以身体反应落地。

**章首样例**：

```text
他握着手机的手在抖，不是怕，是气。
```

**章末样例**：

```text
她按灭屏幕，三分钟后抬头，眼神变了。
```

### 5) 动作钩

**定义**：高张力动作进行中被截断。

**章首样例**：

```text
他挑着担子往后山跑，身后脚步越来越近。
```

**章末样例**：

```text
门正在自己关上，他还没来得及喊。
```

### 6) 对话钩

**定义**：一句对话改写读者认知。

**章首样例**：

```text
“你爷死前留话：小心赵家的人。”
```

**章末样例**：

```text
“从一开始，你就知道了。”
```

## 选择规则（简版）

- 信息投放章：优先悬念钩 / 对话钩。
- 对抗升级章：优先冲突钩 / 动作钩。
- 规则显影章：优先异常钩 / 悬念钩。
- 人物转折章：优先强情绪钩 / 冲突钩。

## 常见误用

- 只用问号当钩子（无事件承载）。
- 用作者预告腔代替钩子（如“下一章会…”）。
- 章首章末钩子与本章职责脱节。

## 过线检查

- 前 200 字至少命中 1 类有效钩子。
- 后 200 字至少留下 1 个未闭合压力点。
- 钩子必须落在“动作 / 物件 / 关系 / 规则后果”之一上。
