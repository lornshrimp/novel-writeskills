# 腾讯专栏黄金三章多维共识增补

来源：《开头，开头，太.....太重要了》7万阅读，写作技巧专栏，多位作者圆桌

---

## P0 核心提炼

### 黄金三章功能定位

- 黄金三章的唯一目标：**让读者有看下去的欲望，而不是展示文笔或设定**
- 黄金三章的作用是**签约门槛**，不是成绩保证；三章过后依然要保持同等节奏，不能前紧后松
- 对没有读者基本盘的新人而言，黄金三章比有粉丝的大神更重要，失败了没有口碑发酵机会

### 黄金三章最忌（多位作者共识）

1. **堆设定**——不要在开头大段堆砌世界观、规则、背景
2. **云里雾里与主角无关**——开篇出现大量与主角无关的内容
3. **抛出太多概念**——不要一口气给读者塞太多陌生概念
4. **出场人物太多**——人物密集导致代入感不足、重心不在主角
5. **无谓堆砌词藻**——秀文笔而非讲故事；读者看小说要看故事，不是看文字游戏

### 黄金三章必须做到（多位作者共识）

1. **主角第一章出现**——根据类型点明世界观或基调；设定要融入剧情而非硬堆
2. **期待感优先**——开头最核心的是**期待感**（参考退婚、三年之约等结构）
3. **简洁明了**——条理清晰，剧情明确，低阅读难度，不秀设定
4. **切入点切在主干**——开头切入点不能切歪，要切在核心矛盾/主角利益上
5. **主角言行符合时代背景**——否则出戏，容易劝退

### 开头四步公式

多位作者归纳出的开头写作顺序：
1. 确定写的是什么**类型**
2. 确定用什么**技法**（爽文 / 逗比文 / 悲情文）
3. 确定**情绪铺垫**（大背景：末日 / 重生 / 穿越 / 国恨家仇；主角入场时的情绪基调）
4. 设计**故事结构**
5. 最后才是**遣词造句**

### 变与不变的平衡

- **不变**：知道目标读者是谁，知道他们需要什么爽点，不随意天马行空违背读者口味
- **变**：在不变的基础上有所创新，不重复雷同情节，不情怀式重复

## 与现有规则的关系

- 本增补提供"多作者实战共识"视角，印证并细化章首抓力的核心方向
- "开头四步公式"可作为章节控制卡写前准备的补充框架
- "黄金三章≠止于三章"是追读链完整性的作者实战警告
